# ⏱️ 数据范围与复杂度

| n 最大值 | 大概能用 | 别用 |
|---:|---|---|
| 10 | 全排列、爆搜 | 基本都能试 |
| 20 | 子集枚举、状态压缩 | 过多嵌套 |
| 100 | 三重循环 | 指数 |
| 2000 | 双重循环 | 三重循环危险 |
| 100000 | O(n log n)、O(n) | O(n²) |
| 1000000 | O(n) | 多次排序要小心 |
| 10000000 | 一次扫描/公式 | 多一层循环 |

## 暴力升级表

| 暴力慢在哪里 | 升级 |
|---|---|
| 每次重新求区间和 | 前缀和 |
| 每次改一段 | 差分 |
| 反复查有没有 | 集合/字典 |
| 有序还线性找 | 二分 |
| 重复算子问题 | DP |
| 每次找最大最小 | 堆 |

> [!tip] 💬 德远的笔记
> 暴力不是错，错的是不知道暴力会不会死。
