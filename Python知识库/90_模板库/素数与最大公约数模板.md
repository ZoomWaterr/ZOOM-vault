# 素数与最大公约数模板

## 判断素数

```python
def is_prime(n: int) -> bool:
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

## 最大公约数 / 最小公倍数

```python
import math

g = math.gcd(a, b)
l = a * b // g
```

## 手写辗转相除法

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```
