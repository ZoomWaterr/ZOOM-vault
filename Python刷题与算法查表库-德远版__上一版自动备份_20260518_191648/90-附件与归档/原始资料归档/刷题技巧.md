# 输入处理

## 单行多个数字 → 列表

```python
nums = list(map(int, input().split()))
a, b = map(int, input().split())        # 两个变量
a, b, c = map(float, input().split())   # 浮点数
```

## 多行输入 → 列表收集

```python
n = int(input())
data = []
for _ in range(n):
    data.append(int(input()))
```

# 输出格式化

## 空格分隔输出列表

```python
a = [1, 2, 3, 4, 5]

# 方法 A：解包（最常用，最快）
print(*a)                              # → 1 2 3 4 5

# 方法 B：join（仅限字符串列表，速度最快）
print(" ".join(map(str, a)))           # → 1 2 3 4 5
```

## 每行输出一个

```python
print('\n'.join(map(str, 列表名)))
```

## 保留小数

```python
print(f"{x:.3f}")       # 保留 3 位小数
print(f"{x:.6g}")       # 6 位有效数字，自动删末尾 0
print(round(x, 3))      # 四舍五入保留 3 位
```

## 补零对齐

```python
print(f"{hour:02d}:{minute:02d}")   # → 08:05
```

# 循环技巧

## 双重循环：不重不漏找所有数对

```python
for i in range(n):         # 外层逐个扫描每个数
    for j in range(i):     # 内层让 i 和它左边所有数两两比较
        ...
```

本质：找出所有 $(i, j)$ 满足 $j < i$ 的组合。

## 拆分 → 处理 → 输出（模板）

```python
lst = input().split()          # 拆分
for item in lst:
    processed = 对item的处理     # 逐个处理
    print(processed, end=' ')   # 同行空格输出
```

# 列表技巧

## 列表推导式

```python
# 基本：生成平方列表
[i ** 2 for i in range(1, 21)]

# 带条件：筛选偶数平方
[i ** 2 for i in range(1, 21) if i % 2 == 0]

# 带判断：奇偶标记
["偶数" if i % 2 == 0 else "奇数" for i in range(10)]
```

## 排序

```python
a.sort()                    # 从小到大（改原列表）
a.sort(reverse=True)        # 从大到小（改原列表）
b = sorted(a)               # 返回新列表（不改原列表）
```

## 获取最小/最大值（非数字标准）

```python
# 按长度找最长单词
max(words, key=len)

# 按出现次数找最多的字符
max(d, key=d.get)
```

# 字符串技巧

## 基本操作

```python
s.replace("-", "")           # 删除所有横杠
s.rstrip(".")                # 去掉末尾句号
s.upper() / s.lower()        # 大小写转换
s[::-1]                      # 反转字符串
```

## join() 的正确用法

```python
# 核心：分隔符.join(字符串列表)
'-'.join(['2025', '01', '15'])   # → 2025-01-15

# 数字列表要先转字符串
' '.join(map(str, [1, 2, 3]))    # → 1 2 3
```

# 字典技巧

## 频率统计（模板）

```python
count = {}
for x in 数据:
    count[x] = count.get(x, 0) + 1

# 找出出现最多的
best = max(count, key=count.get)
```

# 数学工具

## 向上取整

```python
import math
math.ceil(a / b)

# 纯整数版本（不需要 import）
(a + b - 1) // b
```

## 最大公约数 / 最小公倍数

```python
import math
g = math.gcd(a, b)           # 最大公约数
l = a * b // math.gcd(a, b)  # 最小公倍数
```

# 取整方式对比

| 方法 | 适用场景 | 例子 |
|:---|:---|:---|
| `//` | 整数除法（**最常用**） | `sum // len` |
| `math.ceil(x)` | 向上取整 | `math.ceil(5.1) → 6` |
| `math.floor(x)` | 向下取整 | `math.floor(5.9) → 5` |
| `int(x)` | 仅正数取整 | `int(5.9) → 5` |
| `round(x, n)` | 四舍五入保留 n 位 | `round(3.1415, 2) → 3.14` |

# 一句话速查

| 场景 | 用这个 |
|:---|:---|
| 统计次数 | `dict` + `.get(x, 0) + 1` |
