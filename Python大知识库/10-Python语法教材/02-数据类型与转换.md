# 数据类型与转换

> 类型不是概念题，刷题里它决定你能不能把输入变成能算的东西。

## 五种基础类型

| 类型 | 例子 | 刷题用途 |
|:---|:---|:---|
| `int` | `10` | 计数、下标、循环次数 |
| `float` | `3.14` | 小数计算、几何题 |
| `str` | `"abc"` | 字符串处理、逐位处理 |
| `bool` | `True` | 条件判断 |
| `None` | `None` | 占位、默认值 |

## 最常用转换

```python
n = int(input())
x = float(input())
s = str(num)
nums = list(map(int, input().split()))
```

## 关键提醒

`input()` 永远先给你字符串。要算数，就先转数字。

```python
"10" + "1"   # "101"
10 + 1       # 11
```

## 常用判断

```python
s.isdigit()
s.isalpha()
s.isspace()
type(x)
```
