# 逻辑判断与流程控制

tags: #python/流程控制 #python/逻辑

## 应该怎么做

把复杂判断拆成有名字的小判断。

```python
是闰年 = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
if 是闰年:
    print("yes")
```

## 这在干什么

所有逻辑判断，本质都是判对错。复杂表达式不要硬塞进 `if`，先用变量名把它说成人话。

## 高频写法

```python
if 条件1:
    ...
elif 条件2:
    ...
else:
    ...

for 当前元素 in 数据:
    ...

while 条件:
    ...
```

## `for` 和 `while` 怎么选

| 场景 | 用 |
|---|---|
| 已知次数、遍历一组数据 | `for` |
| 不知道循环几次，只知道什么时候停 | `while` |

## `break` 与 `continue`

```python
for x in nums:
    if x < 0:
        continue      # 跳过当前这个
    if x == target:
        break         # 直接结束循环
```

## 我的提醒

你的原话：“while 循环关注的是循环条件，for 循环关注的是遍历每一个元素。”

## 易错点

- `else` 不需要条件。
- `while` 里一定要让条件有机会变化，否则死循环。
- `and` / `or` 有短路机制：左边已经决定结果，右边不执行。
