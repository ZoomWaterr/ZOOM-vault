# 图与网格入门


## BFS 最短路

```python
from collections import deque

q = deque([(sr, sc, 0)])
visited = {(sr, sc)}
while q:
    r, c, dist = q.popleft()
    for dr, dc in dirs:
        nr, nc = r + dr, c + dc
        if 0 <= nr < n and 0 <= nc < m and (nr, nc) not in visited:
            visited.add((nr, nc))
            q.append((nr, nc, dist + 1))
```

## 我容易摔在哪里

行列别反，边界判断统一写成 `0 <= nr < n and 0 <= nc < m`。
