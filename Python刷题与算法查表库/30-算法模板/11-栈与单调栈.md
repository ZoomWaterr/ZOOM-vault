# 栈与单调栈

## 快速结论

栈处理“最近相关”；单调栈处理“下一个更大/更小元素”。

## 什么时候用

- 括号匹配。
- 表达式求值。
- 下一个更大元素。
- 左右第一个比它大/小的位置。
- 柱状图最大矩形。

## 普通栈模板

```python
stack = []
stack.append(x)
top = stack.pop()
```

## 括号匹配

```python
def valid(s):
    match = {')': '(', ']': '[', '}': '{'}
    stack = []

    for ch in s:
        if ch in '([{':
            stack.append(ch)
        else:
            if not stack or stack[-1] != match[ch]:
                return False
            stack.pop()

    return not stack
```

## 单调栈：下一个更大元素

```python
n = len(nums)
ans = [-1] * n
stack = []  # 存下标，保持对应值单调递减

for i, x in enumerate(nums):
    while stack and nums[stack[-1]] < x:
        j = stack.pop()
        ans[j] = x
    stack.append(i)
```

## 复杂度

`O(n)`，每个元素最多入栈一次、出栈一次。

## 易错点

- 栈里存值还是下标要想清楚。需要回填答案时通常存下标。
- 单调递增还是递减取决于题目。
- `<` 和 `<=` 会影响重复元素处理。
