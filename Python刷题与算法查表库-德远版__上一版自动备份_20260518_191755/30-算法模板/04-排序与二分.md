# 🔎 排序 / 二分

## 🚀 快速定位

能用 `bisect` 就先用；二分答案先写 `check()`。

## 💡 一句话

排序让数据有序，二分每次砍掉一半。

## 🎯 适用场景

- 有序数组
- 找位置
- 最大值最小/最小值最大
- 排序后处理

## 🧠 原理理解

二分查找是在数组里找；二分答案是在答案范围里找。

## 📝 标准模板

```python
import bisect
pos = bisect.bisect_left(arr, x)

left, right = 1, max_answer
while left < right:
    mid = (left + right) // 2
    if check(mid):
        right = mid
    else:
        left = mid + 1
```

## ⚠️ 易错点

- 二分答案必须有单调性
- `left < right` 和 `left <= right` 别混
- mid 后边界要收缩

> [!tip] 💬 德远的笔记
> 冒泡排序里你踩过 i/j 混乱，所以排序题更要用语义化变量名。
