# 🌱 成长笔记

> 随时往里写的反思、顿悟、和值得记住的瞬间。
> 格式不限，想到什么写什么。每一条都是成长的印记。

---

## 模板

```markdown
## YYYY-MM-DD · 简短标题

**发生了什么**：（做了一道题 / 发现了一个规律 / 踩了一个坑 / 看到一个大佬的写法）

**我的感受**：（为什么值得记下来）

**行动**：（以后怎么做）
```
