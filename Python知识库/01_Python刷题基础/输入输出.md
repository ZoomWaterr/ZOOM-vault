# 输入输出

刷题里输入输出不是重点，但它会卡住很多题。先把常见格式背熟。

## 单行多个整数

```python
a, b = map(int, input().split())
nums = list(map(int, input().split()))
```

`input()` 得到字符串，`split()` 按空格切开，`map(int, ...)` 把每一段转成整数。

## 多行输入

```python
n = int(input())
arr = []
for _ in range(n):
    arr.append(int(input()))
```

`_` 表示这个循环变量不用关心，只是重复 n 次。

## 多行数组输入

```python
n, m = map(int, input().split())
grid = []
for _ in range(n):
    row = list(map(int, input().split()))
    grid.append(row)
```

## 输出列表

```python
print(*arr)
print(" ".join(map(str, arr)))
print("\n".join(map(str, arr)))
```

## 小数格式

```python
print(f"{x:.2f}")   # 保留 2 位小数
print(f"{x:.6g}")   # 6 位有效数字，自动去掉多余 0
```

## 易错点

- `input()` 永远先得到字符串。
- `map` 本身不是列表，需要 `list(...)` 才能多次使用。
- 输出数字列表时，`join` 需要先 `map(str, arr)`。
