# 排序与二分

## 先看这个

排序让数据有顺序；二分利用“有序或单调”快速缩小范围。

## 什么时候用

- 题目说第 k 大、最小、最大、排名。
- 排序后相邻关系更清楚。
- 要在有序序列里找位置。
- 答案满足单调性。

## 怎么写

```python
nums.sort()

import bisect
pos = bisect.bisect_left(nums, target)
```

## 手写二分

```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = (left + right) // 2
    if nums[mid] < target:
        left = mid + 1
    elif nums[mid] > target:
        right = mid - 1
    else:
        print(mid)
        break
```

## 我容易摔在哪里

二分最容易错边界。新手阶段优先用 `bisect` 做标准查找，手写用于理解。
