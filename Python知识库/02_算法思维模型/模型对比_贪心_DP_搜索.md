# 模型对比_贪心_DP_搜索

这三个经常都能和“选择”有关，所以容易混。

## 一句话区别

- 搜索：把选择都试一遍。
- DP：有重复子问题，把结果存起来。
- 贪心：每一步选当前最优，并证明不会后悔。

## 对比表

| 模型 | 你在做什么 | 风险 |
|---|---|---|
| 搜索 | 枚举所有可能 | 爆炸 |
| DP | 保存状态，避免重复 | 状态定义难 |
| 贪心 | 每步选局部最优 | 可能错误 |

## 判断顺序

1. 能不能暴力搜索？
2. 搜索中有没有重复状态？
3. 如果有，能不能 DP？
4. 有没有局部最优性质？
5. 如果有，能不能贪心并证明？

## 例子

爬楼梯：

- 搜索：每次走 1 或 2，枚举所有走法。
- DP：`dp[i] = dp[i-1] + dp[i-2]`。
- 贪心：不适合，因为问的是方案数。

区间选择最多不重叠：

- 贪心：按结束时间排序，常见正确策略。
- DP：也可能做，但没必要。
