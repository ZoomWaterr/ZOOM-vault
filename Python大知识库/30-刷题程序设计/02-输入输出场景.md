# 02 输入输出场景

刷题输入输出有固定形状。熟悉这些形状，会节省大量脑力。

---

## 一行一个数

```python
n = int(input())
```

## 一行多个数

```python
a, b = map(int, input().split())
```

## 一行列表

```python
nums = list(map(int, input().split()))
```

## n 行数据

```python
n = int(input())
data = []
for _ in range(n):
    data.append(int(input()))
```

## 矩阵

```python
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]
```

---

## 小结

输入输出模板不是偷懒，而是把重复劳动固定下来。刷题时不要每次重新发明读入方式。
