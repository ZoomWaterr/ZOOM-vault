# 读题拆题

## 先看这个

读题不是把文字看完，而是把题目翻译成四件事：输入是什么、要处理什么、输出什么、限制是什么。

## 读题四格

| 问题 | 你要写下来的东西 |
|:---|:---|
| 输入是什么 | 几个数、几行、是不是矩阵、有没有 EOF |
| 要干什么 | 统计、判断、排序、模拟、找最大、求和 |
| 输出什么 | 一个数、一行多个、每行一个、小数位 |
| 限制是什么 | n 多大，能不能双重循环，需不需要优化 |

## 从题意到代码

```text
题目句子：统计每个数字出现几次，输出最多的那个。
翻译动作：读列表 -> 建字典计数 -> max(count, key=count.get) -> 输出。
```

## 别想复杂

不要一边读一边写完整代码。先把动作拆成短句，再逐句变代码。
