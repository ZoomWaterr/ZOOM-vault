# 流程控制 · if / for / while

> `if` 负责分情况，`for` 负责遍历，`while` 负责“不知道要循环几次但知道什么时候停”。

## if

```python
if 条件1:
    ...
elif 条件2:
    ...
else:
    ...
```

## for

```python
for x in nums:
    print(x)

for i in range(n):
    print(i)
```

## while

```python
while 条件:
    ...
```

## 嵌套循环变量名

你的代码里出现过 `i`、`j` 混乱的风险。刷题时可以更直白一点：

```python
for 行 in range(n):
    for 列 in range(m):
...
```

或者：

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
...
```

## for-else

```python
for x in nums:
    if x == target:
print("found")
break
else:
    print("not found")
```

`else` 的意思是：循环没有被 `break` 打断，正常结束了。
