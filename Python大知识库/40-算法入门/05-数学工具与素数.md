# 05 数学工具与素数

刷题中的数学工具不需要一次学完。先掌握最常用的几个。

---

## 最大公约数和最小公倍数

```python
import math

g = math.gcd(a, b)
l = math.lcm(a, b)
```

如果不用 `math.lcm`：

```python
l = a * b // math.gcd(a, b)
```

---

## 素数判断

```python
def is_prime(n):
    if n < 2:
        return False
    i = 2
    while i * i <= n:
        if n % i == 0:
            return False
        i += 1
    return True
```

只检查到平方根，是因为如果 `n = a * b`，那么 `a` 和 `b` 至少有一个不超过平方根。

---

## 取整

```python
a // b
(a + b - 1) // b
```

第二个是正整数向上取整。

---

## 小结

数学工具的价值是减少重复造轮子。先把常用工具掌握，再慢慢补更复杂的数学。
