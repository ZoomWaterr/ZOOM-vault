# 前缀和与差分

## 快速结论

前缀和解决“区间查询”，差分解决“区间修改”。

| 问题 | 用法 |
|---|---|
| 多次查询 `[l,r]` 的和 | 前缀和 |
| 多次给 `[l,r]` 整体加减 | 差分 |
| 统计和为 k 的连续子数组个数 | 前缀和 + 哈希表 |

## 核心思想

前缀和把从开头到当前位置的累计值存起来。区间和就变成两个累计值相减。

```text
nums:  1   2   3   4   5
pre:   0   1   3   6   10  15

sum[1,3] = nums[1]+nums[2]+nums[3] = pre[4] - pre[1]
```

差分记录变化量。给 `[l,r]` 加 `c`，不用真的逐个加，只做两个标记：

```text
diff[l] += c
diff[r+1] -= c
```

最后对 diff 做前缀和，就得到真实数组。

## 一维前缀和模板

```python
nums = list(map(int, input().split()))
pre = [0]
for x in nums:
    pre.append(pre[-1] + x)

# 查询原数组下标 [l, r]
ans = pre[r + 1] - pre[l]
```

## 前缀和 + 哈希模板

统计和为 `k` 的连续子数组个数。

```python
cnt = {0: 1}
pre = 0
ans = 0

for x in nums:
    pre += x
    ans += cnt.get(pre - k, 0)
    cnt[pre] = cnt.get(pre, 0) + 1

print(ans)
```

## 差分模板

```python
n, m = map(int, input().split())
diff = [0] * (n + 1)

for _ in range(m):
    l, r, c = map(int, input().split())
    diff[l] += c
    if r + 1 < n:
        diff[r + 1] -= c

for i in range(1, n):
    diff[i] += diff[i - 1]

print(*diff[:n])
```

## 常见变体

- 二维前缀和：矩形区域求和。
- 二维差分：矩形区域批量加减。
- 前缀异或：区间异或查询。
- 前缀和 + 字典：有负数时替代滑动窗口。

## 复杂度

- 前缀和预处理 `O(n)`，每次查询 `O(1)`。
- 差分每次修改 `O(1)`，最终还原 `O(n)`。

## 易错点

- 推荐 `pre` 前面补 0，这样不用特判 `l=0`。
- 差分是 `r+1` 减，不是 `r` 减。
- 有负数时，滑动窗口不一定能处理子数组和，优先考虑前缀和 + 哈希。
