# 卡住时先看这里

tags: #刷题/流程 #个人系统

## 60 秒自救流程

1. 这题在问什么：求最值、计数、判断、输出方案？
2. 数据范围多少：能不能暴力？
3. 有没有关键词：区间、连续、排序、出现次数、最短、所有方案？
4. 打开 [[../02_算法查表说明书/01_题型识别表]]。
5. 复制对应模板，先让代码跑起来。
6. 跑样例，再跑 [[02_边界测试清单]]。

## 我不会写时最该问

- 输入怎么存？
- 我要维护什么变量？
- 每一步循环后，变量代表什么？
- 有没有重复计算？
- 这题的数据范围允许我这么写吗？

## 我的提醒

不会做不是停在原地。先写暴力，暴力能帮你确认题意；再看有没有前缀和、差分、字典、二分、公式能优化。
