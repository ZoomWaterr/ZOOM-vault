## 应该怎么做

先别背大词。DP 就是：把大问题拆成小问题，并记住小问题的答案。

## 三个问题

1. `dp[i]` 表示什么？
2. `dp[i]` 从哪里来？
3. 初始值是什么？

## 爬楼梯

```python
n = int(input())

dp = [0] * (n + 1)
dp[1] = 1
dp[2] = 2

for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])
```

## 01 背包

每件物品只能选一次，容量倒序。

```python
dp = [0] * (容量 + 1)

for 重量, 价值 in 物品:
    for c in range(容量, 重量 - 1, -1):
        dp[c] = max(dp[c], dp[c - 重量] + 价值)
```
