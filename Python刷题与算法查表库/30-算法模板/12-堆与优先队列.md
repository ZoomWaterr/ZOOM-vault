**快速结论**
堆用于动态维护最小值或最大值。Python 的 `heapq` 是小顶堆。

**什么时候用**
- 第 K 大/第 K 小。
- 持续取当前最小/最大。
- 合并多个有序序列。
- Dijkstra 最短路。

**基础模板**

```python
import heapq

h = []
for x in nums:
    heapq.heappush(h, x)

smallest = heapq.heappop(h)

```

**大顶堆**

```python
h = []
for x in nums:
    heapq.heappush(h, -x)

largest = -heapq.heappop(h)

```

**第 K 大**
维护大小为 k 的小顶堆。

```python
h = []
for x in nums:
    heapq.heappush(h, x)
    if len(h) > k:
        heapq.heappop(h)

print(h[0])  # 第 k 大

```

**复杂度**
插入和弹出 `O(log n)`，查看堆顶 `O(1)`。

**易错点**
- Python 默认小顶堆。
- 大顶堆存负数。
- Top K 问题要控制堆大小为 k。
