# 哈希表与字典计数

tags: #算法/哈希 #python/字典

## 应该怎么做

需要快速查“某个东西有没有出现过/出现几次/对应什么”，就用字典。

## 统计出现次数

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

## 两数之和

```python
seen = {}
for i, x in enumerate(nums):
    need = target - x
    if need in seen:
        print(seen[need], i)
        break
    seen[x] = i
```

## 前缀和 + 哈希

求和为 `k` 的连续子数组个数：

```python
count = {0: 1}
prefix = 0
ans = 0

for x in nums:
    prefix += x
    ans += count.get(prefix - k, 0)
    count[prefix] = count.get(prefix, 0) + 1
```

## 我的提醒

你已经会用 `count.get(i, 0) + 1`。下一步是看到“连续子数组和为 K”时，把前缀和也放进字典里统计。
