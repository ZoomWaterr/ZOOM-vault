# 02 字符串

字符串是一排字符。它和列表很像，但不能原地修改。

---

## 常用操作

| 操作 | 代码 |
|:---|:---|
| 分割 | `s.split()` |
| 去空白 | `s.strip()` |
| 替换 | `s.replace("旧", "新")` |
| 反转 | `s[::-1]` |
| 大小写 | `s.upper()` / `s.lower()` |
| 判断数字 | `s.isdigit()` |
| 判断字母 | `s.isalpha()` |

---

## 字符串方法要接回来

```python
s.strip()
```

这一行不会改变 `s`。要写：

```python
s = s.strip()
```

---

## join

```python
words = ["hello", "world"]
print(" ".join(words))
```

数字列表要先转字符串：

```python
nums = [1, 2, 3]
print(" ".join(map(str, nums)))
```

---

## 小结

字符串题的重点是：拆开、判断、变换、拼回。不要把字符串当成能随便改的列表。
