# 常用内置函数

## 先看这个

内置函数是 Python 的常用工具。刷题里优先熟练这些，比背冷门语法更值。

## 速查表

| 函数 | 用途 | 例子 |
|:---|:---|:---|
| `len` | 长度 | `len(nums)` |
| `sum` | 求和 | `sum(nums)` |
| `max/min` | 最大最小 | `max(nums)` |
| `abs` | 绝对值 | `abs(a-b)` |
| `round` | 四舍五入 | `round(x, 2)` |
| `enumerate` | 同时拿下标和值 | `for i, x in enumerate(nums)` |
| `zip` | 并排遍历 | `for a, b in zip(A, B)` |
| `range` | 生成整数序列 | `range(1, n+1)` |

## 常用组合

```python
ans = max(nums) - min(nums)

for i, x in enumerate(nums):
    print(i, x)

for a, b in zip(list1, list2):
    print(a + b)
```

## 我容易摔在哪里

`range(1, n)` 不包含 n；要到 n 就写 `range(1, n + 1)`。
