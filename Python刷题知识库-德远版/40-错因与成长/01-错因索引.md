# 错因索引

## 先看这个

以后做完一批题，不要只写“这题错了”。要把错因归到这里，让知识库越来越像你自己。

## 错因分类

| 错因 | 典型表现 | 修复入口 |
|:---|:---|:---|
| 输入读错 | 行数、矩阵、EOF 搞错 | [[../10-Python语法速查/01-输入输出]] |
| 输出格式错 | 多空格、小数位、换行 | [[../00-总入口/03-提交前30秒]] |
| 类型错 | 字符串和整数混用 | [[../10-Python语法速查/02-类型与转换]] |
| 循环错 | 下标越界、i/j 混乱 | [[../10-Python语法速查/04-流程控制]] |
| 数据结构没选对 | 用 list 查找导致慢 | [[../10-Python语法速查/08-集合与去重]] |
| 算法复杂度高 | TLE | [[../20-刷题方法与题型/03-从暴力到优化]] |
