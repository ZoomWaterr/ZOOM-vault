# set集合

集合是“不重复的一堆东西”，最常用来去重和判断存在。

## 初始化

```python
s = set()
s = {1, 2, 3}
```

注意：`{}` 是空字典，不是空集合。

## 常用操作

```python
s.add(x)
x in s
s.remove(x)    # 不存在会报错
s.discard(x)   # 不存在不报错
```

## 集合运算

```python
a & b   # 交集
a | b   # 并集
a - b   # 差集
```

## 什么时候用

- 去重。
- 快速判断某个元素是否出现过。
- 判断两个集合有没有共同元素。
