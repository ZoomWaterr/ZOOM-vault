# 集合 set

> 一堆球——不重复、无顺序、可修改。**去重和快速查存在**专用。

---

## 特点

- **无序**：不支持下标访问 `s[0]`
- **不可重复**：自动去重
- **可修改**：可以 add / remove

---

## 创建

```python
s = {1, 2, 3, 4, 5}     # 直接定义
s = set()                # 空集合
# ⚠️ s = {} 是空字典！不是空集合！
```

---

## 核心操作 — 全是 O(1)

```python
s.add(42)                # 添加
s.discard(42)            # 安全删除（不存在也不报错）
s.remove(42)             # 删除（不存在会报错）
if 42 in s:              # 判断存在，O(1)！
    print("出现过")
```

---

## 去重 — 最常见用途

```python
lst = [1, 2, 2, 3, 3, 3]
去重后 = list(set(lst))          # → [1, 2, 3]  (但顺序会乱)
```

> 注意：set 是无序的，去重后顺序不一定跟原来一样。

---

## 集合运算

```python
a = {1, 2, 3}
b = {2, 3, 4}

a & b        # → {2, 3}    交集（两个都有）
a | b        # → {1,2,3,4} 并集（合一起）
a - b        # → {1}       差集（a 有、b 没有）
```

---

## 集合推导式

```python
{表达式 for i in 列表 if 条件}
# 和列表推导式一样，只是外面换成花括号
```

---

## 为什么比列表快

```
if x in list:   → O(n)  从头扫到尾
if x in set:    → O(1)  直接定位
```

> 🔥 循环里查元素是否存在，**必须用 set 而不是 list**。`if x in list` 在循环里是 O(n²)，最常见超时原因。

---

## 💬 德远的笔记

> 集合在刷题里用得不算最多，但在「去重」和「快速判存在」这两个场景下没有替代品。最长无重复子串那道滑动窗口题里，集合用来维护窗口里有哪些字符。
>
> 最需要记住的是：**判断元素在不在，set 是 O(1)，list 是 O(n)**。数据量上万时这是「能过」和「超时」的区别。
