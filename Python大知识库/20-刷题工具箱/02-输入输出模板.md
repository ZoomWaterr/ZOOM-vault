# 📥 输入输出模板 · 复制即用

> 刷题 80% 的代码是输入输出。这些模板背熟，直接复制改。

---

## 输入模板

```python
# ═══════ 单个数字 ═══════
n = int(input())

# ═══════ 一行多个整数 ═══════
a, b = map(int, input().split())
nums = list(map(int, input().split()))

# ═══════ 一行多个浮点数 ═══════
x, y, z = map(float, input().split())

# ═══════ n 行，每行一个数 ═══════
n = int(input())
data = [int(input()) for _ in range(n)]

# ═══════ n 行，每行多个数 → 存成元组列表 ═══════
n = int(input())
coords = [tuple(map(int, input().split())) for _ in range(n)]

# ═══════ 二维矩阵 ═══════
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]

# ═══════ 读到 EOF ═══════
import sys
for line in sys.stdin:
    a, b = map(int, line.split())
    process(a, b)

# ═══════ 极速输入（n > 10⁵）═══════
import sys
raw = sys.stdin.buffer.read().split()
n = int(raw[0])
nums = list(map(int, raw[1:]))
```

## 输出模板

```python
# ═══════ 基本 ═══════
print(ans)                              # 自动换行
print(ans, end=" ")                     # 不换行

# ═══════ 列表空格分隔输出 ← 最常用 ═══════
print(*a)                               # → "1 2 3 4"  解包
print(" ".join(map(str, a)))            # → "1 2 3 4"  join

# ═══════ 每行输出一个 ═══════
for x in a:
    print(x)
# 或
print("\n".join(map(str, a)))

# ═══════ 格式控制 ═══════
print(f"{pi:.3f}")                      # → 3.142     保留 3 位
print(f"{h:02d}:{m:02d}")               # → 08:05     补零对齐
print(f"{x:.6g}")                       # 6 位有效数字，自动删末尾 0
```

## 常用组合模板

### 拆分 → 处理 → 输出

```python
lst = input().split()
for item in lst:
    processed = handle(item)
    print(processed, end=" ")
```

### 读入 → 收集 → join 输出

```python
n = int(input())
result = []
for _ in range(n):
    result.append(str(process(int(input()))))
print(" ".join(result))
```
