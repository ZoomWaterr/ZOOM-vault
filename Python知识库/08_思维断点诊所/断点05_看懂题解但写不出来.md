# 断点05_看懂题解但写不出来

## 症状

看题解觉得“哦，懂了”，关掉之后还是写不出来。

## 根因

你理解的是题解的结果，不是推导过程。

## 正确处理题解

看完题解后不要直接抄代码，按这个顺序做：

```text
1. 关掉题解
2. 用自己的话写题解思路
3. 写变量含义
4. 只看变量含义重写代码
5. 写不出时再回去看缺哪一步
```

## 题解拆解模板

```text
它为什么想到这个模型：
它省掉了哪部分重复计算：
核心变量是什么：
最关键的一行代码是什么：
```

## 训练方法

一道题解至少复现三遍：

1. 看着题解敲。
2. 关掉题解敲。
3. 第二天再敲。

第三遍能写出来，才算变成自己的。
