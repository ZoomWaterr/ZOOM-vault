# 双指针

## 一句话理解

双指针就是用两个下标一起移动，避免无意义的重复枚举。

## 常见类型

### 左右指针

```python
l, r = 0, n - 1
while l < r:
    ...
```

常见于有序数组、回文判断、两数之和。

### 快慢指针

```python
slow = 0
for fast in range(n):
    ...
```

常见于原地去重、移动元素。

## 题目信号

- 有序数组。
- 要找两个数。
- 两边往中间靠。
- 一个指针扩展，一个指针收缩。

## 例子：有序数组两数之和

```python
l, r = 0, n - 1
while l < r:
    s = nums[l] + nums[r]
    if s == target:
        print(l, r)
        break
    elif s < target:
        l += 1
    else:
        r -= 1
```

## 易错点

- 指针移动规则必须能保证不漏答案。
- 如果数组无序，通常不能直接左右指针。
- `while l < r` 还是 `while l <= r` 要看是否允许同一个位置。
