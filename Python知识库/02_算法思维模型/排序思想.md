# 排序思想

排序不是为了把数排好看，而是为了制造“有序性”。

## 排序后能带来什么

- 相同元素靠在一起。
- 可以二分。
- 可以双指针。
- 可以贪心地从小到大或从大到小处理。

## Python 写法

```python
nums.sort()
nums.sort(reverse=True)
arr = sorted(nums)
```

## 按特殊规则排序

```python
words.sort(key=len)
pairs.sort(key=lambda x: x[1])
```

## 复杂度

Python 排序一般是 `O(n log n)`。

## 题目信号

- 要比较大小顺序。
- 要找相邻差值。
- 要让相同元素聚在一起。
- 贪心题。
