# 集合 set

> 一堆球：不重复、无序。**去重和快速查存在**专用。所有操作 O(1)。

---

## 创建

```python
s = {1, 2, 3}
s = set()          # 空集合 ⚠️ {} 是空字典！
```

---

## 核心操作

```python
s.add(42)          # 添加
s.discard(42)      # 安全删除(不存在不报错)
s.remove(42)       # 删除(不存在报错)
if 42 in s:        # 判存在 — O(1)！
```

---

## 去重

```python
list(set([1,2,2,3,3,3]))   # → [1,2,3] 顺序可能乱
```

---

## 集合运算

```python
a & b    # 交集（都有）
a | b    # 并集（合一起）
a - b    # 差集（a有b没有）
```

---

## 为什么比列表快

```
if x in list:  → O(n)  从头扫
if x in set:   → O(1)  直接定位
```

> 🔥 循环里判存在，必须用set。`if x in list` 在循环里是 O(n²)。
