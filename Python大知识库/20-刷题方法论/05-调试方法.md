# 调试方法

> 调试不是乱改，是定位。

## 三层定位

1. 输入有没有读对？
2. 中间变量有没有按预期变化？
3. 输出格式是不是题目要的？

## print 调试

```python
print("nums =", nums)
print("i =", i, "ans =", ans)
```

提交前删掉调试输出。

## 缩小样例

大样例看不懂，就自己造小样例：

```text
1
2
3
```

能手算的小样例，最适合查错。
