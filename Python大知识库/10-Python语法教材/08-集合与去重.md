# 集合 set

> 集合是一堆不重复的东西，最适合去重和快速判断存在。

## 常用写法

```python
s = set()
s.add(x)

if x in s:
    ...
```

## 去重

```python
unique = list(set(nums))
```

注意：`set` 无序，如果题目要求保持原顺序，不能直接这样用。

## 集合运算

```python
a & b   # 交集
a | b   # 并集
a - b   # 差集
```

## 易错点

```python
s = {}
```

这是空字典，不是空集合。空集合要写：

```python
s = set()
```
