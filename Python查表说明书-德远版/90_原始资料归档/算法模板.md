# 前缀和

## 一句话理解

前缀和就是**从头累加**，问"前 i 个的和是多少"直接 $O(1)$ 回答。

## 图示

```
原数组：  1   2   3   4   5
前缀和：  1   3   6   10  15
         ↑   ↑   ↑   ↑   ↑
         1  1+2 3+3 6+4 10+5
```

## 代码

```python
# 原地构造前缀和
for i in range(1, len(a)):
    a[i] += a[i - 1]
```

## 什么时候用

- 多次查询区间和
- 和差分数组配合（前缀和还原差分）

## 做过

- 1501 分苹果

# 差分数组

## 一句话理解

差分数组记录的是**变化量**，前缀和把变化量**积累**成真实值。

## 核心口诀

```
在 L 位置 +C    → 表示从 L 开始，每人多 C 个
在 R+1 位置 -C  → 表示到 R 结束，R+1 之后不多
```

**"一个范围内，批量做同一件事"就是区间操作 —— 区间操作就想差分。**

## 图示

> 5 个小朋友，区间 [2, 4] 每人 +3：

```
差分数组 d：  0   0   0   0   0   0
                 ↑               ↑
             d[2] += 3       d[5] -= 3

变成：       0   0   3   0   0  -3
```

前缀和还原：

```
d:           0   0   3   0   0  -3
前缀和后：   0   0   3   3   3   0    ← 答案！
```

## 代码模板

```python
# 第一步：构建差分数组，打标记
d = [0] * (n + 2)          # +2 防止 R+1 越界
for _ in range(m):
    L, R, C = map(int, input().split())
    d[L] += C
    d[R + 1] -= C

# 第二步：前缀和还原
for i in range(1, n + 1):
    d[i] += d[i - 1]

# 第三步：输出结果（1 到 n）
print(*d[1:n + 1])
```

## 相比暴力的优势

| 方法 | 区间操作 | 最后查询 | 总复杂度 |
|:---|:---|:---|:---|
| 暴力 | $O(n)$ 每次 | $O(1)$ | $O(m \times n)$ |
| 差分 | $O(1)$ 每次 | $O(n)$ 一次 | $O(m + n)$ |

## 做过

- 1501 蓝桥杯算法提高VIP-分苹果

# 斐波那契递推（DP 入门）

## 一句话理解

当前状态 = 前面状态的组合，从小到大填表。

## 核心口诀

```
打好基础（初始条件）
填表格（从小到大递推）
查答案（直接取表里的值）
```

## 代码模板

```python
# 第 n 项斐波那契
arr = [0] * (n + 2)
arr[1] = arr[2] = 1               # 初始条件

for i in range(3, n + 1):
    arr[i] = arr[i - 1] + arr[i - 2]  # 状态转移

print(arr[n])
```

## 什么时候用

- 爬楼梯问题（每次 1 或 2 级，问多少种走法）
- 斐波那契变种
- 任何 "当前只和前几个状态有关" 的递推

## 做过

- 1257 超级楼梯
- 2809 菲波那契数列

# 字典计数

## 一句话理解

遍历一遍，用字典记每个东西出现几次。

## 代码模板

```python
count = {}
for x in 数据:
    count[x] = count.get(x, 0) + 1
```

## 找出现最多的

```python
best = max(count, key=count.get)
print(best, count[best])
```

## 判断是否超过一半

```python
for num, cnt in count.items():
    if cnt > n / 2:
        print(num)
        break
else:          # 注意这个 else 和 for 配对
    print("no")  # for 循环正常结束（没 break）时执行
```

- 2973 出现次数超过一半的数
- 2974 统计字符数

# 冒泡排序

## 一句话理解

每轮把当前最大的数「冒」到最后面。

## 图示

```
第一轮：[4, 3, 2, 1] → [3, 2, 1, 4]   4 就位
第二轮：[3, 2, 1, 4] → [2, 1, 3, 4]   3 就位
第三轮：[2, 1, 3, 4] → [1, 2, 3, 4]   完成
```

## 代码模板

```python
n = len(nums)
for i in range(n):                    # 进行 n 轮
    for j in range(n - 1 - i):        # 每轮少比一个
        if nums[j] > nums[j + 1]:     # 相邻比较
            nums[j], nums[j + 1] = nums[j + 1], nums[j]
```

## 易错点

- `range(n - 1 - i)` 中的 `-1` 是为了不让 `j+1` 越界
- 比较的是 `j` 和 `j+1`（相邻），不是 `j` 和 `i`
- 交换的两个位置也必须是 `j` 和 `j+1`

## 做过

- 2117 车厢重组

# 杨辉三角

## 一句话理解

每个数 = 左上 + 右上。

## 代码模板

```python
n = int(input())
tri = [[1] * (i + 1) for i in range(n)]

for i in range(2, n):
    for j in range(1, i):
        tri[i][j] = tri[i - 1][j - 1] + tri[i - 1][j]

for row in tri:
    print(*row)
```

# 快速判断素数

## 一句话理解

除了 1 和自身外不能被其他数整除。

## 代码模板

```python
def is_prime(n: int) -> bool:
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

## 埃氏筛（批量筛素数）

```python
n = int(input())
is_prime = [True] * (n + 1)
is_prime[0] = is_prime[1] = False

for i in range(2, int(n ** 0.5) + 1):
    if is_prime[i]:
        for j in range(i * i, n + 1, i):
            is_prime[j] = False

# 输出所有素数
for i in range(2, n + 1):
    if is_prime[i]:
        print(i, end=' ')
```

## 做过

- 1022 筛选N以内的素数
