# 二分查找与二分答案

tags: #算法/二分 #python/bisect

## 应该怎么做

有序数组查位置，用 `bisect`。答案有单调性，用二分答案。

## 标准库查找

```python
import bisect

pos = bisect.bisect_left(a, x)   # 第一个 >= x 的位置
pos = bisect.bisect_right(a, x)  # 第一个 > x 的位置
```

## 手写二分

```python
left, right = 0, len(a) - 1
while left <= right:
    mid = (left + right) // 2
    if a[mid] == target:
        print(mid)
        break
    elif a[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```

## 二分答案

题目不是让你在数组里找，而是让你找“最小可行答案”或“最大可行答案”。

```python
def check(x):
    # x 这个答案是否可行
    return True

left, right = 最小可能答案, 最大可能答案
while left < right:
    mid = (left + right) // 2
    if check(mid):
        right = mid
    else:
        left = mid + 1

print(left)
```

## 什么时候翻它

- 有序数组。
- “最大值最小”“最小值最大”。
- 答案越大越容易满足，或越小越容易满足。

## 我的提醒

二分答案最重要的不是代码，是 `check(x)` 的单调性。你得能说清楚：为什么 x 可行后，更大的 x 也可行，或更小的 x 也可行。
