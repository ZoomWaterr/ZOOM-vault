## 你的原始核心句

- input 输入的永远都是字符串类型。
- 列表是刷题最常用容器。
- 字典：需要建立“谁 -> 对应什么”的映射关系都用字典。
- 所有逻辑判断，本质都是判对错。
- `join` 的核心：分隔符 `.join(字符串列表)`。
- `max(列表, key=xxx)`：找最大最小，但标准不是数字。
