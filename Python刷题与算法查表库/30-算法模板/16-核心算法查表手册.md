# 核心算法查表手册

定位：刷题卡住时先翻这里。它不是教材顺序，而是“我现在遇到什么题，应该用什么东西，第一行代码怎么落下去”。

**统一刷题动作**

1. 先翻译题目：输入是什么，输出是什么，是否多组输入。
2. 写出暴力法：暴力的循环有几层，时间复杂度是多少。
3. 看数据范围：`n <= 10^5` 一般不能 `O(n^2)`，`n <= 20` 可以考虑状态压缩或回溯。
4. 识别结构：连续区间、顺序关系、选择关系、图关系、数学关系。
5. 把变量名定死：`left/right`、`pre`、`cnt`、`dp`、`visited`、`parent`。
6. 先跑边界：空、一个、全相同、全不同、有负数、答案不存在、答案在头尾。

**前缀和**

看到“连续子数组的和”“区间查询”“和为 k 的个数”，先想前缀和。它做的事是把一段区间的和变成两个点相减：`sum(l..r) = pre[r + 1] - pre[l]`。

```python
pre = [0]
for x in nums:
    pre.append(pre[-1] + x)

# nums[l:r+1] 的和
total = pre[r + 1] - pre[l]
```

前缀和 + 哈希用于“有多少个子数组和为 k”。关键是：当前位置前缀和是 `s`，如果以前出现过 `s - k`，说明中间那段和是 `k`。

```python
from collections import defaultdict

cnt = defaultdict(int)
cnt[0] = 1
s = 0
ans = 0

for x in nums:
    s += x
    ans += cnt[s - k]
    cnt[s] += 1
```

易错点：有负数时不能用普通滑动窗口找固定和，因为窗口扩大后和不一定变大。遇到负数，优先换成前缀和。

**差分**

看到“多次给区间 `[l, r]` 加一个数，最后输出数组”，用差分。它做的事是只记录变化发生在哪里，不每次扫整个区间。

```python
diff = [0] * (n + 1)

for l, r, val in updates:
    diff[l] += val
    if r + 1 < n:
        diff[r + 1] -= val

arr = [0] * n
cur = 0
for i in range(n):
    cur += diff[i]
    arr[i] = cur
```

易错点：题目如果是 1-indexed，读入后要统一转成 0-indexed；`diff` 通常多开一格，防止 `r + 1` 越界。

**哈希表与计数**

看到“出现次数”“是否见过”“两数之和”“分组归类”，先想 `dict`、`set`、`Counter`。哈希的价值是把查找从线性扫变成均摊 `O(1)`。

```python
from collections import Counter, defaultdict

cnt = Counter(nums)
freq = defaultdict(int)
seen = set()
```

两数之和的核心不是双循环，而是“当前数需要谁来配它”。

```python
seen = {}
for i, x in enumerate(nums):
    need = target - x
    if need in seen:
        return [seen[need], i]
    seen[x] = i
```

易错点：要不要先查再放取决于能不能用同一个元素两次。两数之和通常先查再放。

**双指针**

双指针是在同一批数据上维护两个位置，避免重复枚举。最常见两类：有序数组对撞、原地覆盖。

```python
# 有序数组两数之和
left, right = 0, len(nums) - 1
while left < right:
    s = nums[left] + nums[right]
    if s == target:
        return [left, right]
    if s < target:
        left += 1
    else:
        right -= 1
```

```python
# 原地去重，返回新长度
write = 0
for x in nums:
    if write == 0 or nums[write - 1] != x:
        nums[write] = x
        write += 1
return write
```

易错点：对撞指针通常需要有序；原地覆盖要分清 `read` 读什么、`write` 写到哪里。

**滑动窗口**

滑动窗口专门处理连续子数组/子串。核心问题只有三个：什么时候扩右边，什么时候缩左边，什么时候更新答案。

最常用区间约定：窗口是 `[left, right]` 或 `[left, right)`，一题内必须统一。

```python
left = 0
window = 0
ans = 0

for right, x in enumerate(nums):
    window += x

    while window > target:
        window -= nums[left]
        left += 1

    ans = max(ans, right - left + 1)
```

字符串频率窗口：

```python
from collections import Counter, defaultdict

need = Counter(t)
window = defaultdict(int)
valid = 0
left = 0
best = float("inf")
start = 0

for right, ch in enumerate(s):
    if ch in need:
        window[ch] += 1
        if window[ch] == need[ch]:
            valid += 1

    while valid == len(need):
        if right - left + 1 < best:
            best = right - left + 1
            start = left
        out = s[left]
        left += 1
        if out in need:
            if window[out] == need[out]:
                valid -= 1
            window[out] -= 1

ans = "" if best == float("inf") else s[start:start + best]
```

易错点：最长题通常在窗口合法后更新答案；最短覆盖题通常在收缩前更新答案。固定长度窗口可以用 `if` 缩一次，不一定要 `while`。

**二分查找**

二分不只是“数组里找数”，它是“答案空间可以被砍掉一半”。先判断：如果某个答案 `x` 可行，那么更大/更小是否也一定可行。只要有单调性，就能二分。

闭区间标准查找：

```python
def binary_search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        elif nums[mid] > target:
            right = mid - 1
        else:
            return mid
    return -1
```

左边界：

```python
def left_bound(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        elif nums[mid] > target:
            right = mid - 1
        else:
            right = mid - 1
    if left == len(nums) or nums[left] != target:
        return -1
    return left
```

二分答案：

```python
def check(x):
    # x 是否可行
    return True

left, right = 0, 10**18
while left < right:
    mid = (left + right) // 2
    if check(mid):
        right = mid
    else:
        left = mid + 1
ans = left
```

易错点：`while left <= right` 是闭区间查找；`while left < right` 更适合二分答案。不要混着写。

**DFS**

DFS 是“一条路走到底，再回来换路”。用于连通块、路径枚举、树/图遍历。写之前先定三个东西：当前点、下一步选择、访问标记。

```python
def dfs(x, y):
    if x < 0 or x >= m or y < 0 or y >= n:
        return
    if grid[x][y] != "1":
        return

    grid[x][y] = "0"
    for dx, dy in ((1, 0), (-1, 0), (0, 1), (0, -1)):
        dfs(x + dx, y + dy)
```

易错点：图里必须处理 `visited`，否则有环会死循环；网格题可以原地改格子，也可以用 `visited`。

**BFS**

BFS 是“一层一层向外扩”。看到最短步数、最少操作次数、层数扩散，优先想 BFS。

```python
from collections import deque

q = deque([(start, 0)])
visited = {start}

while q:
    node, dist = q.popleft()
    if node == target:
        return dist
    for nxt in neighbors(node):
        if nxt not in visited:
            visited.add(nxt)
            q.append((nxt, dist + 1))
```

网格 BFS：

```python
q = deque([(sx, sy, 0)])
visited = {(sx, sy)}

while q:
    x, y, d = q.popleft()
    for dx, dy in ((1, 0), (-1, 0), (0, 1), (0, -1)):
        nx, ny = x + dx, y + dy
        if 0 <= nx < m and 0 <= ny < n and (nx, ny) not in visited and grid[nx][ny] != "#":
            visited.add((nx, ny))
            q.append((nx, ny, d + 1))
```

易错点：入队时就标记 visited，不要等出队再标记，否则同一个点会被重复入队。

**回溯**

回溯是“做选择、递归、撤销选择”。看到排列、组合、子集、棋盘放置、所有路径，优先想回溯。

```python
res = []
path = []
used = [False] * len(nums)

def backtrack():
    if len(path) == len(nums):
        res.append(path[:])
        return

    for i, x in enumerate(nums):
        if used[i]:
            continue
        used[i] = True
        path.append(x)
        backtrack()
        path.pop()
        used[i] = False

backtrack()
```

组合：

```python
res = []
path = []

def backtrack(start):
    if len(path) == k:
        res.append(path[:])
        return
    for i in range(start, n + 1):
        path.append(i)
        backtrack(i + 1)
        path.pop()
```

去重排序模板：

```python
nums.sort()
for i in range(start, len(nums)):
    if i > start and nums[i] == nums[i - 1]:
        continue
```

易错点：排列用 `used`，组合用 `start`；加入答案时要 `path[:]`，不能直接放 `path`。

**动态规划**

DP 不是玄学。先问四句话：

| 问题 | 答案写在哪里 |
|---|---|
| 状态是什么 | `dp` 下标 |
| 选择是什么 | 转移来源 |
| `dp` 表示什么 | 代码注释第一句 |
| base case 是什么 | 初始化 |

一维 DP：

```python
dp = [0] * (n + 1)
dp[0] = 1

for i in range(1, n + 1):
    for choice in choices:
        if i >= choice:
            dp[i] += dp[i - choice]
```

最大子数组和：

```python
best = cur = nums[0]
for x in nums[1:]:
    cur = max(x, cur + x)
    best = max(best, cur)
```

二维路径 DP：

```python
dp = [[0] * n for _ in range(m)]
dp[0][0] = 1
for i in range(m):
    for j in range(n):
        if i == 0 and j == 0:
            continue
        if i > 0:
            dp[i][j] += dp[i - 1][j]
        if j > 0:
            dp[i][j] += dp[i][j - 1]
```

易错点：先定义 `dp[i]` 的含义再写循环。循环顺序不是装饰，它决定“能不能重复选”“有没有顺序”。

**01 背包**

每个物品只能选一次。写成一维时，容量必须倒序，防止同一个物品被重复用。

```python
dp = [0] * (capacity + 1)

for w, v in items:
    for c in range(capacity, w - 1, -1):
        dp[c] = max(dp[c], dp[c - w] + v)
```

**完全背包**

每个物品可以选多次。一维容量正序，让当前物品能继续使用。

```python
dp = [0] * (capacity + 1)

for w, v in items:
    for c in range(w, capacity + 1):
        dp[c] = max(dp[c], dp[c - w] + v)
```

恰好装满时，不能把所有状态初始化为 0。

```python
NEG = -10**18
dp = [NEG] * (capacity + 1)
dp[0] = 0
```

易错点：01 背包倒序，完全背包正序；最大值初始化负无穷，最小值初始化正无穷。

**栈与单调栈**

栈处理“最近的未解决对象”。单调栈处理“下一个更大/更小元素”。

```python
stack = []
ans = [-1] * len(nums)

for i, x in enumerate(nums):
    while stack and nums[stack[-1]] < x:
        j = stack.pop()
        ans[j] = x
    stack.append(i)
```

括号匹配：

```python
pairs = {")": "(", "]": "[", "}": "{"}
stack = []
for ch in s:
    if ch in "([{":
        stack.append(ch)
    elif not stack or stack.pop() != pairs[ch]:
        return False
return not stack
```

易错点：栈里放值还是下标取决于后面需不需要距离、位置、原数组回写。

**堆**

堆用于动态最值。Python 的 `heapq` 是小顶堆；要大顶堆就存负数。

```python
import heapq

heap = []
for x in nums:
    heapq.heappush(heap, x)
    if len(heap) > k:
        heapq.heappop(heap)

# heap 里是最大的 k 个，堆顶是第 k 大
ans = heap[0]
```

合并多个有序序列：

```python
heap = []
for i, arr in enumerate(lists):
    if arr:
        heapq.heappush(heap, (arr[0], i, 0))

res = []
while heap:
    val, i, j = heapq.heappop(heap)
    res.append(val)
    if j + 1 < len(lists[i]):
        heapq.heappush(heap, (lists[i][j + 1], i, j + 1))
```

易错点：堆里元组会按第一项、第二项继续比较；对象不可比较时要加一个编号兜底。

**并查集**

并查集处理“两个元素是不是同一组”“把两组关系合并”。核心是 `find` 找根，`union` 合并根。

```python
parent = list(range(n))
size = [1] * n

def find(x):
    while parent[x] != x:
        parent[x] = parent[parent[x]]
        x = parent[x]
    return x

def union(a, b):
    ra, rb = find(a), find(b)
    if ra == rb:
        return False
    if size[ra] < size[rb]:
        ra, rb = rb, ra
    parent[rb] = ra
    size[ra] += size[rb]
    return True
```

易错点：合并前要先找根；统计连通块时，最终要对每个点再 `find(i)` 一次压缩路径。

**图**

图题先把输入转成邻接表。无向图要加两边，有向图只加一边。

```python
from collections import defaultdict

graph = defaultdict(list)
for a, b in edges:
    graph[a].append(b)
    graph[b].append(a)
```

拓扑排序用于“课程依赖”“先后关系是否有环”。

```python
from collections import deque, defaultdict

graph = defaultdict(list)
indeg = [0] * n
for a, b in edges:
    graph[b].append(a)
    indeg[a] += 1

q = deque([i for i in range(n) if indeg[i] == 0])
order = []

while q:
    x = q.popleft()
    order.append(x)
    for y in graph[x]:
        indeg[y] -= 1
        if indeg[y] == 0:
            q.append(y)

ok = len(order) == n
```

易错点：题目说 “学 a 前必须学 b”，边通常是 `b -> a`。

**贪心**

贪心不是看起来顺手就选，而是要能说清“当前选择不会破坏未来最优”。常见可贪心信号：排序后选最早结束、维护最小代价、局部交换能证明。

区间调度：

```python
intervals.sort(key=lambda x: x[1])
ans = 0
end = -10**18

for l, r in intervals:
    if l >= end:
        ans += 1
        end = r
```

跳跃游戏：

```python
far = 0
for i, x in enumerate(nums):
    if i > far:
        return False
    far = max(far, i + x)
return True
```

易错点：贪心题最怕“只凭感觉”。如果想不出为什么局部最优能推出全局最优，先换 DP 或搜索验证小数据。

**数学工具**

最大公约数：

```python
import math
g = math.gcd(a, b)
lcm = a // g * b
```

快速幂：

```python
def fast_pow(a, b, mod):
    res = 1
    a %= mod
    while b:
        if b & 1:
            res = res * a % mod
        a = a * a % mod
        b >>= 1
    return res
```

素数判断：

```python
def is_prime(x):
    if x < 2:
        return False
    d = 2
    while d * d <= x:
        if x % d == 0:
            return False
        d += 1
    return True
```

易错点：数学题先看取模、整除、范围。中间乘法大时，Python 没问题，其他语言要防溢出。
