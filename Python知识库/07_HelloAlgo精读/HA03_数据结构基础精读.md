# HA03_数据结构基础精读

对应原书：第 3 章 数据结构。

## 本章一句话

数据结构的核心是：按什么关系组织数据，才能让后续操作更方便。

## 必须掌握

- 线性结构：数组、链表、栈、队列。
- 非线性结构：树、图、堆。
- 连续存储适合按下标访问。
- 分散存储适合灵活插入删除。

## 和 Python 的关系

| 抽象结构 | Python 常用写法 |
|---|---|
| 数组 / 列表 | `list` |
| 栈 | `list.append` + `list.pop` |
| 队列 | `collections.deque` |
| 哈希表 | `dict` / `set` |
| 堆 | `heapq` |
