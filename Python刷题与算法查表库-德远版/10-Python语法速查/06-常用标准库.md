# 常用标准库

## 快速定位

数学先 `math`，计数先 `Counter`，BFS 先 `deque`，二分先 `bisect`。

## 一句话

Python 标准库是刷题加速器，能不手搓就别手搓。

## 适用场景

- GCD/LCM
- 计数
- 队列
- 二分位置
- 堆

## 原理理解

标准库把常见底层操作封装好了，让你把精力放在题意和模型上。

## 标准模板

```python
import math, bisect, heapq
from collections import Counter, deque, defaultdict

math.gcd(a, b)
Counter(nums)
q = deque([start])
bisect.bisect_left(arr, x)
heapq.heappush(heap, x)
```

## 易错点

- `heapq` 默认小顶堆
- 大顶堆存负数
- `deque.popleft()` 比列表头删快

这不是偷懒，这是 Python 的优势。会用工具也是能力。
