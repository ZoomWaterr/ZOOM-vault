# 前缀和与差分

## 前缀和一句话

前缀和就是从头累加，让区间和查询变快。

```text
原数组：  1  2  3  4  5
前缀和：  1  3  6 10 15
```

```python
for i in range(1, len(a)):
    a[i] += a[i - 1]
```

## 差分一句话

差分记录变化量，最后用前缀和还原真实值。

核心口诀：

```text
在 L 位置 +C
在 R+1 位置 -C
```

## 差分模板

```python
n, m = map(int, input().split())
d = [0] * (n + 2)

for _ in range(m):
    L, R, C = map(int, input().split())
    d[L] += C
    d[R + 1] -= C

for i in range(1, n + 1):
    d[i] += d[i - 1]

print(*d[1:n + 1])
```

## 什么时候用

看到“一个范围内，批量做同一件事”，就想差分。
