# Python大知识库

> 一本面向刷题入门阶段的 Python 教材。
> 它不追求把所有知识都塞进来，而追求让每一章都能解释清楚：这个概念是什么、为什么要用、刷题时怎么落到代码。

---

## 本书的定位

你现在不需要一本“百科全书式 Python 手册”，也不需要一个堆满题目的仓库。

你需要的是一本能长期使用的教材：

- 前面讲语言基础，让写法稳定下来。
- 中间讲数据组织和刷题程序设计，让题目能被拆开。
- 后面讲算法入门，让你刷到新题型时有路可走。
- 最后保留学习档案，把你的真实代码、注释和错因沉淀下来。
