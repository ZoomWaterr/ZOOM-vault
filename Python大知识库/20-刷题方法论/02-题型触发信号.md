# 题型触发信号

| 看到什么 | 先想什么 | 例子 |
|:---|:---|:---|
| 统计出现次数 | 字典计数 | 字符数、超过一半 |
| 最长连续 | 当前长度 + 最长长度 | 最长连号 |
| 批量区间加 | 差分 | 分苹果 |
| 排列、交换、相邻比较 | 排序/冒泡 | 车厢重组 |
| 回文、反转 | 字符串切片 | 回文判断 |
| 质数、约数 | 数学工具 | 质数口袋 |
| 前一项推后一项 | 递推 | 超级楼梯、斐波那契 |

## 用 n 的范围倒推

| n 大小 | 常见可接受复杂度 |
|:---:|:---|
| 100 | `O(n^3)` 可能可以 |
| 2,000 | `O(n^2)` 勉强可以 |
| 100,000 | `O(n log n)` 或 `O(n)` |
| 1,000,000 | 基本要 `O(n)` |

Python 不怕写得清楚，怕无意识暴力。
