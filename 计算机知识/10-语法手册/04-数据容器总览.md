# 数据容器总览 · 五大金刚

> 五种容器各有各的用场。这张表帮你一眼判断该用谁。

---

## 一张表认全

| 类型 | 创建方式 | 能不能改 | 能不能重复 | 有序吗 | 刷题频率 |
|:---|:---|:---:|:---:|:---:|:---:|
| `list` 列表 | `[1,2,3]` `[0]*10` `[]` | ✅ | ✅ | ✅ | ⭐⭐⭐⭐⭐ |
| `dict` 字典 | `{"a":1}` `{}` `dict()` | ✅ | ❌ key 不能 | ✅ (3.7+) | ⭐⭐⭐⭐ |
| `str` 字符串 | `"hello"` `""` | ❌ | ✅ | ✅ | ⭐⭐⭐ |
| `set` 集合 | `{1,2,3}` `set()` | ✅ | ❌ | ❌ | ⭐⭐ |
| `tuple` 元组 | `(1,2,3)` `()` `tuple()` | ❌ | ✅ | ✅ | ⭐ |

---

## 一句话区分

```
列表   → 一排格子，啥都能放       [1, 2, 2]
元组   → 列表但锁死了，不能改       (1, 2, 2)
字符串 → 一排字符，不能改           "hello"
集合   → 一堆球，不重复，无顺序     {1, 2, 3}
字典   → 一本名册，名字对应数值     {"a": 1}
```

---

## 什么时候用谁

| 场景 | 用哪个 | 例子 |
|:---|:---|:---|
| 存一组数据，要改要排序 | `list` | 成绩列表、前缀和数组 |
| 统计出现次数 | `dict` | 字符频率、投票计数 |
| 建立映射关系 | `dict` | 名字→分数，数字→下标 |
| 去重 | `set` | 去掉重复元素 |
| 快速判断存不存在 | `set` 或 `dict` | "这个数出现过吗" |
| 字符串处理题 | `str` 方法 | split、find、replace |
| 存固定不变的数据 | `tuple` | 坐标、RGB值 |

## 遍历方式速查

```python
# 列表 / 字符串 / 元组：直接遍历元素
for item in 容器:
    print(item)

# 需要下标时
for idx, val in enumerate(列表):
    print(f"{idx}: {val}")

# 字典遍历
for k, v in 字典.items():
    print(f"{k} → {v}")

# 集合遍历（无序！）
for item in 集合:
    print(item)
```

## 💬 德远的笔记

> 最开始老搞混 `{}` 是空字典还是空集合。后来记住了一句话：**花括号空的是字典，集合要用 set()**。因为这个在代码里被坑过一次——想建空集合写了个 `s = {}`，结果后面 `s.add(1)` 直接报错。
>
> 还有 `[0]*5` 对数字安全，但二维列表千万别用 `[[0]*3]*3`——每行都指向同一个列表！用 `[[0]*3 for _ in range(3)]` 才对。这个坑在「车厢重组」做二维数组时候踩过。
