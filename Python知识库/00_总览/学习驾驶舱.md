# 学习驾驶舱

这个页面是每天打开知识库时最先看的地方。

## 今天做什么

先从下面选一个，不要同时开太多线。

- 如果最近代码写不顺：看 [[for循环]]、[[变量角色]]、[[下标与区间]]。
- 如果题目能写但超时：看 [[复杂度分析]]、[[暴力到优化]]。
- 如果看不出模型：看 [[题型识别总表]]、[[从题目到算法的思考流程]]。
- 如果做完题没有沉淀：看 [[如何复盘一道题]]、[[题目笔记模板]]。

## 每日最小闭环

```text
1 个知识点
2 道同类题
1 条复盘
1 个易错点
```

这比一天乱刷 10 道题更有效。

## 当前主线

你的当前主线建议是：

```text
循环和变量
→ 哈希计数
→ 前缀和 / 差分
→ 双指针 / 滑动窗口
→ 二分
→ DFS / BFS
→ DP
```
