# 🐍 语法地图 · 按我要做什么找

> 这页不是按术语背书，是把“脑子里的动作”翻译成 Python 写法。

## 输入输出

| 我想…… | 代码 |
|:---|:---|
| 读一个整数 | `n = int(input())` |
| 读一行多个整数 | `a, b = map(int, input().split())` |
| 读一行整数列表 | `nums = list(map(int, input().split()))` |
| 读 n 行矩阵 | `matrix = [list(map(int, input().split())) for _ in range(n)]` |
| 空格输出列表 | `print(*nums)` |
| 保留 2 位小数 | `print(f"{x:.2f}")` |
