# 🧭 搜索 / 回溯

## 🚀 快速定位

最短步数 BFS；所有方案回溯。

## 💡 一句话

DFS 走到底，BFS 一层层，回溯选了再撤销。

## 🎯 适用场景

- 所有方案
- 最短步数
- 路径/连通
- 排列组合

## 🧠 原理理解

labuladong 的回溯框架：路径、选择列表、结束条件。

## 📝 标准模板

```python
def backtrack(path, choices):
    if finished(path):
        ans.append(path[:])
        return
    for x in choices:
        path.append(x)
        backtrack(path, next_choices)
        path.pop()
```

## ⚠️ 易错点

- 回溯必须恢复现场
- BFS 要 visited
- DFS 注意递归深度
