# 调试方法

## 先看这个

调试不是乱改。调试是确认：程序在哪一步开始和你想的不一样。

## 三招

### print 大法

```python
print("nums =", nums)
print("i =", i, "ans =", ans)
```

### 缩小样例

把大输入缩成 3 个数、2 行矩阵、1 个边界，手算期望输出。

### 二分定位

如果代码长，先确认前半段对不对，再确认后半段。

## 常见报错

| 报错 | 大概率原因 |
|:---|:---|
| `IndexError` | 下标越界、行列写反 |
| `KeyError` | 字典 key 不存在 |
| `TypeError` | 类型混用，比如列表和整数运算 |
| `ValueError` | 输入转换失败 |
| `TLE` | 复杂度过高或输入输出太慢 |
