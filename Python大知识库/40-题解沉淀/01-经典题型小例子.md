# 经典题型小例子

> 这里放少量能说明知识点的例子，不批量堆题。

## 字典计数：超过一半

```python
n = int(input())
nums = list(map(int, input().split()))

count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

for num, cnt in count.items():
    if cnt > n / 2:
print(num)
break
else:
    print("no")
```

## 差分：区间加

```python
n, m = map(int, input().split())
d = [0] * (n + 2)

for _ in range(m):
    L, R, C = map(int, input().split())
    d[L] += C
    d[R + 1] -= C

for i in range(1, n + 1):
    d[i] += d[i - 1]

print(*d[1:n + 1])
```

## 连续长度：最长连号

```python
n = int(input())
nums = list(map(int, input().split()))

cur = 1
best = 1
for i in range(1, n):
    if nums[i] == nums[i - 1] + 1:
cur += 1
    else:
cur = 1
    best = max(best, cur)

print(best)
```
