# 前缀和

tags: #算法/前缀和 #区间

## 应该怎么做

如果题目频繁问“某一段的和”，先建前缀和。

## 这在干什么

你的原话：“前缀和就是从头累加。”

```text
原数组：  1  2  3  4  5
前缀和：  1  3  6 10 15
```

每个位置存的是“从开头到这里的总和”。

## 优秀模板：原地前缀和

```python
for i in range(1, len(a)):
    a[i] += a[i - 1]
```

查 `[l, r]`：

```python
if l == 0:
    ans = prefix[r]
else:
    ans = prefix[r] - prefix[l - 1]
```

## 更稳模板：前面补 0

```python
nums = list(map(int, input().split()))
prefix = [0]

for x in nums:
    prefix.append(prefix[-1] + x)

# 查询原数组下标 [l, r]
ans = prefix[r + 1] - prefix[l]
```

## 什么时候翻它

- 多次查询区间和。
- 连续子数组求和。
- 子矩阵求和。
- 想把一段一段重复累加变快。

## 我的提醒

前缀和解决的是“我别每次都重新从头加”。它是用一次预处理，换后面每次 O(1) 查询。
