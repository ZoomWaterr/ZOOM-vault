# P1420 最长连号
所以你真正缺的不是语法，而是这个意识：

**只要题目里出现“前一个、后一个、下一项、第几个位置”，就应该优先想到下标。**
你这次的问题体现出两个点：

1. **你会描述逻辑，但还不熟悉怎么把“逻辑关系”翻译成“下标关系”。**  
    你说“这个数和下一个数对比”，其实已经说对了。代码翻译过来就是：
    
    `num[i] 和 num[i + 1] 对比`
    
2. **你容易把“值”和“位置”混在一起。**  
    这在刚学列表时特别常见。以后可以强制自己这样命名：
    
    `for x in num: # x 表示值 for i in range(n): # i 表示下标`

# P1075 [NOIP 2012 普及组] 质因数分解
你这个感觉其实已经摸到门了：**不是你缺很多知识，而是你还没把“质因数分解”翻译成代码动作。**

这题有个非常关键的隐藏简化：

**题目已经保证 n 是两个不同质数的乘积。**

所以你不需要真的完整判断质数。你只要找到一个能整除 n 的数，另一个数就是：

`n // i`

比如：

`21 = 3 * 7`

当你找到 3 能整除 21，另一个数自然就是：

`21 // 3 = 7`

代码可以这样写：

`n = int(input()) for i in range(2, n): if n % i == 0: print(n // i) break`
