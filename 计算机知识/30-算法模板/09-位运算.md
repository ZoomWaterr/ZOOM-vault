# 位运算 ⭐⭐

---

## 速查表

| 做什么 | 怎么写 |
|:---|:---|
| 判奇偶 | `x & 1` |
| 取第k位 | `(x >> k) & 1` |
| 第k位置1 | `x \| (1 << k)` |
| 第k位置0 | `x & ~(1 << k)` |
| 翻转第k位 | `x ^ (1 << k)` |
| 去最低位1 | `x & (x-1)` — **统计1的个数核心** |
| 保留最低位1 | `x & -x` |
| 判2的幂 | `x>0 and x&(x-1)==0` |
| 乘2/除2 | `x<<1` / `x>>1` |

---

## 统计二进制中1的个数

```python
def count_bits(x):
    cnt = 0
    while x:
        x &= x-1     # 每次消最低位1
        cnt += 1
    return cnt
```

## 异或妙用

```python
# 找唯一出现一次的数（其他恰好两次）
def single(nums):
    res = 0
    for x in nums: res ^= x
    return res

# 判同号
def same_sign(a, b): return (a ^ b) >= 0
```

🎯 二进制 · 状态压缩 · "只出现一次" · 位标记
