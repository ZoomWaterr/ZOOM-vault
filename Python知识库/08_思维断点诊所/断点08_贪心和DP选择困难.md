# 断点08_贪心和DP选择困难

## 症状

看到“最大、最小、最优”，不知道是贪心还是 DP。

## 快速判断

先问：

```text
当前这一步选了以后，会不会影响后面的选择？
```

如果会影响，而且需要比较多种未来情况，偏 DP。

如果每一步都能证明选当前最优不会后悔，偏贪心。

## 对比

| 特征 | 贪心 | DP |
|---|---|---|
| 选择方式 | 每步选当前最优 | 保存多个状态 |
| 是否需要证明 | 很需要 | 需要状态转移正确 |
| 是否保存子问题 | 通常不保存 | 必须保存 |
| 常见信号 | 排序、最早结束、最小代价 | 方案数、最大最小、依赖前面 |

## 一个判断动作

尝试构造反例。

如果你说“每次选最大的”，就问：

> 有没有一种情况，先选最大的反而导致后面更差？

如果有反例，贪心就不成立。
