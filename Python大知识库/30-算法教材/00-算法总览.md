# ⚔️ 算法总览

> 算法不用提前吓自己。现在的策略：刷到哪里，学到哪里；每学一个，就沉淀成“什么时候用 + 怎么写 + 易错点”。

| 阶段 | 算法 | 现在状态 |
|:---|:---|:---|
| 基础 | 枚举与模拟 | 已经大量使用 |
| 基础 | 排序 | 已接触冒泡、排序函数 |
| 基础 | 字典计数 | 已经会写 |
| 入门算法 | 前缀和与差分 | 已接触分苹果 |
| 入门算法 | 递推/DP | 已接触斐波那契、超级楼梯 |
| 后续 | 双指针、搜索、贪心 | 刷到再系统学 |

## 学算法的正确姿势

1. 先看一句话理解。
2. 再看最小模板。
3. 最后看“我容易摔在哪里”。
