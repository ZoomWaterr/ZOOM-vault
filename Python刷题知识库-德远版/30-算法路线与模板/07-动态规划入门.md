# 动态规划入门

## 先看这个

DP 就是把大问题拆成小问题，并把小问题答案存下来。

## 什么时候用

- 当前答案依赖前面的答案。
- 暴力递归会重复计算。
- 题目问最值、方案数、可不可行。

## 三个问题

1. `dp[i]` 表示什么？
2. `dp[i]` 从哪里来？
3. 初始值是什么？

## 爬楼梯

```python
dp = [0] * (n + 1)
dp[1] = 1
dp[2] = 2
for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]
print(dp[n])
```

## 我容易摔在哪里

DP 不要先背一堆公式。先把 `dp[i]` 的含义说成人话。
