# 算法选择决策树

这是一棵做题时用的决策树。

## 第 1 问：题目数据是什么

```text
数组 / 字符串 → 看区间、统计、排序、双指针
树 → 看递归、DFS、BFS
图 → 看 DFS、BFS、最短路
状态变化 → 看搜索、DP
```

## 第 2 问：题目要什么

```text
判断存在 → set / dict / DFS
统计次数 → 哈希计数
区间查询 → 前缀和
区间修改 → 差分
最短步数 → BFS
所有方案 → 回溯
最优值 → 贪心 or DP
边界值 → 二分
```

## 第 3 问：暴力能不能过

如果能过，先写暴力，不要过度设计。

如果不能过，问：

```text
重复计算在哪里？
有无单调性？
有无有序性？
能否预处理？
能否用额外空间换时间？
```
