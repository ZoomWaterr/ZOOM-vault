# 断点07_DFS和BFS选择困难

## 症状

看到搜索题，不知道用 DFS 还是 BFS。

## 快速判断

| 题目要什么 | 优先考虑 |
|---|---|
| 能不能到达 | DFS / BFS 都可 |
| 所有路径、所有方案 | DFS / 回溯 |
| 最短步数、最少操作 | BFS |
| 连通块数量 | DFS / BFS 都可 |
| 树的递归遍历 | DFS |
| 一层一层扩散 | BFS |

## 本质区别

DFS：

```text
一条路走到底，再回头。
```

BFS：

```text
一层一层扩散。
```

## 最短路为什么用 BFS

在无权图里，BFS 第一次到达终点时，走的层数最少。

DFS 可能先走一条很长的路，不能保证第一次到达就是最短。
