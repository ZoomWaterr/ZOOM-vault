# 二分查找

## 一句话理解

二分查找是在有序范围里，每次砍掉一半。

## 标准查找

```python
left, right = 0, len(nums) - 1

while left <= right:
    mid = (left + right) // 2
    if nums[mid] == target:
        print(mid)
        break
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```

## 题目信号

- 有序数组。
- 找某个值。
- 找第一个大于等于某值的位置。
- 搜索空间有单调性。

## 易错点

- `left <= right` 是闭区间写法。
- 更新时要 `mid + 1` 或 `mid - 1`，否则可能死循环。
- 二分不只是数组，也可以二分答案，见 [[二分答案]]。
