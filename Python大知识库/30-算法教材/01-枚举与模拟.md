# 枚举与模拟

> 枚举就是把可能情况一个个试；模拟就是按题目规则一步步走。

## 什么时候用

- 数据范围小。
- 题目规则明确。
- 还没想到优化，先写暴力确认理解。

## 模板

```python
ans = 0
for x in data:
    if 满足条件:
ans += 1
print(ans)
```

## 易错点

- 循环范围少 1 或多 1。
- 变量更新顺序错。
- 嵌套循环里 `i`、`j` 混。
