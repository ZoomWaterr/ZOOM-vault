**快速结论**
数学题先手算小例子，再决定用公式、枚举还是标准库。

**GCD / LCM**

```python
import math

math.gcd(a, b)
math.lcm(a, b)

```

手写辗转相除：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

```

**素数判断**

```python
def is_prime(n):
    if n < 2:
        return False
    i = 2
    while i * i <= n:
        if n % i == 0:
            return False
        i += 1
    return True

```

**埃氏筛**

```python
def sieve(n):
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False

    i = 2
    while i * i <= n:
        if is_prime[i]:
            for j in range(i * i, n + 1, i):
                is_prime[j] = False
        i += 1

    return [i for i in range(n + 1) if is_prime[i]]

```

**快速幂**

```python
ans = pow(a, b, mod)

```

手写：

```python
def fast_pow(a, b, mod):
    ans = 1
    a %= mod
    while b:
        if b & 1:
            ans = ans * a % mod
        a = a * a % mod
        b >>= 1
    return ans

```

**数学题推导流程**

```text
手算 n=1,2,3,4,5 -> 找规律 -> 写公式 -> 用 n=6 验证 -> 写代码

```

**易错点**
- 0 和 1 不是素数。
- 模运算里除法不能直接取模。
- Python 整数不溢出，但算法仍然可能超时。
- 向上取整：正整数用 `(a + b - 1) // b`。
