# 分治

分治就是把大问题拆成几个相同类型的小问题，分别解决后合并。

## 典型结构

```python
def solve(l, r):
    if l == r:
        return 基础答案
    mid = (l + r) // 2
    left = solve(l, mid)
    right = solve(mid + 1, r)
    return merge(left, right)
```

## 常见例子

- 归并排序。
- 快速排序。
- 二分搜索。
- 构建树。

## 和递归的关系

分治通常用递归实现，但递归不一定都是分治。
