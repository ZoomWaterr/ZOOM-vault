# 习惯养成清单

## 目前最值得养的 5 个习惯

| 习惯 | 为什么 |
|:---|:---|
| 提交前测 `n=1` | 低成本防 WA |
| 输出列表用 `print(*a)` | 避免列表括号 |
| 嵌套循环用语义变量名 | 降低 i/j 混乱 |
| 数学题先手算 5 个例子 | 把模糊感变清楚 |
| 每题尝试 `solve()` | 让代码有边界 |

## 不强迫

不要一次改十个习惯。每周抓一个就够了。
