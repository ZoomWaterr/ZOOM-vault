# 数据类型与变量

> Python 的基础数据类型，以及变量命名的规则。

---

## 五种基础类型

| 类型 | 说明 | 例子 |
|:---|:---|:---|
| `int` 整数 | 整数，Python 自动大数处理 | `10`, `-5`, `2**1000` |
| `float` 浮点数 | 小数，约 15 位有效数字 | `3.14`, `-0.5` |
| `str` 字符串 | 文本，不可修改 | `"hello"`, `"我用Python"` |
| `bool` 布尔 | 真/假，本质是 1 和 0 | `True`, `False` |
| `NoneType` 空值 | 表示"空"，只有一个值 | `None` |

---

## Python 的整数永远不会溢出

```python
# C++ int 最大约 2×10⁹，long long 约 9×10¹⁸
# Python int → 无限大！内存够就能算

巨大数字 = 2 ** 1000    # 302 位数字
```

> 竞赛里算阶乘、幂次、大数都不用担心溢出，这是 Python 最大的优势之一。

---

## type() 查看类型

```python
type("hello")    # → <class 'str'>
type(10)         # → <class 'int'>
type(True)       # → <class 'bool'>
```

---

## 类型转换

```python
# str → int / float（input() 返回的是字符串，必须转换！）
int("123")       # → 123
float("3.14")    # → 3.14

# int → str
str(123)         # → "123"
f"答案是{123}"    # → "答案是123"  ← f-string 更优雅

# 注意：int() 是截断，不是四舍五入！
int(3.9)         # → 3
round(3.9)       # → 4  这才是四舍五入
```

---

## 标识符命名规则

- 由字母、数字、下划线组成
- 数字不能开头
- 不能使用关键字：`if` `else` `elif` `for` `while` `and` `or` 等
- 严格区分大小写

---

## 命名规范

- 多部分用下划线：`new_color`
- 见名知意：宁可长一点，别让人猜
- 英文字母全小写（变量名）

> 🔥 嵌套循环时禁止用 `i` `j`，用中文名：「轮数」「当前位置」「行」「列」——名字会说人话，bug 少一半。
