# 字符串 str

> 一排字符，**不可修改**。所有方法都返回新字符串。

---

## 基本操作

```python
s = "Hello World"
s[0]          # → 'H'
s[-1]         # → 'd'
len(s)        # → 11
s[::-1]       # → "dlroW olleH"  反转！

# ⚠️ 不能 s[0] = 'h' → TypeError
```

---

## 常用方法

| 方法 | 作用 | 例子 |
|:---|:---|:---|
| `.split()` | 按空格分割→列表 | `"a b".split()` → `["a","b"]` |
| `.split(",")` | 按逗号分割 | `"1,2".split(",")` |
| `.strip()` | 去两端空白 | `" hi ".strip()` → `"hi"` |
| `.rstrip(".")` | 去末尾字符 | `"hi.".rstrip(".")` → `"hi"` |
| `.replace("旧","新")` | 全部替换 | `"aba".replace("a","x")` → `"xbx"` |
| `.find("x")` | 找子串位置 | `"abc".find("b")` → `1` 找不到返-1 |
| `.upper()` `.lower()` | 大小写 | `"Hi".lower()` → `"hi"` |
| `.isalpha()` | 全字母？ | `"abc".isalpha()` → True |
| `.isdigit()` | 全数字？ | `"123".isdigit()` → True |
| `.startswith("x")` | 以x开头？ | |
| `.endswith("x")` | 以x结尾？ | |

> ⚠️ 所有方法**不改原字符串**！必须 `s = s.strip()`。

---

## join — 列表拼成字符串

```python
"-".join(["2025","01","15"])   # → "2025-01-15"
" ".join(map(str, [1,2,3]))    # → "1 2 3"  数字先转str
```

> join 是**字符串的方法**，写在分隔符上。
