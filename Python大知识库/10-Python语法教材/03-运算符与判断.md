# 运算符与判断

> 所有判断，本质都是算出 True 或 False。

## 算术运算

| 运算 | 说明 |
|:---|:---|
| `+ - *` | 加减乘 |
| `/` | 普通除法，结果是小数 |
| `//` | 整除，刷题很常用 |
| `%` | 取余 |
| `**` | 幂 |

## 比较与逻辑

```python
a == b
a != b
0 <= x <= 100

if a > 0 and b > 0:
    ...
```

## 口诀

| 写法 | 口诀 |
|:---|:---|
| `and` | 全真才真 |
| `or` | 一个真就真 |
| `not` | 取反 |

## 易错点

- `=` 是赋值，`==` 才是判断相等。
- 字符串比较按字典序：`"10" > "2"` 是错觉来源。
- 多条件别写成 `x == 1 or 2`，要写 `x == 1 or x == 2`。
