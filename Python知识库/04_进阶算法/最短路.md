# 最短路

最短路解决“从一个点到另一个点代价最小”的问题。

## 先分清图

| 情况 | 常用方法 |
|---|---|
| 无权图最短步数 | [[BFS]] |
| 边权非负 | Dijkstra |
| 有负权边 | Bellman-Ford / SPFA，入门阶段先了解 |

## Dijkstra 核心想法

每次从还没确定的点里，选当前距离最小的点扩展。

Python 常用堆优化：

```python
import heapq

dist = [float("inf")] * n
dist[start] = 0
heap = [(0, start)]

while heap:
    d, x = heapq.heappop(heap)
    if d != dist[x]:
        continue
    for y, w in graph[x]:
        if dist[y] > d + w:
            dist[y] = d + w
            heapq.heappush(heap, (dist[y], y))
```
