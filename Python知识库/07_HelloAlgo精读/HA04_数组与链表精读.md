# HA04_数组与链表精读

对应原书：第 4 章 数组与链表。

## 本章一句话

数组强在按下标访问，链表强在节点连接；Python 刷题大多数时候先把数组练熟。

## 必须掌握

- 数组的下标访问是核心优势。
- 插入删除中间元素可能移动很多数据。
- 链表通过节点连接，不适合随机访问。
- Python 的 `list` 是刷题主力。
