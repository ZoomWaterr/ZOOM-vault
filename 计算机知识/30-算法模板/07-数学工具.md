## 素数判断 ⭐

```python
def is_prime(n):
    if n < 2: return False
    for i in range(2, int(n**0.5)+1):   # 到√n即可
        if n % i == 0: return False
    return True
```
⏱️ O(√n)

## 埃氏筛 ⭐⭐

```python
def sieve(n):
    is_p = [True]*(n+1); is_p[0]=is_p[1]=False
    for i in range(2, int(n**0.5)+1):
        if is_p[i]:
            for j in range(i*i, n+1, i):   # 从i²开始
                is_p[j] = False
    return [i for i in range(2, n+1) if is_p[i]]
```
⏱️ O(n log log n)

## GCD & LCM ⭐

```python
import math
math.gcd(12, 8)          # → 4
math.lcm(12, 8)          # → 24

# 手写GCD（辗转相除）
def gcd(a, b):
    while b: a, b = b, a % b
    return a

# LCM
def lcm(a, b): return a // gcd(a, b) * b
```

## 快速幂 ⭐⭐

```python
def fast_pow(base, exp, mod):
    """(base^exp) % mod  O(log exp)"""
    res = 1
    while exp:
        if exp & 1: res = (res * base) % mod
        base = (base * base) % mod
        exp >>= 1
    return res

# Python内置：pow(base, exp, mod) 也是O(log n)
```

## 模运算 ⭐⭐

```python
MOD = 10**9 + 7
(a + b) % MOD = ((a%MOD) + (b%MOD)) % MOD
(a - b) % MOD = ((a%MOD) - (b%MOD) + MOD) % MOD   # ★减法+MOD防负数
(a * b) % MOD = ((a%MOD) * (b%MOD)) % MOD
# 除法需费马小定理求逆元：a/b ≡ a * pow(b, MOD-2, MOD)
```
