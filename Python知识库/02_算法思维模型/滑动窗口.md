# 滑动窗口

## 一句话理解

滑动窗口是在连续区间上维护一个窗口，用右指针扩展，用左指针收缩。

## 题目信号

- 连续子数组。
- 连续子串。
- 最长/最短满足条件的一段。
- 窗口内字符次数、数字和。

## 基本模板

```python
left = 0
window = {}

for right, x in enumerate(nums):
    # 1. 把 nums[right] 加进窗口
    window[x] = window.get(x, 0) + 1

    # 2. 如果窗口不合法，就移动 left
    while 窗口不合法:
        y = nums[left]
        window[y] -= 1
        left += 1

    # 3. 此时窗口合法，更新答案
    ans = max(ans, right - left + 1)
```

## 关键理解

- `right` 负责让窗口变大。
- `left` 负责让窗口恢复合法。
- 窗口永远表示 `[left, right]` 这一段。

## 易错点

- 不要忘记移出左端元素时同步修改计数。
- 最短问题通常在 `while 合法` 时更新答案并收缩。
- 最长问题通常在窗口合法后更新答案。
