# 列表与切片

**快速定位**
一组数据先 `nums = list(...)`，要反转就 `nums[::-1]`，要排序就 `nums.sort()`。

**一句话**
列表是一排格子，能按下标访问，也能排序、切片、做前缀和。

**适用场景**
- 存一组数
- 排序后处理
- 需要下标
- 前缀和/差分

**原理理解**
hello-algo 强调数组的核心是连续存储和下标访问；Python 里最常用的数组就是 list。

**标准模板**
```python
nums = [0] * n
nums.append(x)
nums.sort()
rev = nums[::-1]
grid = [ [0] * m for _ in range(n) ]
```

**易错点**
- `sort()` 改原列表，`sorted()` 返回新列表
- 二维列表不要复制同一行
- 切片前闭后开

列表是你所有算法的地基，前缀和、差分、排序、DP 都是列表在工作。
