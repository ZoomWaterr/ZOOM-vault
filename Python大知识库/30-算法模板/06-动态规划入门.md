## DP 核心思想 ⭐⭐

> 💡 把大问题拆成小问题，**从小到大填表**，后面用前面的结果，避免重复计算。

DP 和暴力的区别就像**背九九乘法表**和每次都要重新算——DP 把算过的存起来。

### 三步入门

| 步骤 | 问题 | 举例：斐波那契 |
|:---|:---|:---|
| ① 定义 | `dp[i]` 代表什么？ | `dp[i]` = 第 i 项的值 |
| ② 转移 | `dp[i]` 怎么由之前推出？ | `dp[i] = dp[i-1] + dp[i-2]` |
| ③ 初始 | `dp[0]`、`dp[1]` 是多少？ | `dp[0]=0, dp[1]=1` |

## 爬楼梯（DP 最简入门）⭐

> 每次走 1 级或 2 级，到第 n 级有几种走法？

```
dp[0] = 1   (不动)
dp[1] = 1   (走 1 级)
dp[2] = dp[1] + dp[0] = 2   (1+1 或 2)
dp[3] = dp[2] + dp[1] = 3
dp[4] = dp[3] + dp[2] = 5
...斐波那契数列
```

```python
def climb_stairs(n: int) -> int:
    if n <= 1:
        return 1
    dp = [0] * (n + 1)
    dp[0] = dp[1] = 1

    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]

    return dp[n]

# 空间优化版（只保留最近两个）
def climb_stairs_o1(n: int) -> int:
    prev2, prev1 = 1, 1
    for _ in range(2, n + 1):
        prev2, prev1 = prev1, prev2 + prev1
    return prev1
```

⏱️ O(n)

## 01 背包 ⭐⭐

> 💡 每件物品**要么拿要么不拿**。在重量限制内，求最大价值。

```
口诀：外层遍历物品，内层倒序遍历容量

物品：A(重2,价3)  B(重3,价4)  C(重4,价5)  背包容量=5

dp[j] = 容量为 j 时能获得的最大价值

关键：内层循环必须倒序！
倒序 = 每件物品只用一次（01 背包）
正序 = 每件物品可用无限次（完全背包）
```

```python
def knapsack(weights, values, capacity):
    dp = [0] * (capacity + 1)

    for w, v in zip(weights, values):
        # ★ 倒序！保证每件物品只用一次
        for j in range(capacity, w - 1, -1):
            dp[j] = max(dp[j],              # 不选
                        dp[j - w] + v)      # 选了

    return dp[capacity]

# weights=[2,3,4], values=[3,4,5], capacity=5 → 7 (A+B)
```

⏱️ O(n × C)

### ⚠️ 易错点

- 内层容量**必须倒序**！正序 = 完全背包（每件可用无限次）
- `range(capacity, w-1, -1)` 的第三个参数是 -1

## 最长递增子序列 ⭐⭐

> `dp[i]` = "以第 i 个元素结尾的最优值"

```python
def lis_length(nums):
    n = len(nums)
    dp = [1] * n                  # 每个元素自己至少构成长度 1

    for i in range(n):
        for j in range(i):        # 枚举 i 之前的所有位置
            if nums[j] < nums[i]: # 能衔接上
                dp[i] = max(dp[i], dp[j] + 1)

    return max(dp)

# [10,9,2,5,3,7,101,18] → 4: [2,3,7,101]
```

⏱️ O(n²)
