# Obsidian使用建议

这套库在 Obsidian 里建议这样用。

## 推荐入口

每次先打开：

- [[学习驾驶舱]]
- [[Python算法知识库 首页]]
- [[题型识别总表]]

## 建题目笔记

重要题可以复制 [[题目笔记模板]]。

命名建议：

```text
平台-题号-题名.md
```

例如：

```text
洛谷-P1428-小鱼比可爱.md
```

## 标签建议

```text
#复盘
#哈希计数
#前缀和
#超时
#模型误判
```

## 双链建议

题目笔记里一定要链回模型：

```md
正确模型：[[哈希计数]]
易错点：[[下标与区间]]
```

这样以后打开图谱，会看到题和知识点连起来。
