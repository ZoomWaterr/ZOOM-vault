# 📏 前缀和 / 差分

## 🚀 快速定位

区间求和用前缀和；区间批量加用差分。别混。

## 💡 一句话

前缀和管区间查询，差分管区间修改。

## 🎯 适用场景

- 多次查询区间和
- 区间批量加减
- 子数组和

## 🧠 原理理解

前缀和把过去累计起来；差分只记录变化，最后再累计还原。

## 📝 标准模板

```python
# 前缀和
pre = [0]
for x in nums:
    pre.append(pre[-1] + x)
ans = pre[r + 1] - pre[l]

# 差分
d = [0] * (n + 2)
d[L] += C
d[R + 1] -= C
```

## ⚠️ 易错点

- 差分是 R+1 减
- 前缀和下标容易错
- 有负数时子数组和常配字典
