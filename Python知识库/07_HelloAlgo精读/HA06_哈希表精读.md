对应原书：第 6 章 哈希表。

## 本章一句话

哈希表用额外空间换快速查找，是刷题里最常用的优化工具之一。

## 必须掌握

- 哈希表适合映射：`key -> value`。
- 冲突是不同 key 落到同一位置的问题。
- Python 的 `dict` 和 `set` 是高频工具。
