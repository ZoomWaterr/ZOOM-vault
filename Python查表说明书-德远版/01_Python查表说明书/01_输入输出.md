# 输入输出

tags: #python/输入输出 #刷题/高频

## 应该怎么做

先把输入变成你真正要处理的数据类型，再开始写算法。

```python
n = int(input())
a, b = map(int, input().split())
nums = list(map(int, input().split()))
```

## 这在干什么

`input()` 拿到的一定是字符串。你想算数，就必须 `int()` 或 `float()`；你想一行多个数，就先 `split()` 拆开，再 `map(int, ...)` 转换。

## 常用模板

```python
# 一行一个整数
n = int(input())

# 一行多个整数
a, b, c = map(int, input().split())

# 一行整数列表
nums = list(map(int, input().split()))

# n 行，每行一个整数
n = int(input())
nums = [int(input()) for _ in range(n)]

# n 行，每行多个整数
n = int(input())
grid = [list(map(int, input().split())) for _ in range(n)]
```

## 输出

```python
print(ans)
print(*nums)                         # 1 2 3
print(" ".join(map(str, nums)))       # 1 2 3
print("\n".join(map(str, nums)))     # 每行一个
print(f"{x:.3f}")                     # 保留 3 位小数
print(f"{x:.6g}")                     # 6 位有效数字，自动去掉多余 0
```

## 我的提醒

你的原话：“如果列表里都是数字，那么可以直接进行解包。”

```python
num = [1, 2, 3]
num.sort()
print(*num)
```

`join` 只能拼字符串。数字列表要先 `map(str, nums)`。

## 什么时候翻它

- 题目刚开始，输入格式看不顺眼。
- 输出要求“空格分隔”“每行一个”“保留小数”。
- 你脑子里冒出“我这里到底要不要 list”的时候。
