# 调试输出模板

## 打印循环变量

```python
for i in range(n):
    print("i =", i)
```

## 打印双层循环

```python
for i in range(n):
    print("当前 i =", i, "nums[i] =", nums[i])
    for j in range(i):
        print("  j =", j, "nums[j] =", nums[j])
```

## 打印窗口

```python
print("left =", left, "right =", right, "window =", nums[left:right + 1])
```

## 打印 DP

```python
print("i =", i, "dp =", dp)
```

## 规则

- 调试输出只在本地用，提交前删掉。
- 打印变量时顺便打印它的含义。
- 不要乱 print，一次只验证一个怀疑点。
