# 类型与转换

## 先看这个

        `input()` 是字符串；题目要算数，就先转成 `int` 或 `float`。

        ## 什么时候用

        读入、拼接输出、判断数字和字符串关系时。

        ## 怎么写

        ```python
        s = input()
n = int(s)
x = float(s)
text = str(n)
print(type(n))
        ```

        ## 我容易摔在哪里

        最容易错在字符串和整数混用：`'10' + 1` 会报错，`'10' > '2'` 是按字符串比，不是按数字比。

## 完整解释

| 类型 | 例子 | 刷题里常见用途 |
|:---|:---|:---|
| `int` | `10` | 计数、下标、循环次数 |
| `float` | `3.14` | 小数计算、几何题 |
| `str` | `"abc"` | 字符串处理、逐位数字 |
| `bool` | `True` | 条件判断 |
| `None` | `None` | 占位、默认值 |

## 小技巧

数字逐位处理有两条路：

```python
# 字符串路线：适合反转、逐位输出
digits = list(str(n))

# 数学路线：适合取个位、十位、百位
ge = n % 10
shi = n // 10 % 10
bai = n // 100
```

## 我容易摔在哪里

你早期笔记里很重视“input 永远是字符串”，这是对的。这个点基础，但非常高频。
