# 如何让 AI 批量复盘

## 先看这个

你不想每天逐题录入，那就做一波题后让 AI 批量分析。关键是给它清楚任务。

## 推荐提示词

```text
请分析我这一批 Python 刷题代码，不要逐题流水账。
你要输出：
1. 我已经掌握得不错的能力；
2. 高频语法问题；
3. 高频思维问题；
4. 最值得进入 Obsidian 知识库的 5 个错因；
5. 每个错因对应应该链接到哪个知识点；
6. 下阶段训练建议。
语气要白话、直接、像教练，不要堆术语。
```

## 复盘后怎么入库

- 新错因进 [[../40-错因与成长/01-错因索引]]
- 新语法坑进对应语法页
- 新算法套路进对应算法模板
- 典型代码片段进 [[../40-错因与成长/02-代码旁白精选]]
