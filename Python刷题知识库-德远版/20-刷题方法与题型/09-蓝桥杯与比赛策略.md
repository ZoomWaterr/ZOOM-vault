# 蓝桥杯与比赛策略

## 先看这个

比赛不是平时学习。比赛目标是尽量拿分，不是证明自己会最优解。

## 赛中顺序

1. 先扫全题，标出明显能做的。
2. 先拿基础分，别被难题拖住。
3. 暴力能过部分数据就先写。
4. 每题提交前至少跑样例和一个自造边界。

## Python 组提醒

- 输入输出要稳，数据大时用快读。
- 大整数是优势。
- 复杂度不够时 Python 更容易 TLE，所以要早看数据范围。
- 小题可以用内置函数快速完成。

## 个人提醒

你已经打过蓝桥杯并拿到三等奖，这说明基础执行力是有的。下一步不是盲目焦虑，而是把基础题模式练稳定。
