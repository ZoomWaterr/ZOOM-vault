# 字典与集合

## 快速定位

统计次数直接写 `count[x] = count.get(x, 0) + 1`。

## 一句话

字典管映射和次数，集合管有没有出现过。

## 适用场景

- 出现次数
- 是否出现
- 频率最高
- 两数之和

## 原理理解

字典是 key 到 value 的快速通道；集合是只保留 key 的快速通道。

## 标准模板

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

seen = set()
for x in nums:
    if x in seen:
        print("重复")
    seen.add(x)
```

## 易错点

- 只判断有没有用 set
- 需要次数/位置/映射用 dict
- `{}` 是空字典，不是空集合

你的原句已经是口诀：需要建立“谁 -> 对应什么”的映射关系都用字典。
