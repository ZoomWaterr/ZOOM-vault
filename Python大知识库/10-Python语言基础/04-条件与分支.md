# 04 条件与分支

分支结构解决的是：不同情况走不同代码。

---

## 基本结构

```python
if 条件1:
    ...
elif 条件2:
    ...
else:
    ...
```

`else` 不需要条件，它表示前面的条件都不成立。

---

## 连续比较

Python 可以写：

```python
if 0 <= x <= 100:
    ...
```

这比 `x >= 0 and x <= 100` 更像数学表达。

---

## 逻辑运算

| 写法 | 含义 |
|:---|:---|
| `and` | 全真才真 |
| `or` | 有一个真就真 |
| `not` | 取反 |

---

## 常见错误

不要写：

```python
if x == 1 or 2:
    ...
```

这不是判断 `x` 等于 1 或 2。应该写：

```python
if x == 1 or x == 2:
    ...
```

或者：

```python
if x in [1, 2]:
    ...
```

---

## 小结

分支的关键不是语法，而是分情况。写 `if` 前先问：题目到底分成哪几类情况？
