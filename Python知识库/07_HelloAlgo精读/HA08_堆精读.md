# HA08_堆精读

对应原书：第 8 章 堆。

## 本章一句话

堆是动态维护最大值或最小值的结构，Top-k 是它的典型题。

## 必须掌握

- Python `heapq` 默认小根堆。
- 入堆和出堆是 `O(log n)`。
- 堆顶查询是 `O(1)`。
- 大根堆可以用负数模拟。

## Python 小练习

1. 用 `heapq` 维护最小值。
2. 用负数实现最大堆。
3. 找数组第 k 大。

## 做题信号

- 第 k 大 / 第 k 小。
- 每次取当前最小。
- 动态维护优先级。
- 合并代价最小。
