# 枚举与模拟

tags: #算法/枚举 #算法/模拟

## 应该怎么做

数据范围小：所有可能一个个试。题目描述流程：照着流程写。

## 枚举在干什么

枚举就是“不知道答案在哪，那就一个个看”。关键是别漏、别重、别超时。

```python
ans = []
for 当前数字 in range(1, n + 1):
    if 当前数字 % 3 == 0 and 当前数字 % 10 == 7:
        ans.append(当前数字)
```

## 双重循环

你的原话：“本质就是不重不漏找出所有数对。”

```python
for 右边位置 in range(n):
    for 左边位置 in range(右边位置):
        # 处理 (左边位置, 右边位置)
```

## 模拟在干什么

模拟就是题目怎么说，你就怎么写。难点不是技巧，是结构。

## 我的提醒

模拟题先写流程，再填细节。不要上来就把所有变量揉在一起。

如果一题能暴力 AC，暴力不是罪；但你要知道它为什么能过，是因为数据范围允许。
