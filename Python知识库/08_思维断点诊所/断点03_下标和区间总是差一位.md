# 断点03_下标和区间总是差一位

## 症状

代码大方向对，但总是少算一个、多算一个，或者 `IndexError`。

## 根因

你没有统一区间规则。

## 先定规则

Python 里尽量默认使用：

```text
左闭右开：[l, r)
```

也就是包含 `l`，不包含 `r`。

## 三个固定事实

```python
range(n)
```

是 `0` 到 `n-1`。

```python
nums[l:r]
```

包含 `l`，不包含 `r`。

```python
prefix[r + 1] - prefix[l]
```

表示原数组闭区间 `[l, r]` 的和。

## 检查方法

用最小例子：

```python
nums = [10, 20, 30]
```

如果你想取 `20, 30`，切片应该是：

```python
nums[1:3]
```

不是 `nums[1:2]`。
