# 知识库使用方法

## 每天怎么用

1. 从 [[我的学习路线]] 选一个小主题。
2. 看对应知识点的“一句话理解”和“题目信号”。
3. 默写模板。
4. 做 2 道同类题。
5. 在 [[洛谷题型索引]] / [[C语言网题型索引]] / [[蓝桥云课题型索引]] 里补一行复盘。

## 不建议的用法

- 一天打开十几个知识点，只看不写。
- 直接背模板，不解释变量。
- 看完题解就算会了。
- 只追求 AC，不分析复杂度。

## 这套库的核心规则

每个知识点都尽量回答：

- 它解决什么问题？
- 题目会怎么暗示它？
- 暴力法是什么？
- 它怎么优化？
- Python 怎么写？
- 变量分别是什么意思？
- 易错点在哪里？
