# 排序与二分

## 排序

```python
nums.sort()
nums.sort(reverse=True)
new_nums = sorted(nums)
```

## 按规则排序

```python
words.sort(key=len)
pairs.sort(key=lambda x: x[1])
```

## 冒泡排序理解

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
if nums[当前位置] > nums[当前位置 + 1]:
    nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
```

## 二分先放这里

目前你还没大量刷二分，先记一句：有序、答案单调、查边界，就可能用二分。
