# 列表 list

> 列表是一排格子，刷题最常用。能存、能改、能排序、能切片。

## 常用写法

| 我想…… | 代码 |
|:---|:---|
| 空列表 | `a = []` |
| n 个 0 | `a = [0] * n` |
| 添加 | `a.append(x)` |
| 排序原列表 | `a.sort()` |
| 排序生成新列表 | `b = sorted(a)` |
| 反转 | `a[::-1]` |
| 求和 | `sum(a)` |
| 最大最小 | `max(a)` / `min(a)` |

## 列表推导式

```python
squares = [i * i for i in range(1, n + 1)]
evens = [x for x in nums if x % 2 == 0]
```

## 二维列表

```python
matrix = [[0] * m for _ in range(n)]
```

不要写：

```python
matrix = [[0] * m] * n
```

因为每一行会指向同一个列表，改一行全都跟着变。
