# 03 字典与计数

字典保存的是“谁对应什么”。

看到统计次数、出现最多、是否超过一半，优先想字典。

---

## 频率统计模板

```python
count = {}
for x in data:
    count[x] = count.get(x, 0) + 1
```

这行的含义是：

```text
如果 x 没出现过，就从 0 开始；
如果出现过，就拿原来的次数；
然后加 1。
```

---

## 遍历字典

```python
for key, value in count.items():
    print(key, value)
```

---

## 找出现最多

```python
best = max(count, key=count.get)
```

注意：`max(count)` 找的是最大的 key，不是出现次数最多。

---

## 小结

字典让你从“到处找”变成“直接查”。它是从暴力走向稳定的重要一步。
