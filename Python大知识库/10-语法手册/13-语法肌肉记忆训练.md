# 🧠 语法肌肉记忆训练

> 你现在很多卡顿不是“不懂”，而是“写法还没自动化”。这页不讲大道理，只练最常用的动作。

---

## 动作 1：输入一行整数列表

看到：一行给出若干整数。

```python
nums = list(map(int, input().split()))
```

你要形成的肌肉记忆：

```text
input()       读一行字符串
split()       拆成字符串列表
map(int, ...) 每个转整数
list(...)     收成列表
```

---

## 动作 2：空格输出列表

看到：输出一排数字，中间空格。

```python
print(*nums)
```

别写：

```python
print(nums)
```

因为它会输出 `[1, 2, 3]`，OJ 不认。

## 动作 3：统计出现次数

看到：出现次数、最多、超过一半、频率。

```python
count = {}
for x in data:
    count[x] = count.get(x, 0) + 1
```

口诀：**没有就从 0 开始，有就加 1。**

## 动作 4：字符串反转/数字反转

```python
s[::-1]
```

数字想按位处理时，先变字符串通常更爽：

```python
n = input()
print(n[::-1])
```

---

## 动作 5：保留小数

```python
print(f"{x:.2f}")
print(f"{x:.6g}")
```

`:.2f` 是固定两位小数；`:.6g` 是 6 位有效数字，会自动去掉多余 0。

---

## 动作 6：嵌套循环别让 i/j 打架

冒泡排序这种题，新手期可以写成：

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
        if nums[当前位置] > nums[当前位置 + 1]:
            nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
```

变量名会说人话，bug 就少很多。

每次刷题前花 2 分钟，把这 6 个动作手打一遍。别复制，手打。你缺的不是知道，是自动化。
