# 双指针与滑动窗口

## 快速结论

双指针用两个位置减少重复枚举；滑动窗口专门处理连续区间。

## 什么时候用

- 有序数组两数之和。
- 连续子数组、连续子串。
- 最长/最短满足条件的一段。
- 原地去重。

## 对撞指针模板

适用于有序数组。

```python
left, right = 0, len(nums) - 1
while left < right:
    s = nums[left] + nums[right]
    if s == target:
        print(left, right)
        break
    elif s < target:
        left += 1
    else:
        right -= 1
```

## 滑动窗口模板

适用于窗口扩大/缩小时状态有单调性的题。

```python
left = 0
window_sum = 0
ans = 0

for right, x in enumerate(nums):
    window_sum += x

    while window_sum > target:
        window_sum -= nums[left]
        left += 1

    ans = max(ans, right - left + 1)
```

## 最长无重复子串模板

```python
seen = set()
left = 0
ans = 0

for right, ch in enumerate(s):
    while ch in seen:
        seen.remove(s[left])
        left += 1
    seen.add(ch)
    ans = max(ans, right - left + 1)
```

## 常见变体

- 快慢指针：原地去重、链表环。
- 左右指针：有序数组两数之和。
- 滑动窗口 + 哈希：字符不重复、频率限制。

## 复杂度

通常 `O(n)`，因为左右指针都只向前走。

## 易错点

- 有负数时，窗口和不单调，滑动窗口可能失效。
- 更新答案的位置要根据题意决定。
- `left` 和 `right` 表示闭区间还是半开区间要统一。
