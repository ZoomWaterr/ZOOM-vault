# HA02_复杂度分析精读

对应原书：第 2 章 复杂度分析。

## 本章一句话

复杂度不是数学装饰，而是判断代码能不能在数据范围内跑完的工具。

## 必须掌握

- 迭代和递归都可以描述重复过程。
- 时间复杂度看增长趋势，不是精确秒数。
- 空间复杂度看额外占用。
- 数据范围能反推算法等级。

## 你要会判断

```python
for i in range(n):
    ...
```

是 `O(n)`。

```python
for i in range(n):
    for j in range(n):
        ...
```

是 `O(n^2)`。

```python
while n > 1:
    n //= 2
```

是 `O(log n)`。
