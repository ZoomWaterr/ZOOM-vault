# 基础语法 · 数据类型 + 运算符

---

## 五种基础类型

| 类型 | 例子 | 说明 |
|:---|:---|:---|
| `int` | `10`, `-5`, `2**1000` | Python整数永不溢出 |
| `float` | `3.14` | 约15位有效数字 |
| `str` | `"hello"` | 不可修改 |
| `bool` | `True`, `False` | 本质是1和0 |
| `NoneType` | `None` | 表示空 |

```python
type(x)           # 查看类型
isinstance(x, int)  # 判断类型(推荐)
```

---

## 类型转换

```python
int("123")        # → 123    ⚠️ int(3.9)→3 截断不是四舍五入！
float("3.14")     # → 3.14
str(123)          # → "123"
bool(0)           # → False  0、空串、空列表为假，其余为真
```

## 算术运算符

| 运算符 | 说明 | 注意 |
|:---|:---|:---|
| `+` `-` `*` | 加减乘 | |
| `/` | 除 | ⚠️ 永远返回float！`4/2→2.0` |
| `//` | 整除 | 整数运算靠它 |
| `%` | 取余 | 判奇偶：`n % 2` |
| `**` | 幂 | `3**2 → 9` |

## 比较与逻辑

```python
==  !=  >  <  >=  <=       # 比较
0 <= x <= 100               # Python独有：连续比较
and         # 全真才真
or          # 一真就真
not         # 取反
```

短路求值：`b != 0 and a/b > 1` — b=0时右边不执行，安全。

---

## 取整方式

| 方法 | 适用 | 例子 |
|:---|:---|:---|
| `//` | 整除，**最常用** | `sum // len` |
| `math.ceil(x)` | 向上取整 | `math.ceil(5.1)→6` |
| `math.floor(x)` | 向下取整 | `math.floor(5.9)→5` |
| `int(x)` | 截断(仅正数安全) | `int(5.9)→5` |

纯整数向上取整：`(a + b - 1) // b`

> `round()`、`abs()`、`sum()` 等 → [[03-内置函数]]

---

## 布尔值转数字

```python
int(True)   # → 1
int(False)  # → 0
c * 1       # True/False → 1/0
```

## 变量命名

- 字母、数字、下划线；数字不能开头
- 不能是关键字：`if for while and or` 等
- **嵌套循环禁止用 i/j**：用「轮数」「当前位置」「行」「列」
