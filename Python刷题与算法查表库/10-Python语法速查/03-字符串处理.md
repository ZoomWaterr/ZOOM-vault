# 字符串处理

**快速定位**
先拆：`words = s.split()`；再清：`replace/strip`；再拼：`join`。

**一句话**
字符串是一排不可修改的字符，处理它要接收新结果。

**适用场景**
- 单词、字符、回文、反转
- 清理标点
- 按长度找最长

**原理理解**
字符串方法不会修改原字符串，所有处理都像生成一个新版本。

**标准模板**
```python
s = input()
words = s.replace(".", " ").split()
longest = max(words, key=len)
print(longest)
print(" ".join(w[::-1] for w in words))
```

**易错点**
- `rstrip('.')` 只处理右端
- `join` 拼字符串列表
- 空字符串要单独想

`[::-1]`、`split`、`join`、`max(key=len)` 是你的 Python 感，继续强化。
