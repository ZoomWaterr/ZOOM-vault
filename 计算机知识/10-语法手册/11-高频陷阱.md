## 列表

### pop(下标) vs remove(值)

```python
# ❌ 想删最大值
a.pop(max(a))          # max返回99，pop(99)当成下标→IndexError
# ✅
a.remove(max(a))       # 按值删除
```

### pop 返回值

```python
# ❌ num不再是列表
num = num.pop(0)       # num变成被删的那个整数
# ✅
num.pop(0)             # 直接调，不赋值给原变量
```

### 二维列表

```python
# ❌ 三行同一个列表
bad = [[0]*5]*3
# ✅
good = [[0]*5 for _ in range(3)]
```


### 循环里 x in list

```python
# ❌ O(n²) 超时
for x in data:
    if x in lst: ...
# ✅ 先转set
s = set(lst)
```

### 遍历时删除

```python
# ❌ 漏元素
for x in lst:
    if cond: lst.remove(x)
# ✅
lst = [x for x in lst if not cond]
```

## 字符串

### 忘了接收返回值

```python
# ❌ s没变
s.strip()
# ✅
s = s.strip()
```

## 字典

### 直接用 d[key] 而不是 get

```python
# ❌ key不存在→KeyError
count[x] += 1
# ✅
count[x] = count.get(x, 0) + 1
```

### {} 是空字典不是空集合

```python
d = {}       # 空字典
s = set()    # 空集合
```

## 循环

### 嵌套循环 i/j 混乱

```python
# ❌ i是外层轮数，j才是内层位置
for i in range(n):
    for j in range(n-1-i):
        if nums[j] > nums[i]:     # 应该是 j+1！
# ✅ 用中文变量名
for 轮数 in range(n):
    for 当前位置 in range(n-1-轮数):
        if nums[当前位置] > nums[当前位置+1]:
```

## 算术

### / 和 // 搞混

```python
4 / 2    # → 2.0  float！
4 // 2   # → 2    整数
```

### round 不补零

```python
round(7, 2)     # → 7    OJ要7.00→WA！
f"{7:.2f}"      # → "7.00"  ✅
```

### int() 截断不是四舍五入

```python
int(3.9)    # → 3
round(3.9)  # → 4
```
