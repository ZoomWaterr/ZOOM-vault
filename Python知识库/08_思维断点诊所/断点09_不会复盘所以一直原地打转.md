# 断点09_不会复盘所以一直原地打转

## 症状

题做了不少，但下次遇到类似题还是不会。

## 根因

你只记录了代码，没有记录“怎么想到”。

## 复盘必须回答

```text
这题是什么模型？
我一开始为什么没看出来？
哪个变量最关键？
暴力怎么写？
优化优化了哪里？
下次看到什么信号要想到它？
```

## 最小复盘

没时间时也至少写 5 行：

```text
模型：
变量：
暴力：
优化：
信号：
```

## 不合格复盘

```text
这题要注意。
```

太空了，下次还是不会。

## 合格复盘

```text
看到“左边有多少个比当前小”，用 j in range(i) 枚举左边。
cnt 每个 i 都要重置为 0。
```

这种下次能救你。
