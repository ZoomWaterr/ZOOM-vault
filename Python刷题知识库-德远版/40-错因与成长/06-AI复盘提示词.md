# AI复盘提示词

## 代码复盘提示词

```text
我会给你一批 Python 刷题代码。请你按我的当前阶段分析：我是 Python 基础刷题期，算法系统学习刚开始。

请输出：
1. 我这批代码体现出的能力；
2. 高频语法卡点；
3. 高频思维卡点；
4. 最值得进入知识库的错因；
5. 每个错因应该链接到哪个 Obsidian 知识点；
6. 哪些代码片段适合做“个人旁白”；
7. 下一批题建议训练什么。

要求：
- 不要逐题流水账；
- 语气白话、直接、像教练；
- 不要只批评，也要指出写得好的地方；
- 输出可直接搬进 Obsidian。
```

## 新知识点扩写提示词

```text
请把这个知识点写成适合我的 Obsidian 笔记：
- 顶部 30 秒速查；
- 什么时候用；
- 怎么写；
- 为什么这样写；
- 我容易摔在哪里；
- 别想复杂；
- 相关链接和底部参考。
风格白话、专业、直接。
```
