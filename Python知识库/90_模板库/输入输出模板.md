# 输入输出模板

## 一行多个整数

```python
a, b = map(int, input().split())
```

## 一行数组

```python
nums = list(map(int, input().split()))
```

## n 行整数

```python
n = int(input())
arr = [int(input()) for _ in range(n)]
```

## n 行矩阵

```python
n, m = map(int, input().split())
grid = [list(map(int, input().split())) for _ in range(n)]
```

## 输出列表

```python
print(*arr)
print(" ".join(map(str, arr)))
```
