# 输入输出

## 先看这个

        刷题里最先要稳定的是输入输出。题会不会做另说，输入先别读歪。

        ## 什么时候用

        题目给一行、多行、矩阵、保留小数、空格分隔输出时。

        ## 怎么写

        ```python
        n = int(input())
a, b = map(int, input().split())
nums = list(map(int, input().split()))

matrix = [list(map(int, input().split())) for _ in range(n)]

print(*nums)
print(" ".join(map(str, nums)))
print(f"{x:.2f}")
        ```

        ## 我容易摔在哪里

        最容易错在：忘记 `int` 转换；把多行读成一行；输出多了逗号或列表括号；小数位不按题目要求。

## 完整解释

`input()` 每次只读一行，而且读到的是字符串。`split()` 把一行按空格拆成字符串列表，`map(int, ...)` 把每个字符串转成整数。

### 常见输入形状

```python
# 一行一个整数
n = int(input())

# 一行多个整数
a, b, c = map(int, input().split())

# 一行整数列表
nums = list(map(int, input().split()))

# n 行，每行一个整数
data = [int(input()) for _ in range(n)]

# n 行，每行多个整数
rows = [list(map(int, input().split())) for _ in range(n)]
```

### 常见输出形状

```python
print(ans)
print(a, b, c)
print(*nums)
print("\n".join(map(str, nums)))
print(f"{x:.3f}")
```
