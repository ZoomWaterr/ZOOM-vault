# 🧮 数学工具

## 🚀 快速定位

先手算 n=1,2,3,4,5，别在脑子里空想。

## 💡 一句话

数学题先小例子找规律，再写公式或工具函数。

## 🎯 适用场景

- GCD/LCM
- 素数
- 取模
- 向上取整
- 斐波那契

## 🧠 原理理解

数学优化通常是把循环计数变成公式，或把暴力判断变成性质判断。

## 📝 标准模板

```python
import math

math.gcd(a, b)
math.lcm(a, b)

def is_prime(n):
    if n < 2:
        return False
    i = 2
    while i * i <= n:
        if n % i == 0:
            return False
        i += 1
    return True
```

## ⚠️ 易错点

- 素数判断到 sqrt(n)
- 取模别乱除法
- 向上取整注意整数公式
