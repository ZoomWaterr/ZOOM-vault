# 流程控制 · if / for / while

---

## if 条件判断

```python
if 条件1:
    ...
elif 条件2:
    ...
else:
    ...
```

**真/假规则**：`0` `0.0` `""` `[]` `{}` `set()` `None` 为 False，其余为 True。

```python
if lst:       # 列表非空→True
if s:         # 字符串非空→True
0 <= x <= 100 # 连续比较，Python独有
```

---

## for — 遍历

```python
for i in range(5):              # 0,1,2,3,4
for i in range(2, 8):           # 2,3,4,5,6,7
for i in range(0, 10, 2):       # 0,2,4,6,8

for item in lst:                # 遍历元素
for i, x in enumerate(lst):     # 要下标和值
for a, b in zip(lst1, lst2):    # 同时遍历两个
```

---

## while — 条件循环

```python
while 条件:
    ...
```

for vs while：for 关注遍历每个元素，while 关注循环条件（次数未知）。

---

## break / continue / for-else

```python
break      # 跳出整个循环
continue   # 跳过本轮，下一轮

# for-else：正常跑完(没break)才执行
for x in lst:
    if x == target:
        print("找到了")
        break
else:
    print("没找到")    # ← 只有没break才执行
```

---

## 双重循环

```python
# 不重不漏找所有数对 (i > j)
for i in range(n):
    for j in range(i):
        # 这里 i 和 j 就是所有 j < i 的组合
```

> 🔥 **嵌套循环禁止用 i/j**！用「轮数」「当前位置」「行」「列」——名字会说人话。
