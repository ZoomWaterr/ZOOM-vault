# 05 二维列表与矩阵

二维列表就是列表里再放列表。它常用来表示矩阵、表格、棋盘。

---

## 读入矩阵

```python
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]
```

---

## 遍历矩阵

```python
for i in range(n):
    for j in range(m):
        print(matrix[i][j])
```

这里建议把 `i`、`j` 理解成行和列。

---

## 创建二维列表

正确写法：

```python
a = [[0] * m for _ in range(n)]
```

不推荐：

```python
a = [[0] * m] * n
```

因为每一行会指向同一个列表，改一行会影响所有行。

---

## 小结

二维题的关键是坐标感：行、列、边界。只要这三个稳住，矩阵题就不会乱。
