这题有个非常关键的隐藏简化：

**题目已经保证 n 是两个不同质数的乘积。**

所以你不需要真的完整判断质数。你只要找到一个能整除 n 的数，另一个数就是：

`n // i`

比如：

`21 = 3 * 7`

当你找到 3 能整除 21，另一个数自然就是：

`21 // 3 = 7`

代码可以这样写：

`n = int(input()) for i in range(2, n): if n % i == 0: print(n // i) break`

你这个思路其实**大方向是对的**：

`读入 n 读入所有分数 去掉一个最高分 去掉一个最低分 求平均数 保留两位小数`

问题主要出在 **Python 列表方法没分清楚**。

你的代码这里有两个核心问题：

`num = num.pop(max(num)) num = num.pop(min(num))`

**问题 1：pop() 里面放的是“下标”，不是“值”**

比如样例：

`num = [9, 5, 6, 8, 9]`

`max(num)`

得到的是 9。

然后你写：

`num.pop(9)`

Python 会理解成：

> 删除下标为 9 的元素。

但这个列表只有 5 个数，下标是：

`0 1 2 3 4`

所以 pop(9) 会直接报错。

如果你想按“值”删除，应该用：

`num.remove(max(num)) num.remove(min(num))`

remove(x) 的意思是：删除列表中第一个值为 x 的元素。

这也刚好符合题目要求：如果有多个最高分，只删除一个。

---

**问题 2：pop() 会返回被删除的那个元素**

就算你写的是：

`num = num.pop(0)`

它的意思也不是“删除之后继续让 num 保持列表”。

比如：

`num = [9, 5, 6] num = num.pop(0)`

执行完之后，num 变成了：

`9`

它不再是列表了，而是一个整数。

所以你后面再写：

`sum(num) len(num)`

也会出问题。

---

正确写法可以这样：

`n = int(input()) num = list(map(int, input().split())) num.remove(max(num)) num.remove(min(num)) ans = sum(num) / len(num) print(f"{ans:.2f}")`

这里最后也建议别用：

`print(round(ans, 2))`

因为题目要求“输出两位小数”，比如答案是 7，你要输出：

`7.00`

但 round(7, 2) 打印出来可能是：

`7`

或者：

`7.0`

而：

`f"{ans:.2f}"`

会稳定输出两位小数。

你这份代码体现出来的不足主要是三点：**知道算法流程，但对列表函数的参数含义不够熟；不太清楚函数有没有修改原列表、返回什么；格式化输出还没形成固定写法。**

不过这题你的“解题方向”是对的，真正差的是语法工具熟练度，不是脑子没思路。这个阶段很正常，多踩几次 pop/remove 这种坑，后面就记住了。
