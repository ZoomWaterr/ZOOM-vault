## 爬楼梯 ⭐

> 每次1或2级，到第n级有几种？

```python
def climb(n):
    if n <= 1: return 1
    dp = [0]*(n+1); dp[0]=dp[1]=1
    for i in range(2, n+1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 空间优化
def climb_o1(n):
    a, b = 1, 1
    for _ in range(2, n+1): a, b = b, a+b
    return b
```

## 01背包 ⭐⭐

> 每件物品要么拿要么不拿。容量限制求最大价值。

```python
def knapsack(weights, values, cap):
    dp = [0]*(cap+1)
    for w, v in zip(weights, values):
        for j in range(cap, w-1, -1):    # ★倒序！保证每件只用一次
            dp[j] = max(dp[j], dp[j-w] + v)
    return dp[cap]
```

⚠️ 内层倒序=01背包(每件一次)，正序=完全背包(无限次)

🎯 "选或不选""不超过限制""总价值最大"

## 最长递增子序列 ⭐⭐

```python
def lis(nums):
    n = len(nums)
    dp = [1]*n
    for i in range(n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j]+1)
    return max(dp)
```
⏱️ O(n²)
