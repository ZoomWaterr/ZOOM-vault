# 维护手册

## 先看这个

这个库以后会继续长。长可以，但要有规矩，否则会变乱。

## 新增内容原则

- 新知识点：放进对应主题页，不要随手新建孤岛。
- 新错因：先放 [[../40-错因与成长/01-错因索引]]，再链接到知识点。
- 新题解：只有典型题才建单题笔记。
- 新算法：先写触发信号，再写模板，不要只贴代码。

## 每篇笔记的标准

```text
先看这个
什么时候用
怎么写
为什么这样写
我容易摔在哪里
别想复杂
相关
参考
```

## 标签规则

- `python/语法`
- `刷题/流程`
- `刷题/题型`
- `算法/xxx`
- `错因/xxx`
- `状态/常忘`
- `状态/复盘`
