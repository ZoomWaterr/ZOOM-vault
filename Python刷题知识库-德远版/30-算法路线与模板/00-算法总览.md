# 算法总览

## 先看这个

算法学习不要一口吞。你现在最需要的是：看到题目时知道“这类题可能属于哪一派”。

## 路线

| 阶段 | 内容 | 目的 |
|:---|:---|:---|
| 基础题 | 枚举、模拟、排序、字符串、计数 | 把 Python 写顺 |
| 入门算法 | 前缀和、差分、双指针、二分 | 学会从暴力优化 |
| 核心算法 | 搜索、贪心、DP | 建立算法思维 |
| 数据结构 | 栈、队列、堆、并查集 | 用结构降低复杂度 |
