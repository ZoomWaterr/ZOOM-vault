# 怎么使用这个库

tags: #使用说明 #obsidian

## 最推荐的用法

左侧文件夹点进去：

1. 不会写 Python 语法：进 `01_Python查表说明书`。
2. 不知道题目该用什么算法：进 `02_算法查表说明书`。
3. 写完不确定：进 `03_我的做题系统`。
4. 想找旧代码例子：进 `04_我的题目索引`。

## 做题卡住时

打开 [[03_我的做题系统/00_卡住时先看这里]]，按 60 秒流程走。

## 用搜索时搜什么

| 想找 | 搜 |
|---|---|
| 区间求和 | `前缀和` |
| 区间批量加 | `差分` |
| 出现次数 | `字典` / `计数` |
| 最长连续 | `滑动窗口` |
| 有序查找 | `二分` |
| 最大公约数 | `gcd` / `辗转相除` |
| 输出一行列表 | `print(*nums)` / `join` |

## 标签怎么用

你不必依赖标签，但标签能辅助搜索：

- `#python/字符串`
- `#python/字典`
- `#算法/前缀和`
- `#刷题/检查`
- `#个人系统`

## 添加新笔记时

照这个顺序写：

1. 应该怎么做
2. 这在干什么
3. 优秀模板
4. 我的提醒
5. 什么时候翻它

这能保证以后你不是“读文章”，而是在查说明书。
