## collections

```python
from collections import deque, Counter, defaultdict

# deque — BFS标配，popleft是O(1)
q = deque([起点])
cur = q.popleft()

# Counter — 自动统计
c = Counter(data)
c.most_common(2)           # 前2名

# defaultdict — 不用判断key存在
cnt = defaultdict(int)     # 默认0
cnt[x] += 1                # 不用get
```

## math

```python
import math
math.gcd(12, 8)            # → 4
math.lcm(12, 8)            # → 24
math.isqrt(10)             # → 3
math.ceil(3.1)             # → 4
math.floor(3.9)            # → 3
```

## bisect / heapq / itertools

```python
import bisect
bisect.bisect_left(arr, x)  # 第一个>=x的位置

import heapq
heapq.heappush(heap, x)
heapq.heappop(heap)         # 弹出最小
# 大顶堆：存负数

from itertools import combinations, permutations, accumulate
list(combinations([1,2,3], 2))    # 组合
list(accumulate([1,2,3,4,5]))     # 前缀和
```
