# 辗转相除法

是一种用于计算两个整数a和b的最大公约数的算法

### 核心概念

1. 公约数与最大公约数：
	- 公约数：能够同时整除两个或多个整数的数。
	- 最大公约数（GCD）：两个或多个整数公约数中最大的一个。
2. 余数的性质：
	- 当我们用a除以b时，得到的余数c（即a % b）具有一个性质：a和b的公约数也是b和c的公约数。
	- 反过来，b和c的公约数也是a和b的公约数。

``` c
#include <stdio.h>
int main()
{
    int a = 18;
    int b = 24;
    int c = 0;
    while (c = a % b)
    {
        a = b;
        b = c;
    }

    printf("%d\n", b);
    return 0;
}
```
