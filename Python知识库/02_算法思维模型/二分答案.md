# 二分答案

## 一句话理解

二分答案不是在数组里找，而是在“答案范围”里找满足条件的边界。

## 题目信号

- 求最小的最大值。
- 求最大的最小值。
- 问“至少多少”“最多多少”。
- 给一个答案 `x`，能判断它行不行。

## 核心结构

```python
def check(x):
    # 返回答案 x 是否可行
    return True

left, right = 最小可能答案, 最大可能答案
while left < right:
    mid = (left + right) // 2
    if check(mid):
        right = mid
    else:
        left = mid + 1

print(left)
```

## 关键

`check(x)` 必须具有单调性：

- 如果 `x` 可行，那么比它更大的也可行。
- 或者如果 `x` 可行，那么比它更小的也可行。

没有单调性，就不能二分答案。
