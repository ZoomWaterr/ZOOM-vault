# 标准库与数学工具

> 标准库不是外挂，是 Python 自带工具箱。刷题里先掌握少数高频就够了。

## math

```python
import math

math.gcd(a, b)
math.lcm(a, b)
math.ceil(x)
math.isqrt(n)
```

## 向上取整

```python
import math
ans = math.ceil(a / b)

# 如果 a、b 都是正整数，更推荐：
ans = (a + b - 1) // b
```

## 取余与整除

```python
ge = n % 10
shi = n // 10 % 10
bai = n // 100
```

## 快速幂

```python
pow(a, b)
pow(a, b, mod)
```

## collections

目前先知道 `Counter` 存在就行。你已经会用普通字典计数，基础阶段普通字典足够。

```python
from collections import Counter
count = Counter(nums)
```
