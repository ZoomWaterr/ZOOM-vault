### 列表切片：

**`s[开始索引:结束索引:步长]`**	

~~~python
范围遵循前闭后开：
第一个为开始，省略后默认从第一个开始
第二个为结束，省略后默认到len(s)(最后元素的下一位，即包含最后一个元素)
第三为步长（含义就是走到后面的第几个）省略后默认为1 (就是一个一个来)，可正可负，就是正反方向问题
~~~



*易错点：*

**`s[:-2]`这个只有结束索引，代表到倒数第二个结束，这可不是步长为反向的2**

~~~ python
# 定义函数

	函数体
    ...

函数名(参数)
~~~

使用时注意事项：

+ 函数必须先定义，再使用
+ 函数定义时，并不会执行，只有在调用时才会执行
+ 函数中通过缩进描述归属关系

~~~ python
def reg_stu(name, age, gender, city):
    print(f"注册成功，姓名：{name}，年龄：{age}，性别：{gender}，城市：{city}")
    return {"name": name, "age": age, "gender": gender, "city": city}

#位置参数
s = reg_stu("德远","18","男","北京")

#关键字参数
a = reg_stu(name="刘德远",age="18",gender="男",city="北京")
~~~



也可设置默认参数

```python
def reg_stu(name, age, gender, city="北京"):
#这就是设置了默认城市北京，默认参数要在普通参数之后，有多个默认参数想要间隔覆盖时用关键字参数设置
```



**不定长参数——位置传递**

不定长参数的关键字传递------->使用(**kwargs)	keywordargs

~~~python
def just_do(*args，**kwargs):	#这两个都是约定俗成的关键字
    ...
 
#调用参数

#这些参数会被封装到一个元组之中 args是元组类型,后面这种键值对是字典类型，被kwargs接受
~~~

# 刷题心得：

### ***join()***

处理列表转指定字符串时，比循环打印更简洁
核心：分隔符.join(字符串列表)

要拼接字符串列表，就用 join；要拼数字列表，先 map (str) 再 join



按空格拆分→逐个处理→空格分隔输出：

~~~ python
# 模板：拆分→循环处理→空格打印
lst = input().split()  # 拆分	从输入的字符串，分成用空格拆分的列表
for item in lst:
    processed_item = 对item的处理逻辑  # 比如反转、计算、替换等
    print(processed_item, end=' ')
~~~



### 双重循环：

for i in range(n)

​	for j in range(i)

外层‘i’逐个扫描每个数

内层‘j’让‘i’和它左边所有数两两比较

本质就是不重不漏找出所有数对



### 格式输出

要求输出时用空格分割

~~~ python
print(' '.join(map(str,列表名)))

print(*列表名)

print('\n'.join(map(str,列表名)))	#每行输出一个
~~~
