# 哈希计数模板

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

## 找最多

```python
best = max(count, key=count.get)
print(best, count[best])
```

## 遍历

```python
for x, c in count.items():
    print(x, c)
```
