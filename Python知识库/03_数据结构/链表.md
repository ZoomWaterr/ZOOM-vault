# 链表

链表由一个个节点组成，每个节点知道下一个节点在哪里。

## Python 节点写法

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

## 链表和数组区别

| 结构 | 访问第 i 个 | 插入删除 |
|---|---|---|
| 数组 | 快 | 中间插删慢 |
| 链表 | 慢 | 已知位置时快 |

## 常见技巧

- 虚拟头节点 `dummy`。
- 快慢指针。
- 反转链表。

竞赛入门阶段链表不如数组常见，但它能训练指针思维。
