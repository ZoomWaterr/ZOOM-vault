# 前缀和与差分

## 先看这个

前缀和解决“多次查一段和”；差分解决“多次改一段”。

## 什么时候用

- 反复问 `[l, r]` 的和。
- 对很多区间做加减，最后问每个位置是多少。

## 前缀和怎么写

```python
pre = [0]
for x in nums:
    pre.append(pre[-1] + x)

# 查询 l 到 r，0 基下标，包含两端
ans = pre[r + 1] - pre[l]
```

## 差分怎么写

```python
diff = [0] * (n + 2)
for _ in range(m):
    l, r, c = map(int, input().split())
    diff[l] += c
    diff[r + 1] -= c

cur = 0
ans = []
for i in range(1, n + 1):
    cur += diff[i]
    ans.append(cur)
print(*ans)
```

## 我容易摔在哪里

差分最容易漏 `r + 1`，所以数组多开一位。前缀和最容易混下标，建议先统一 0 基或 1 基，不要混着来。
