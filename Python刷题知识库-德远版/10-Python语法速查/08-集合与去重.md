# 集合与去重

## 先看这个

        集合是不重复的一堆东西，最强的地方是去重和快速判断存在。

        ## 什么时候用

        题目问不同元素个数、某东西是否出现过、两个集合交并差时。

        ## 怎么写

        ```python
        s = set(nums)
if x in s:
    print("seen")

common = set(a) & set(b)
all_items = set(a) | set(b)
diff = set(a) - set(b)
        ```

        ## 我容易摔在哪里

        `{}` 是空字典，不是空集合。空集合要写 `set()`。

## 为什么比列表快

`x in list` 要一个个找，最坏 O(n)。`x in set` 平均 O(1)，数据大时差很多。

## 常见用途

```python
# 去重后数量
ans = len(set(nums))

# 判断是否有重复
has_dup = len(set(nums)) != len(nums)
```
