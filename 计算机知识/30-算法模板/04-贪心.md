# 贪心算法 ⭐⭐

> 每步做当前最好的选择，不反悔。

```python
# 排队打水：时间短的先打 → 总等待最小
times = [3, 6, 1, 4, 2]
times.sort()
total = elapsed = 0
for t in times:
    total += elapsed       # 前面人的时间 = 我的等待
    elapsed += t
print(total)               # → 20
```
⏱️ O(n log n)
