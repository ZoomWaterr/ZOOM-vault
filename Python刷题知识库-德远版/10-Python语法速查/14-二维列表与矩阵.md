# 二维列表与矩阵

## 先看这个

        二维列表就是列表里套列表。访问时先行后列：`grid[row][col]`。

        ## 什么时候用

        题目给矩阵、棋盘、地图、表格、行列变换时。

        ## 怎么写

        ```python
        grid = [list(map(int, input().split())) for _ in range(n)]

for row in range(n):
    for col in range(m):
        print(grid[row][col])
        ```

        ## 我容易摔在哪里

        最容易错在行列反了，或者用 `[[0]*m]*n` 创建导致所有行绑在一起。

## 正确创建

```python
grid = [[0] * m for _ in range(n)]
```

## 转置

```python
trans = []
for col in range(m):
    row_data = []
    for row in range(n):
        row_data.append(grid[row][col])
    trans.append(row_data)
```
