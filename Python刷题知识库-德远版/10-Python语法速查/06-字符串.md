# 字符串

## 先看这个

        字符串可以像列表一样遍历和切片，但不能原地修改。

        ## 什么时候用

        题目处理单词、字符、数字位、大小写、回文、格式清理时。

        ## 怎么写

        ```python
        s = input()
chars = list(s)
rev = s[::-1]
words = s.split()
clean = s.rstrip(".")
out = " ".join(words)
        ```

        ## 我容易摔在哪里

        字符串方法返回新字符串，不会改原来的 `s`。`rstrip('.')` 会去掉末尾连续的点，不是只去一个。

## 常用动作

```python
s.lower()
s.upper()
s.replace("-", "")
s.strip()
s.split()
" ".join(parts)
```

## 回文判断

```python
if s == s[::-1]:
    print("yes")
```

## 我容易摔在哪里

你在最长单词、单词翻转这类题里已经自然用了 `split`、`rstrip`、切片反转。这个方向很好：字符串题先想“拆分、清理、变换、拼回去”。
