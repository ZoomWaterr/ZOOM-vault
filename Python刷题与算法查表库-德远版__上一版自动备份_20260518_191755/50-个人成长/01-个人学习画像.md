# 🧑‍💻 德远 · 编程画像

## 你在哪个阶段

你已经过了“看不懂”的阶段，正在从“写不对”走向“写得稳”。现在最重要的是把知道的东西变成条件反射。

## 三个强能力

1. **你有 Python 语感**：`[::-1]`、`max(key=len)`、`dict.get`、`for-else` 都已经在用。
2. **你会自我观察**：你写的活注释很珍贵，能记录当时怎么想。
3. **你愿意造系统**：`stats.py` 说明你不是只完成任务，而是想追踪和改进。

## 三个训练点

| 问题 | 训练 |
|---|---|
| 嵌套循环索引混乱 | 中文语义变量名 |
| 数学模糊就退缩 | 小例子推公式 |
| 平铺代码不分区 | 每题至少 `solve()` |

## 一句话

> 你不是不会，你是不确定。训练的目标是让不确定变少。
