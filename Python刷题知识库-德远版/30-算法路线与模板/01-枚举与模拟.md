# 枚举与模拟

## 先看这个

枚举就是把可能情况一个个试；模拟就是按题目规则老老实实走。

## 什么时候用

- 数据范围小。
- 题目规则多，但没有明显数学公式。
- 要统计所有满足条件的情况。

## 怎么写

```python
ans = 0
for x in range(1, n + 1):
    if check(x):
        ans += 1
print(ans)
```

## 完整理解

基础题里，枚举和模拟是最重要的地基。它们不华丽，但能训练你把题意翻译成代码。

## 我容易摔在哪里

模拟题代码容易长。先写主流程，再补细节，别在一个循环里塞太多含义。
