# 从暴力到优化

> 暴力不是丢人，暴力是理解题意的第一稿。但如果 n 很大，就要想怎么少做重复事。

## 常见优化方向

| 暴力在做什么 | 优化方向 |
|:---|:---|
| 每次都重新求一段和 | 前缀和 |
| 每次给区间里每个数都加 | 差分 |
| 每次都线性查找出现次数 | 字典 / 集合 |
| 重复计算前几项 | 递推 / DP |
| 双重循环找有序答案 | 二分 / 双指针 |

## 给你的提醒

你能写暴力模拟，这是基本功。下一步不是否定暴力，而是学会问：我是不是把同一件事重复做了很多遍？
