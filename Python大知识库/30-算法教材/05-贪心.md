# 贪心

> 贪心就是每一步都做当前看起来最好的选择，但它需要证明，不是凭感觉冲。

## 什么时候可能用

- 排序后从小到大/从大到小选。
- 每一步局部最优能推出整体最优。
- 题目问最少次数、最多数量。

## 当前策略

先把贪心当作“看到再学”的章节，不提前压负担。
