# 🧮 哈希 / 计数

## 🚀 快速定位

先写字典计数模板。

## 💡 一句话

哈希表快速回答：出现过吗、出现几次、对应什么。

## 🎯 适用场景

- 统计次数
- 两数之和
- 出现最多
- 前缀和计数

## 🧠 原理理解

字典把查找从一遍遍扫描变成直接定位。

## 📝 标准模板

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

best = max(count, key=count.get)
```

## ⚠️ 易错点

- 只判断有没有用 set
- 需要次数用 dict
- key 不存在时用 get

> [!tip] 💬 德远的笔记
> 你已经掌握 `dict.get`，现在要把它升级成题型反射。
