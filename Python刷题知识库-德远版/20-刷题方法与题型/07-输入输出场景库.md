# 输入输出场景库

## 先看这个

输入输出不是小事，很多基础题错在这里。

## 场景表

| 题目格式 | 模板 |
|:---|:---|
| 一个整数 | `n = int(input())` |
| 一行两个整数 | `a, b = map(int, input().split())` |
| 一行 n 个整数 | `nums = list(map(int, input().split()))` |
| n 行一个整数 | `arr = [int(input()) for _ in range(n)]` |
| n 行多个整数 | `rows = [list(map(int, input().split())) for _ in range(n)]` |
| 读到 EOF | `for line in sys.stdin:` |
| 空格输出列表 | `print(*arr)` |
| 每行输出一个 | `print("\n".join(map(str, arr)))` |

## 先判断输入形状

读题时把输入部分圈出来，先不要看故事背景。输入形状读准，代码就稳了一半。
