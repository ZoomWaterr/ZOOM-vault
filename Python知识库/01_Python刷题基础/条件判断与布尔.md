# 条件判断与布尔

所有条件判断，本质都是得到 `True` 或 `False`。

## 基本写法

```python
if x > 0:
    print("positive")
elif x == 0:
    print("zero")
else:
    print("negative")
```

## 逻辑运算

```python
a and b   # 两个都真才真
a or b    # 有一个真就真
not a     # 取反
```

## 闰年例子

```python
leap = (year % 4 == 0 and year % 100 != 0) or year % 400 == 0
print(int(leap))
```

## 易错点

- Python 代码里冒号必须是英文 `:`。
- `=` 是赋值，`==` 才是判断相等。
- `input()` 得到字符串，和数字比较前要转类型。
