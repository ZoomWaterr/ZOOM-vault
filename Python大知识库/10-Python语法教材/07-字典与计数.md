# 字典与计数 dict

> 看到“统计出现次数”“谁对应什么”“查某个东西出现过没有”，优先想字典。

## 频率统计模板

```python
count = {}
for x in data:
    count[x] = count.get(x, 0) + 1
```

## 遍历

```python
for key, value in count.items():
    print(key, value)
```

## 找出现最多

```python
best = max(count, key=count.get)
print(best, count[best])
```

## 超过一半

```python
for num, cnt in count.items():
    if cnt > n / 2:
        print(num)
        break
else:
    print("no")
```

这个结构你已经写过，而且写得很 Python。

## 我容易摔在哪里

- `d[key]`：没有 key 会报错。
- `d.get(key, 0)`：没有 key 就给默认值。
- `max(count)` 找的是最大的 key，不是出现最多的 key。
- 出现最多要写 `max(count, key=count.get)`。
