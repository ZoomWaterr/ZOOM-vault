# 输入输出模板 · 复制即用

## 输入

```python
n = int(input())                                    # 一个整数
a, b = map(int, input().split())                    # 一行两个
nums = list(map(int, input().split()))              # 一行→列表
data = [int(input()) for _ in range(n)]             # n行每行一个
coords = [tuple(map(int, input().split())) for _ in range(n)]  # n行多个
n, m = map(int, input().split())                    # 读行列
matrix = [list(map(int, input().split())) for _ in range(n)]   # 二维矩阵

# 读到EOF / 极速输入(>10⁵)
import sys
for line in sys.stdin: a, b = map(int, line.split())  # EOF
raw = sys.stdin.buffer.read().split()                # 极速
```

## 输出

```python
print(ans)                          # 自动换行
print(*a)                           # 列表空格 → "1 2 3"
print(" ".join(map(str, a)))        # join方式
print(f"{pi:.3f}")                  # 保留3位
print(f"{h:02d}:{m:02d}")           # 补零
print("\n".join(map(str, a)))       # 每行一个
```
