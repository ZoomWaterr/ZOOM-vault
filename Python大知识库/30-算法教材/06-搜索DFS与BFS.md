# 搜索 DFS 与 BFS

> 搜索是系统地试所有可能。DFS 往深处走，BFS 一层层扩。

## DFS 触发

- 所有方案。
- 递归选择。
- 回溯。

## BFS 触发

- 最短步数。
- 一层一层扩展。
- 网格走路。
