# 03 前缀和与差分

前缀和与差分是一对很重要的入门算法。

前缀和解决“频繁查询区间和”。差分解决“频繁修改区间”。

---

## 前缀和

```text
原数组：  1  2  3  4  5
前缀和：  1  3  6 10 15
```

```python
for i in range(1, len(a)):
    a[i] += a[i - 1]
```

---

## 差分

差分记录的是变化量。

核心口诀：

```text
L 位置 +C
R+1 位置 -C
```

```python
d = [0] * (n + 2)
for _ in range(m):
    L, R, C = map(int, input().split())
    d[L] += C
    d[R + 1] -= C

for i in range(1, n + 1):
    d[i] += d[i - 1]
```

---

## 什么时候用

看到“区间里批量做同一件事”，就想差分。

---

## 小结

前缀和是把累计结果提前算好。差分是只记录变化，最后统一还原。
