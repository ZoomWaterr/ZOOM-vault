# 背包问题

**快速结论**
背包是 DP 的经典模型：在容量限制下选择物品。

**什么时候用**
- 每个物品选或不选。
- 有容量、重量、费用限制。
- 求最大价值、可行性、方案数。

**01 背包**
每件物品最多选一次，容量倒序。

```python
# items = [(weight, value), ...]
dp = [0] * (capacity + 1)

for weight, value in items:
    for c in range(capacity, weight - 1, -1):
        dp[c] = max(dp[c], dp[c - weight] + value)

print(dp[capacity])
```

**完全背包**
每件物品可以选无限次，容量正序。

```python
dp = [0] * (capacity + 1)

for weight, value in items:
    for c in range(weight, capacity + 1):
        dp[c] = max(dp[c], dp[c - weight] + value)
```

**恰好装满**
```python
NEG = -10**18
dp = [NEG] * (capacity + 1)
dp[0] = 0

for weight, value in items:
    for c in range(capacity, weight - 1, -1):
        dp[c] = max(dp[c], dp[c - weight] + value)
```

**复杂度**
`O(物品数 * 容量)`。

**易错点**
- 01 背包倒序，完全背包正序。
- 求最大价值、方案数、可行性，初始化不同。
- 容量太大时不能直接开 DP 数组。
