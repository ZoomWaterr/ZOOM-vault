# 01 程序、值与名字

程序不是一堆符号。程序是在告诉计算机：先得到哪些值，把这些值放到哪些名字里，再按什么顺序处理。

---

## 值

写在代码里的固定内容叫值。

```python
10
3.14
"hello"
True
```

刷题时最常见的值是整数、字符串和列表里的元素。

---

## 名字

变量名就是给值起名字。

```python
n = 10
name = "Python"
```

读代码时不要只看 `n`，要问：这个名字代表什么？是数量、下标、答案，还是当前状态？

---

## 变量名的质量

`i`、`j` 可以用，但嵌套循环里容易混。新手期不丢人地写清楚一点：

```python
for 行 in range(n):
    for 列 in range(m):
        ...
```

好变量名不是为了漂亮，是为了减少脑内负担。

---

## 小结

程序的第一层能力，是能看出每个名字背后的含义。写题时，如果变量名全都说不清楚，思路通常也还没真正落地。
