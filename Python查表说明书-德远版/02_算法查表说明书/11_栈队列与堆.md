# 栈、队列与堆

tags: #算法/栈 #算法/队列 #算法/堆

## 栈

后进先出。适合括号匹配、最近相关、表达式。

```python
stack = []
stack.append(x)
top = stack.pop()
```

括号匹配：

```python
def valid(s):
    mp = {")": "(", "]": "[", "}": "{"}
    stack = []
    for ch in s:
        if ch in "([{":
            stack.append(ch)
        elif not stack or stack.pop() != mp[ch]:
            return False
    return not stack
```

## 队列

先进先出。BFS 用它。

```python
from collections import deque

q = deque([start])
q.append(x)
cur = q.popleft()
```

## 堆

随时取最小值。

```python
import heapq

h = []
heapq.heappush(h, x)
x = heapq.heappop(h)
```

大顶堆存负数。

## 什么时候翻它

- 括号匹配：栈。
- 最短步数：队列/BFS。
- 第 K 大、动态最值：堆。
