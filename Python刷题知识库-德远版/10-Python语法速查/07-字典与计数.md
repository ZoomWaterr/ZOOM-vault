# 字典与计数

## 先看这个

        字典就是“一个东西对应另一个东西”。刷题里最常见用途是统计出现次数。

        ## 什么时候用

        题目问出现次数、最多、是否超过一半、每个元素对应一个值时。

        ## 怎么写

        ```python
        count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

best = max(count, key=count.get)
        ```

        ## 我容易摔在哪里

        `count[x] += 1` 第一次遇到 x 会 KeyError。新手阶段计数直接背 `get` 模板。

                ## 完整解释

                `dict` 的核心是 key-value。你拿 key 去找 value。

                ```python
                d = {"apple": 3, "banana": 5}
                print(d["apple"])
                print(d.get("orange", 0))
                ```

                ## 典型题：出现次数超过一半

                ```python
                n = int(input())
                nums = list(map(int, input().split()))

                count = {}
                for x in nums:
                    count[x] = count.get(x, 0) + 1

                for num, cnt in count.items():
                    if cnt > n / 2:
                        print(num)
                        break
                else:
                    print("no")
                ```

                ## 你的代码旁白
