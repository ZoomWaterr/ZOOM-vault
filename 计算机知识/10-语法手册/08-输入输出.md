## 输入模板

```python
# 一个数
n = int(input())

# 一行多个数
a, b = map(int, input().split())
nums = list(map(int, input().split()))

# n行每行一个数
data = [int(input()) for _ in range(n)]

# 二维矩阵
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]

# 读到EOF
import sys
for line in sys.stdin:
    a, b = map(int, line.split())
    ...

# 极速输入(>10⁵)
raw = sys.stdin.buffer.read().split()
n = int(raw[0]); nums = list(map(int, raw[1:]))
```

---

## 输出模板

```python
print(ans)                       # 自动换行
print(*a)                        # 列表空格输出 → "1 2 3"
print(" ".join(map(str, a)))     # join方式
print(f"{x:.2f}")                # 保留2位(会补零!)
print(f"{h:02d}:{m:02d}")        # 补零 → "08:05"
print("\n".join(map(str, a)))    # 每行一个
print(x, end="")                 # 不换行
```

## 拆分→处理→输出

```python
lst = input().split()
for item in lst:
    processed = handle(item)
    print(processed, end=" ")
```
