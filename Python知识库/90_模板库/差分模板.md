# 差分模板

```python
n, m = map(int, input().split())
diff = [0] * (n + 2)

for _ in range(m):
    l, r, c = map(int, input().split())
    diff[l] += c
    diff[r + 1] -= c

for i in range(1, n + 1):
    diff[i] += diff[i - 1]

print(*diff[1:n + 1])
```
