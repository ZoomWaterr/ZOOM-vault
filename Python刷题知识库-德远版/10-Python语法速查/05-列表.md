# 列表

## 先看这个

        列表就是一排可修改的数据，刷题里最常用。

        ## 什么时候用

        题目给一串数、要排序、要找最大最小、要批量处理元素时。

        ## 怎么写

        ```python
        nums = list(map(int, input().split()))
nums.append(x)
nums.sort()
best = max(nums)
rev = nums[::-1]
squares = [i * i for i in range(1, n + 1)]
        ```

        ## 我容易摔在哪里

        `sort()` 改原列表，`sorted()` 返回新列表。切片是左闭右开，`a[l:r]` 不包含 `r`。

## 常用方法

| 写法 | 用途 |
|:---|:---|
| `append(x)` | 末尾加一个 |
| `pop()` | 删除并返回末尾 |
| `sort()` | 原地排序 |
| `reverse()` | 原地反转 |
| `count(x)` | 统计 x 出现几次 |
| `index(x)` | 找 x 第一次出现位置 |

## key 很重要

```python
words = ["a", "hello", "py"]
longest = max(words, key=len)
words.sort(key=len)
```

## 别想复杂

基础题里，列表常常就是“收集起来，再统一处理”。先把数据收进来，很多题会变清楚。
