# 运算符与表达式

> 算术、比较、逻辑——三大类运算符，加上 Python 的一些特殊技巧。

---

## 算术运算符

| 运算符 | 说明 | 例子 |
|:---|:---|:---|
| `+` | 加 | `3 + 2 = 5` |
| `-` | 减 | `3 - 2 = 1` |
| `*` | 乘 | `3 * 2 = 6` |
| `/` | 除 | `3 / 2 = 1.5` — ⚠️ 永远返回 float |
| `//` | 整除（向下取整） | `3 // 2 = 1` — 整数运算靠它 |
| `%` | 取余 | `3 % 2 = 1` — 判断奇偶：`n % 2` |
| `**` | 幂 | `3 ** 2 = 9` |

> ⚠️ `/` 永远返回 float！整数运算全用 `//`。

---

## 比较运算符

返回布尔值：

| 运算符 | 说明 |
|:---|:---|
| `==` | 等于 |
| `!=` | 不等于 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

Python 独有：**连续比较**

```python
0 <= x <= 100    # 等价于 x >= 0 and x <= 100
```

## 逻辑运算符

| 运算符 | 说明 | 口诀 |
|:---|:---|:---|
| `and` | 逻辑与 | 全真才真 |
| `or` | 逻辑或 | 一真就真 |
| `not` | 逻辑非 | 取反 |

### 短路求值

```python
b != 0 and a / b > 1    # 安全！b=0 时左边为 False，右边不执行
```

---

## 布尔值转数字

```python
int(True)    # → 1
int(False)   # → 0
True * 5     # → 5    (因为 True == 1)
```

> 小技巧：`c * 1` 可以把 `True/False` 转成 `1/0`

---

## 取整方式对比

| 方法 | 适用 | 例子 |
|:---|:---|:---|
| `//` | 整数除法，**最常用** | `sum // len` |
| `math.ceil(x)` | 向上取整 | `math.ceil(5.1) → 6` |
| `math.floor(x)` | 向下取整 | `math.floor(5.9) → 5` |
| `int(x)` | 仅正数截断 | `int(5.9) → 5` — ⚠️ 不是四舍五入！ |

### 向上取整的纯整数写法

```python
(a + b - 1) // b    # 不依赖 math，a 和 b 都必须是整数
```

## 💬 德远的笔记

> 刚学 Python 时 `//` 和 `/` 老搞混。`/` 永远返回小数，就算 4/2 也是 2.0 不是 2。在某道 OJ 题里因为用了 `/` 导致输出是浮点数格式被判 WA——OJ 要的是整数格式。从那以后只要是整数运算，无脑 `//`。
>
> 还有一次在「上学迟到 (P5707)」那题用到了 `math.ceil()` 来向上取整走路时间——这个真的常用，别忘了 `import math`。
