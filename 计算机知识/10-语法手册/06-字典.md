# 字典 dict

> 名册：一个key对应一个value。**统计、映射、存关系**全靠它。所有操作几乎 O(1)。

---

## 创建 / 增 / 改 / 删

```python
d = {}                          # 空字典
d = {"张三":95, "李四":88}      # 直接写

d["王五"] = 76                  # 增或改：key不存在就加，存在就覆盖
del d["张三"]                   # 删除
```

---

## 查 — 两种方式

```python
d["张三"]               # 直接取，key不存在→KeyError
d.get("赵六", 0)        # 安全取，key不存在→返回默认值0
```

> 🔥 `d.get(key, default)` 是刷题最常用的手法。

---

## 频率统计 — 必背模板

```python
count = {}
for x in data:
    count[x] = count.get(x, 0) + 1
```

---

## 找出现最多的

```python
best = max(count, key=count.get)   # key=字典.get → 按value比大小
print(best, count[best])
```

---

## 判断是否超过一半 (附 for-else)

```python
for num, cnt in count.items():
    if cnt > n / 2:
        print(num)
        break
else:                    # ← 和 for 配对，没 break 才执行
    print("no")
```

---

## 遍历

```python
for k in d:                     # 只遍历key
for k, v in d.items():         # key+value ← 最常用
```

---

## 什么时候用字典

- "统计出现次数"
- "记录每个x对应的y"
- "找最多/最少"
- "判断某个东西出现过没有" — `if key in d:` O(1)

---

## 💬 德远的笔记

> `count.get(key, 0)` 一开始觉得别扭。2973 出现次数超过一半那题反复用了几次才记住。关键：安全取出。你要直接 `count[x] += 1`，第一次碰到这个 key 时字典里还没有，直接 KeyError。
>
> `max(count, key=count.get)` 翻译：找 value 最大的那个 key。
