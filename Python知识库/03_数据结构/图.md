# 图

图由点和边组成，适合表示“关系”。

## 邻接表

```python
graph = [[] for _ in range(n)]
for _ in range(m):
    u, v = map(int, input().split())
    graph[u].append(v)
    graph[v].append(u)  # 无向图
```

## 常见问题

- 从一个点能到哪些点：[[DFS]]
- 最短步数：[[BFS]]
- 最短路径权值：[[最短路]]
- 判断连通块。
- 拓扑排序。

## 易错点

- 点编号从 0 还是 1 开始。
- 有向图还是无向图。
- 边有没有权值。
- 是否可能重复边、自环。
