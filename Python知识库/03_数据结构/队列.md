# 队列

队列是先进先出：先进去的，先出来。

## Python 写法

```python
from collections import deque

q = deque()
q.append(x)
x = q.popleft()
```

不要用 `list.pop(0)` 做队列，数据大时会慢。

## 题目信号

- 排队。
- 层层扩散。
- 最短步数。
- BFS。

## 和 BFS 的关系

BFS 用队列保证“先到的一层先处理”，所以第一次到达终点通常就是最短步数。

见 [[BFS]]。
