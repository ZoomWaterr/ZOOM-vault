# 字符串 str

> 一排字符，**不可修改**、有序、可被 for 遍历。

---

## 特点

- **不可变**：不能写 `s[0] = 'x'`，任何修改都会产生新字符串
- **有序**：有下标，可以从 0 开始取 `s[0]`
- **可迭代**：`for ch in s:` 能遍历每个字符

---

## 基本操作

```python
s = "Hello World"
s[0]        # → 'H'      下标访问
s[-1]       # → 'd'      倒数第一个
len(s)      # → 11       长度（空格也算）
s[::-1]     # → "dlroW olleH"  反转！

# ⚠️ 不能修改！
# s[0] = 'h'  → TypeError!
```

---

## 常用方法（全部返回新字符串）

| 方法 | 作用 | 例子 | 结果 |
|:---|:---|:---|:---|
| `.split()` | 按空格分割 → 列表 | `"a b c".split()` | `["a","b","c"]` |
| `.split(",")` | 按逗号分割 | `"1,2,3".split(",")` | `["1","2","3"]` |
| `.strip()` | 去掉两端空白 | `" hi ".strip()` | `"hi"` |
| `.rstrip(".")` | 去掉右端指定字符 | `"hi.".rstrip(".")` | `"hi"` |
| `.find("x")` | 找子串位置 | `"abc".find("b")` | `1` — 找不到返回 -1 |
| `.replace("旧","新")` | 全部替换 | `"aba".replace("a","x")` | `"xbx"` |
| `.upper()` / `.lower()` | 转大小写 | `"Hi".lower()` | `"hi"` |
| `.isalpha()` | 全是字母？ | `"abc".isalpha()` | `True` |
| `.isdigit()` | 全是数字？ | `"123".isdigit()` | `True` |
| `.startswith("x")` | 以 x 开头？ | `"abc".startswith("a")` | `True` |
| `.endswith("x")` | 以 x 结尾？ | `"abc".endswith("c")` | `True` |

## join() — 把列表粘成字符串

```python
# 核心：分隔符.join(字符串列表)

"-".join(["2025", "01", "15"])    # → "2025-01-15"
" ".join(["I", "am"])             # → "I am"

# 数字列表要先转字符串
" ".join(map(str, [1, 2, 3]))     # → "1 2 3"
```

> `join()` 是**字符串的方法**，不是列表的方法。写在分隔符上：`"分隔符".join(列表)`

---

## ⚠️ 关键提醒

所有字符串方法**都不修改原字符串**！必须用变量接收返回值：

```python
s = s.strip()    # ✅ 正确
s.strip()        # ❌ 错误！返回值被丢弃，原变量没变
```

除了 `join()` 之外，其他方法都是**调在字符串上**的。

## 💬 德远的笔记

> 在「最长单词2 (2961)」那题里踩过一个坑：用 `s.rstrip(".")` 只去掉末尾**一个**句号。如果句子是 `"hello... world..."`，它只去掉最后一个点。OJ 样例刚好只有一个句号所以 AC 了——但换个样例就 WA。更稳的做法是 `s.rstrip(".")` 再加循环，或者用 `replace`。
>
> 还有字符串反转 `s[::-1]` 真的太好用了！回文判断一行就完事：`s == s[::-1]`。在 P1217 回文质数里直接用上了。
>
> 另外 `split()` 不带参数是按**任意空白**（空格、制表符、换行）分割，但多个空格会被当成一个。带参数 `split(" ")` 的话每个空格都是分隔符。
