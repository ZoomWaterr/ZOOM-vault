# 🧠 动态规划

## 🚀 快速定位

先写三件事：状态定义、转移来源、初始值。

## 💡 一句话

DP 是把大问题拆成小问题，并记住答案。

## 🎯 适用场景

- 选或不选
- 最优值
- 方案数
- 爬楼梯/背包

## 🧠 原理理解

labuladong 的 DP 四问：状态、选择、base case、状态转移。

## 📝 标准模板

```python
dp = [0] * (n + 1)
dp[1] = 1
dp[2] = 2
for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]
```

## ⚠️ 易错点

- `dp[i]` 含义说不清就别写代码
- 初始值容易漏
- 01 背包容量倒序
