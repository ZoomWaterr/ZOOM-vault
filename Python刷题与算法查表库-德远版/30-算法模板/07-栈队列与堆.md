# 📦 栈 / 队列 / 堆

## 🚀 快速定位

括号用栈，最短路用队列，动态最值用堆。

## 💡 一句话

栈管最近，队列管顺序扩展，堆管动态最值。

## 🎯 适用场景

- 括号匹配
- BFS
- 第 K 大/小
- 持续取最大最小

## 🧠 原理理解

这些是不同的出入顺序：后进先出、先进先出、最小优先。

## 📝 标准模板

```python
from collections import deque
import heapq

stack = []
stack.append(x)
stack.pop()

q = deque([start])
q.popleft()

heapq.heappush(h, x)
heapq.heappop(h)
```

## ⚠️ 易错点

- heapq 是小顶堆
- BFS 用 popleft
- 栈空时别 pop
