## 暴力模型

```python
for i in range(a, b + 1):
    if is_prime(i) and is_hw(i):
        print(i)
```

问题在于：

- 区间可能很大。
- 每个数判断素数又要循环。
- 很多明显不可能的数也被检查了。

## 第一层优化：素数判断到 sqrt(n)

```python
def is_prime(n):
    if n < 2:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    for i in range(3, int(n ** 0.5) + 1, 2):
        if n % i == 0:
            return False
    return True
```

这比检查到 `n-1` 快很多。

## 第二层优化：先生成回文数

题目要的是“回文质数”，与其枚举所有数，不如只枚举回文数。

思路：

```text
枚举一半数字 -> 拼成回文 -> 判断是否在范围内 -> 判断素数
```

## 下次识别信号

如果题目要求同时满足两个条件：

- 条件 A 很稀有，比如回文。
- 条件 B 检查较贵，比如素数。

可以考虑先生成稀有的那一类，再检查另一个条件。
