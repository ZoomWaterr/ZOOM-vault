# 断点06_DP状态不会定义

## 症状

一看到 DP 就知道要写 `dp`，但不知道 `dp[i]` 到底表示什么。

## 根因

你跳过了“状态是什么”这一步，直接想公式。

## DP 状态定义方法

先找题目问什么。

如果题目问：

> 到第 i 个位置的最大值

那么可能是：

```text
dp[i] = 到第 i 个位置时的最大结果
```

如果题目问：

> 前 i 个物品，容量 j 的最大价值

那么可能是：

```text
dp[i][j] = 只考虑前 i 个物品，容量为 j 时的最大价值
```

## DP 五问

每次写 DP 前必须填：

```text
dp[i] 表示：
dp[i] 从哪里来：
初始值：
遍历顺序：
答案位置：
```

## 不要先背公式

公式是状态定义之后自然来的。

状态没定义清楚，公式背了也会乱。
