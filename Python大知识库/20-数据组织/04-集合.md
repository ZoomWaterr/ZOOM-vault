# 04 集合

集合是一堆不重复的元素。

它最适合两件事：去重、快速判断存在。

---

## 创建集合

```python
s = set()
```

注意：

```python
s = {}
```

这是空字典，不是空集合。

---

## 常用操作

```python
s.add(x)

if x in s:
    ...
```

---

## 去重

```python
unique = list(set(nums))
```

但集合没有顺序。如果题目要求保留原顺序，就不能直接这样做。

---

## 小结

集合的价值不在“存很多东西”，而在“查得快”和“不重复”。
