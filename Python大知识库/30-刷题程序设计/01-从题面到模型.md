# 01 从题面到模型

刷题最重要的能力之一，是把自然语言题面翻译成程序模型。

---

## 三个问题

拿到题先问：

```text
输入给了什么？
我要维护什么？
输出要什么？
```

如果这三个问题说不清，代码通常会越写越乱。

---

## 例子：统计超过一半的数

题面说：给一组数，判断有没有某个数出现次数超过一半。

翻译成模型：

| 题面 | 程序 |
|:---|:---|
| 一组数 | `nums` |
| 出现次数 | `count` 字典 |
| 超过一半 | `cnt > n / 2` |
| 没有就输出 no | `for-else` |

代码自然就出来了：

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

---

## 小结

建模不是高深词，它只是把题面里的名词和动作，变成变量和更新规则。
