# 函数与拆分

## 先看这个

        函数就是把一段有明确目的的代码打包。代码一长，函数能让你不迷路。

        ## 什么时候用

        模拟题变长、同一段逻辑重复出现、需要单独判断一个条件时。

        ## 怎么写

        ```python
        def is_prime(x):
    if x < 2:
        return False
    for i in range(2, int(x ** 0.5) + 1):
        if x % i == 0:
            return False
    return True

if is_prime(n):
    print("yes")
        ```

        ## 我容易摔在哪里

        不要为了写函数而写函数。函数名必须说明它干什么，比如 `is_prime`、`read_matrix`、`calc_score`。

## 什么时候必须考虑拆函数

- 同样逻辑出现两次以上。
- 一个循环里塞了很多判断。
- 调试时你说不清某一段代码的目的。
- 题目有“判断某个东西是否满足条件”。

## 返回多个值

```python
def get_min_max(nums):
    return min(nums), max(nums)

mn, mx = get_min_max(nums)
```

## 个人提醒

你现有代码里函数很少，这是基础题阶段的正常现象。下一阶段做模拟题、搜索题、DP 题时，要开始练习拆函数。
