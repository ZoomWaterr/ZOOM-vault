## 栈 ⭐⭐

> 后进先出。括号匹配。

```python
def is_valid(s):
    stack = []
    pair = {')':'(', ']':'[', '}':'{'}
    for ch in s:
        if ch in "([{": stack.append(ch)
        elif not stack or stack.pop() != pair[ch]: return False
    return not stack
```

## 单调栈 ⭐⭐

> 栈内始终保持单调。找"下一个更大元素" O(n)。

```python
def next_greater(nums):
    n = len(nums); ans = [-1]*n; stack = []
    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            ans[stack.pop()] = nums[i]
        stack.append(i)
    return ans
# [2,1,2,4,3] → [4,2,4,-1,-1]
```

## 并查集 ⭐⭐

> 管理分组和连通。近乎O(1)。

```python
class DSU:
    def __init__(self, n):
        self.p = list(range(n))
    def find(self, x):
        if self.p[x] != x:
            self.p[x] = self.find(self.p[x])  # 路径压缩
        return self.p[x]
    def union(self, a, b):
        self.p[self.find(a)] = self.find(b)
    def same(self, a, b):
        return self.find(a) == self.find(b)
```

🎯 分组 · 连通性 · 朋友圈 · 生成树

## 堆（优先队列）⭐⭐

```python
import heapq
heap = []
heapq.heappush(heap, x)
heapq.heappop(heap)          # 总是弹出最小的
# 大顶堆：存负数

# 持续维护前K大
def top_k(stream, k):
    h = []
    for x in stream:
        heapq.heappush(h, x)
        if len(h) > k: heapq.heappop(h)  # 踢出最小
    return h
```

🎯 Top-K · 持续取最小/最大 · Dijkstra
