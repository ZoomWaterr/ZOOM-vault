# 字典与计数

tags: #python/字典 #算法/哈希 #刷题/高频

## 应该怎么做

需要建立“谁 -> 对应什么”的关系，就用字典。

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

## 这在干什么

字典是映射：通过 key 快速找到 value。刷题中最常用的是统计次数、记录位置、判断某个东西是否出现过。

## 频率统计模板

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

for num, cnt in count.items():
    if cnt > len(nums) / 2:
        print(num)
        break
else:
    print("no")
```

## 找出现最多的

```python
best = max(count, key=count.get)
print(best, count[best])
```

## 标准库增强

```python
from collections import Counter

count = Counter(nums)
print(count.most_common(1))
```

## 我的提醒

你的原话：

- “统计出现次数”用字典。
- “记录每个 xx 对应的 xx”用字典。
- “判断某个东西出现过没有的”用字典。

这句话够硬，应该背。

## 什么时候翻它

- 题目问“出现次数最多/超过一半/第一个不重复”。
- 你发现自己在列表里反复 `count()`，可能要换字典。
- 需要快速判断某个数之前有没有出现过。
