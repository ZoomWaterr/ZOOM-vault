# 二、标识符

## 命名规则

- 由字母（a~z, A~Z）、数字、下划线（`_`）组成
- 数字不能开头
- 不能使用关键字：`if`、`else`、`elif`、`for`、`while`、`and`、`or` 等
- 严格区分大小写

## 命名规范

- 多部分用下划线连接：`new_color`
- 见名知意
- 英文字母全小写

# 六、流程控制

## if 条件判断

```python
if 条件1:
    操作1
elif 条件2:          # 可以有多个 elif
    操作2
else:                # else 不需要条件，前面都不成立时执行
    操作3
```

## match...case（模式匹配）

```python
match day:
    case "1":
        print("周一")
    case "6" | "7":      # | 表示"或"
        print("周末")
    case _:               # _ 匹配所有其他情况
        print("输入错误")
```

## while 循环

条件满足时一直循环，**关注的是循环的条件**（次数未知）：

```python
while 条件表达式:
    循环体
```

- `break`：跳出整个循环
- `continue`：跳过当前轮，进入下一轮

## for 循环

遍历数据集或已知次数，**关注的是遍历每一个元素**：

```python
for 元素 in 待处理数据集:
    循环体
```

### range() 生成数字序列

```python
range(5)         # → 0, 1, 2, 3, 4
range(2, 8)      # → 2, 3, 4, 5, 6, 7
range(0, 10, 2)  # → 0, 2, 4, 6, 8（步长为2）
```

### for 与 while 的选择

| | for | while |
|:---|:---|:---|
| 适用场景 | 遍历数据集、已知次数 | 条件满足时一直循环 |
| 关键点 | 关注每个元素 | 关注循环条件 |

# 七、数据容器

## 总览

| 类型 | 初始化 | 特点 | 刷题用途 |
|:---|:---|:---|:---|
| `list` 列表 | `a = []` 或 `a = [0] * n` | 有序、可重复、可修改 | 存数据、排序、前缀和、差分，**用得最多** |
| `tuple` 元组 | `t = ()` 或 `t = (1, 2, 3)` | 有序、可重复、不可修改 | 固定不变的数据，**刷题很少用** |
| `str` 字符串 | `s = ""` 或 `s = input()` | 有序、不可修改 | 字符串处理题，split/find/replace |
| `set` 集合 | `s = set()` 或 `s = {1, 2, 3}` | 无序、不可重复、可修改 | 去重、快速判断元素存不存在 |
| `dict` 字典 | `d = {}` 或 `d = dict()` | 键值对、key 不可重复 | 统计次数、建立映射，**用得很多** |

### 一句话区分

```
列表   → 一排格子，啥都能放       [1, 2, 2]
元组   → 列表但锁死了，不能改      (1, 2, 2)
字符串 → 一排字符，不能改          "hello"
集合   → 一堆球，不重复，无顺序    {1, 2, 3}
字典   → 一本名册，名字对应数值    {"a": 1}
```

### 使用频率

```
列表   ⭐⭐⭐⭐⭐  几乎每道题都用
字典   ⭐⭐⭐⭐    统计类题必用
字符串 ⭐⭐⭐      字符串处理题
集合   ⭐⭐       去重/查找用
元组   ⭐        基本用不上
```

### 通用工具

| 函数 | 作用 |
|:---|:---|
| `min()` | 求最小值 |
| `max()` | 求最大值 |
| `sum()` | 求和 |
| `len()` | 获取元素个数 |
| `sum()/len()` | 求平均数 |
| `count()` | 统计出现次数 |

## 列表 list

### 列表推导式

```python
# 基本格式
[表达式 for 变量 in 列表 if 条件]

# 例子：1-20 的平方列表
num_list = [i ** 2 for i in range(1, 21)]
```

### 列表切片

```
s[开始索引 : 结束索引 : 步长]
```

- 范围遵循**前闭后开**
- 开始省略 → 从第一个开始
- 结束省略 → 到最后一个（含）
- 步长省略 → 默认为 1；可为负数（反向）

**易错点**：`s[:-2]` 是到倒数第二个结束，不是步长为 -2。

### 常用方法

| 方法 | 作用 | 样例 |
|:---|:---|:---|
| `append(x)` | 末尾添加元素 | `a.append(3)` |
| `pop(i)` | 删除并返回第 i 个元素 | `a.pop()` |
| `sort()` | 从小到大排序（改原列表） | `a.sort(reverse=True)` |
| `sorted()` | 返回新排序列表（不改原列表） | `b = sorted(a)` |
| `len(a)` | 列表长度 | `len(a)` |
| `a[i:j]` | 切片取子列表 | `a[0:3]` |
| `index(x)` | 找 x 第一次出现的下标 | `s.index(x)` |

## 字符串 str

**特点**：不可修改、有序、可被 for 遍历。

### 常用方法

| 方法 | 作用 | 样例 |
|:---|:---|:---|
| `split()` | 按空格分割成列表 | `s.split()` |
| `strip()` | 去掉两端空白/字符 | `s.strip('.')` |
| `rstrip()` | 去掉右端字符 | `s.rstrip('.')` |
| `find(x)` | 找子串位置，没有返回 -1 | `s.find('ab')` |
| `replace(a, b)` | 把 a 全部替换成 b | `s.replace('.', ' ')` |
| `join()` | 用指定符号拼接列表 | `' '.join(lst)` |
| `upper()` / `lower()` | 转大写 / 转小写 | `s.upper()` |
| `isalpha()` | 判断是否全是字母 | `s.isalpha()` |

除了join 其他的全都不会修改原字符串，都需要变量接受返回值！

## 元组 tuple

**创建后不可变的列表**。

```python
t = (1, 2, 3)
t = ()
t = tuple()
```

### 组包和解包

```python
t = (9, 2, 34, 123)

# 基础解包
a, b, c, d = t

# 拓展解包（* 收集剩余元素）
a, *b, c = t     # a=9, b=[2,34], c=123
```

## 集合 set

**特点**：无序、不可重复、可修改。

```python
s = {1, 2, 3, 4, 5}
s = set()         # 注意：{} 是空字典，不是空集合
```

### 常用方法

| 方法 | 作用 |
|:---|:---|
| `set(list)` | 列表转集合（自动去重） |
| `add(x)` | 添加元素 |
| `remove(x)` | 删除元素，不存在则报错 |
| `discard(x)` | 删除元素，不存在不报错 |
| `union()` | 并集（合并两个集合） |
| `intersection()` | 交集（两个集合共有的） |
| `difference()` | 差集（在 A 不在 B 的） |

### 运算符简写

```python
a & b   # 交集
a | b   # 并集
a - b   # 差集
```

### 集合推导式

```python
{表达式 for i in 列表 if 条件}
```

## 字典 dict

在其他语言中叫：哈希表（hash）、映射（map）。

**特点**：键值对存储、key 不能重复、可修改。

```python
# 定义
d = {key: value, key: value, ...}
d = {}
d = dict()

# 根据 key 获取 value
值 = d[key]
```

### 常用操作

```python
# 安全获取：有则返回，没有则返回默认值
count.get(key, default)

# 遍历 key
for k in d.keys():
    print(f"{k} : {d[k]}")

# 遍历 key 和 value
for k, v in d.items():
    print(f"{k} : {v}")
```

# 八、函数

## 定义与调用

```python
# 定义
def 函数名(参数列表):
    函数体
    return 返回值

# 调用
函数名(参数)
```

- 必须先定义，再使用
- 定义时不执行，调用时才执行

## 参数传递

```python
# 位置参数
s = reg_stu("德远", "18", "男", "北京")

# 关键字参数
a = reg_stu(name="德远", age="18", gender="男", city="北京")

# 默认参数（默认参数要放在普通参数之后）
def reg_stu(name, age, gender, city="北京"):
    ...
```

## 不定长参数

```python
def just_do(*args, **kwargs):
    ...

# *args   → 位置参数，封装成元组
# **kwargs → 关键字参数，封装成字典
```

## 类型注解

```python
def reg_stu(name: str, age: int, city: str = "北京") -> dict:
    ...
```
