# 题目笔记模板

> 单题笔记可以写，但不要每题都硬写。只有这题真的有收获时再写。

```markdown
# 题名

## 题目要我干嘛

用一句人话复述。

## 我第一反应

写真实想法，不用漂亮。

## 正解/当前解法

```python
# 关键代码
```

## 我摔在哪里

- 语法？
- 边界？
- 数学推导？
- 输出格式？

## 以后提醒我

一句话。
```
