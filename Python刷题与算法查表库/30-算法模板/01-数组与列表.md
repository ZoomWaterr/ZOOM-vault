**快速定位**
先把数据放进列表：`nums = list(map(int, input().split()))`。

**一句话**
列表是一排格子，按下标访问快，是大多数算法的地基。

**适用场景**
- 存一组数
- 需要下标
- 排序、前缀和、DP

**原理理解**
hello-algo 对数组的核心解释是：连续存储，所以按下标访问快；中间插入删除慢。

**标准模板**

```python
nums = list(map(int, input().split()))
nums.append(x)
nums.sort()
print(nums[i])

```

**易错点**
- 注意下标越界
- 切片前闭后开
- 二维列表不要复制同一行

你的 Python 刷题能力，大半建立在列表熟不熟上。
