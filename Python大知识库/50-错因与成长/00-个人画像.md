        ## 你现在在哪

        你处在“基础语法 → 基础题单 → 入门算法边缘”的阶段。你已经不是看不懂 Python，而是正在把“想到”稳定翻译成“写对”。

        - 你能自然使用 Python 写法：切片、`dict.get()`、`max(key=...)`、`for-else` 都已经出现。
