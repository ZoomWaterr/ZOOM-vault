# 双指针与滑动窗口

tags: #算法/双指针 #算法/滑动窗口

## 应该怎么做

看到连续子数组/子串，且左右边界会移动，先想滑动窗口。

## 这在干什么

双指针就是两个位置一起配合移动，避免重复枚举所有区间。

## 对撞指针

```python
left, right = 0, len(nums) - 1
while left < right:
    s = nums[left] + nums[right]
    if s == target:
        print(left, right)
        break
    elif s < target:
        left += 1
    else:
        right -= 1
```

## 滑动窗口

```python
left = 0
window_sum = 0

for right, x in enumerate(nums):
    window_sum += x

    while window_sum > target:
        window_sum -= nums[left]
        left += 1

    # 此时 [left, right] 是一个满足条件的窗口
```

## 什么时候翻它

- 连续子数组。
- 连续子串。
- 最长/最短满足条件的一段。
- 数组是有序的，两端往中间夹。

## 我的提醒

滑动窗口适合“窗口变大后能通过移动左边界恢复条件”的题。如果有负数导致窗口和不单调，要警惕，可能要换前缀和 + 字典。
