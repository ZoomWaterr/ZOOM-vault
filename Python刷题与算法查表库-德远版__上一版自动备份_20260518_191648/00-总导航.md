## 这版和旧库不同在哪里

旧版 `F:\ZOOM-vault\计算机知识` 的优点我保留：入口清楚、callout 生动、成长区有温度。  
但这版不复刻它，而是改成更适合“关键时刻查表”的结构：

| 旧版容易出现的问题 | 新版解决方式 |
|---|---|
| 算法大章偏长，关键时刻还要二次定位 | 每张算法卡固定分成 🚑快速定位 / 🧠再理解 / 📝模板 |
| 刷题记录容易变成流水账 | 改成 [[40-错题复盘/00-错题复盘总表|错题复盘]]，只收有价值的坑和顿悟 |
| 语法、题型、算法之间链接不够主动 | 新增 [[01-快速查表/03-易混概念对照|易混概念对照]] 和 [[20-解题方法/02-数据范围与复杂度|数据范围与复杂度]] |
| 个人编辑区不够制度化 | 每张核心卡都有 ✍️ 我的编辑区 |
