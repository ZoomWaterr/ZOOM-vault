# 🧮 格式化与取整动作

## 🚀 快速定位

正整数向上取整背：`(a + b - 1) // b`。

## 💡 一句话

输出格式和取整经常不是难点，但最容易丢分。

## 🎯 适用场景

- 保留小数
- 补零时间
- 向上取整
- 平均值

## 🧠 原理理解

格式化是把答案变成题目要的样子；取整要分清向上、向下、截断。

## 📝 标准模板

```python
print(f"{x:.3f}")
print(f"{h:02d}:{m:02d}")

ceil_div = (a + b - 1) // b
floor_div = a // b
```

## ⚠️ 易错点

- `int(3.9)` 是截断
- `/` 返回 float
- 负数时 `//` 是向下，不是截断
