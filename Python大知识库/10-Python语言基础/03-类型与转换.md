# 03 类型与转换

类型决定一个值能做什么。

`10` 可以参与计算，`"10"` 更像一段文字。它们看起来像，但行为完全不同。

---

## 常见类型

| 类型 | 例子 | 刷题用途 |
|:---|:---|:---|
| `int` | `10` | 计数、下标、循环次数 |
| `float` | `3.14` | 小数、几何、平均数 |
| `str` | `"abc"` | 字符、单词、逐位处理 |
| `bool` | `True` | 判断条件 |
| `None` | `None` | 占位、默认值 |

---

## 字符串和整数不要混用

```python
"10" + "1"  # "101"
10 + 1      # 11
```

刷题时遇到输入，先判断：它是用来计算，还是用来逐字符处理？

- 用来计算：转成 `int` 或 `float`。
- 用来逐字符处理：保留为 `str`。

---

## 数字转字符串

数字反转、逐位判断时，字符串路线很方便：

```python
n = input()
print(n[::-1])
```

如果已经是整数：

```python
s = str(n)
```

---

## 小结

类型转换的核心不是背函数，而是知道“我接下来要对这个值做什么”。要算，就转数字；要拆字符，就保留字符串。
