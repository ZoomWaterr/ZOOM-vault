# 语法速查 · 按场景找

> 不是按知识点排，是按**「我要做什么」**排。想法 → 代码，一步到位。

---

## 输入

| 我想…… | 代码 |
|:---|:---|
| 读一个整数 | `n = int(input())` |
| 读一行两个整数 | `a, b = map(int, input().split())` |
| 读一行变整数列表 | `nums = list(map(int, input().split()))` |
| 读 n 行每行一个数 | `[int(input()) for _ in range(n)]` |
| 读 n 行每行多个数 | `[list(map(int, input().split())) for _ in range(n)]` |
| 读浮点数 | `x, y = map(float, input().split())` |

---

## 输出

| 我想…… | 代码 |
|:---|:---|
| 列表空格输出 | `print(*a)` |
| 保留2位小数(会补零!) | `f"{x:.2f}"` |
| 补零对齐 08:05 | `f"{h:02d}:{m:02d}"` |
| 每行输出一个 | `print("\n".join(map(str, a)))` |
| 不换行 | `print(x, end="")` |

---

## 列表 list

| 我想…… | 代码 |
|:---|:---|
| 创建 n 个 0 | `[0] * n` |
| 推导式一行生成 | `[i**2 for i in range(1,11)]` |
| 筛选 | `[x for x in lst if x > 0]` |
| 末尾添加 | `a.append(x)` |
| **按值删除** | `a.remove(x)` — 删第一个值为x的 |
| **按下标删除** | `a.pop(i)` — 删下标i，返回被删值 |
| **删最后一个** | `a.pop()` |
| 从小到大排序(改原列表) | `a.sort()` |
| 排序不改原列表 | `sorted(a)` |
| 反转 | `a[::-1]` |
| 取子列表 | `a[2:5]` |
| 找下标 | `a.index(x)` — 找不到报错 |
| 二维列表(正确!) | `[[0]*m for _ in range(n)]` |

> 🔥 `pop(值)` 是错的！pop 的参数是**下标**。按值删用 `remove(值)`。

## 字符串 str

| 我想…… | 代码 |
|:---|:---|
| 按空格分割 | `s.split()` |
| 按逗号分割 | `s.split(",")` |
| 去两端空白 | `s.strip()` |
| 全部替换 | `s.replace("旧","新")` |
| 找子串位置 | `s.find("abc")` — 找不到返-1 |
| 列表拼成字符串 | `"-".join(lst)` |
| 大小写 | `s.upper()` `s.lower()` |
| 反转 | `s[::-1]` |
| 判断全字母/全数字 | `s.isalpha()` `s.isdigit()` |

> ⚠️ 字符串方法**不改原字符串**！必须 `s = s.strip()`。

## 字典 dict

| 我想…… | 代码 |

| 统计频率 | `cnt[x] = cnt.get(x, 0) + 1` |

## 循环

| 我想…… | 代码 |
|:---|:---|
| 跑 n 次 | `for i in range(n):` |
| 从 a 到 b | `for i in range(a, b+1):` |
| 同时拿下标和值 | `for i, x in enumerate(lst):` |
| 同时遍历两个列表 | `for a, b in zip(lst1, lst2):` |
| 条件循环 | `while 条件:` |
| 跳过本轮 | `continue` |
| 跳出 | `break` |
| 正常结束才执行 | `else:` (和for/while对齐) |
| 双重循环不重不漏 | `for i in range(n): for j in range(i):` |

## 一句话区分五种容器

```
列表 [1,2,2]  → 一排格子，可改       ⭐⭐⭐⭐⭐
字典 {"a":1}  → 名册，key→value     ⭐⭐⭐⭐
字符串"hello" → 一排字符，不可改      ⭐⭐⭐
集合 {1,2,3}  → 不重复，无顺序       ⭐⭐
元组 (1,2,3)  → 列表锁死版           ⭐
```
