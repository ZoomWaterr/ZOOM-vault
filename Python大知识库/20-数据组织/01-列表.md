# 01 列表

列表是一排有顺序的格子。它是刷题里最常用的数据结构。

---

## 创建列表

```python
a = []
a = [0] * n
nums = list(map(int, input().split()))
```

---

## 常用操作

| 操作 | 代码 |
|:---|:---|
| 添加 | `a.append(x)` |
| 排序 | `a.sort()` |
| 不改原列表排序 | `b = sorted(a)` |
| 反转 | `a[::-1]` |
| 求和 | `sum(a)` |
| 最大最小 | `max(a)` / `min(a)` |

---

## 列表推导式

```python
squares = [i * i for i in range(1, n + 1)]
evens = [x for x in nums if x % 2 == 0]
```

推导式适合“从旧列表生成新列表”。如果逻辑复杂，就老老实实写循环。

---

## 小结

列表的核心是顺序。只要题目给你一排数，要排序、反转、求和、找最大，基本都先想到列表。
