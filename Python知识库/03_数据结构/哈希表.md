# 哈希表

Python 的 `dict` 和 `set` 底层都可以理解为哈希表。

## 它解决什么

快速查找。

```python
x in seen
count[x]
```

平均情况下接近 `O(1)`。

## 常见用途

- 计数：[[哈希计数]]
- 去重：[[set集合]]
- 建映射：[[dict字典与哈希]]
- 两数之和：数 -> 下标

## 易错点

- 哈希表快，但会占额外空间。
- key 必须是可哈希对象，比如数字、字符串、元组。
- list 不能作为 dict 的 key。
