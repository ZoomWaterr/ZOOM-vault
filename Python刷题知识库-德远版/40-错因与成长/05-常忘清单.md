# 常忘清单

## 先看这个

忘不是问题，没入口才是问题。忘了就来这里找。

## 目前常忘点

- 输入永远是字符串。
- `range` 右边不包含。
- `/` 是 float，`//` 是整除。
- 字符串方法要接返回值。
- 字典计数用 `get`。
- 矩阵先行后列。
- 嵌套循环变量要有含义。
- 提交前检查输出格式。
