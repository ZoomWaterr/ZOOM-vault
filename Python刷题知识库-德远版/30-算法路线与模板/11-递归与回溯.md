# 递归与回溯

## 先看这个

递归是函数调用自己；回溯是在尝试一种选择后，撤销它，再试下一种。

## 什么时候用

- 所有排列。
- 所有组合。
- 棋盘搜索。
- 需要“选/不选”的题。

## 回溯模板

```python
def backtrack(path):
    if 满足结束条件:
        ans.append(path[:])
        return

    for choice in choices:
        if 不合法:
            continue
        path.append(choice)
        backtrack(path)
        path.pop()
```

## 我容易摔在哪里

回溯最重要的四个字：恢复现场。你 append 了什么，回来就 pop 什么。
