# 列表与切片

tags: #python/列表 #python/切片 #刷题/高频

## 应该怎么做

一组有顺序的数据，优先想到列表。

```python
nums = []
nums.append(3)
nums.sort()
print(nums[0], nums[-1])
```

## 这在干什么

列表是一排格子：有序、可重复、可修改。刷题里存数组、排序、前缀和、差分数组，基本都靠它。

## 常用模板

```python
a = [0] * n
a.append(x)
a.pop()
a.sort()
b = sorted(a)
```

## 切片

```python
s[开始:结束:步长]
```

范围是前闭后开。

```python
a[:3]      # 前 3 个
a[3:]      # 从下标 3 到末尾
a[::-1]    # 反转
a[:-2]     # 到倒数第二个之前，不是步长 -2
```

## 列表推导式

```python
squares = [i * i for i in range(1, 21)]
even_squares = [i * i for i in range(1, 21) if i % 2 == 0]
```

## 我的提醒

你的原话：“原汁原味的 Python 表达！很有用。”

`sort()` 会直接修改原列表，`sorted()` 返回新列表。

## 什么时候翻它

- 要存一组数字。
- 要排序后处理。
- 要反转、截取、每隔几个取一个。
- 写前缀和、差分数组前。
