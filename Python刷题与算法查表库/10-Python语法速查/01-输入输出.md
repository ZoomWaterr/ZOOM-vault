# 输入输出

**快速定位**
先复制：`nums = list(map(int, input().split()))`；输出列表优先 `print(*nums)`。

**一句话**
`input()` 负责拿字符串，`split/map/list` 负责把它变成能算的数据。

**适用场景**
- 题目开头输入格式复杂
- 输出要求空格分隔、每行一个、保留小数

**原理理解**
OJ 给你的都是文本。你要先把文本拆成结构，再交给算法。

**标准模板**
```python
n = int(input())
a, b = map(int, input().split())
nums = list(map(int, input().split()))

print(*nums)
print(" ".join(map(str, nums)))
print(f"{x:.3f}")
```

**易错点**
- `join` 只能拼字符串
- `input()` 永远先是字符串
- 小数格式用 f-string

你原来的 `print(*num)` 很好，保留它，这是刷题里最快的列表输出。
