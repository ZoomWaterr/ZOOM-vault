# 数据类型与转换

tags: #python/基础 #python/类型

## 应该怎么做

先判断变量是什么东西，再选择操作方法。

| 类型 | 例子 | 刷题用途 |
|---|---|---|
| `int` | `10` | 计数、下标、答案 |
| `float` | `3.14` | 小数、几何、平均值 |
| `str` | `"abc"` | 字符串题、输入原文 |
| `bool` | `True` / `False` | 判断条件 |
| `None` | `None` | 暂无数据、占位 |

## 转换模板

```python
int("123")         # 123
float("3.14")     # 3.14
str(123)          # "123"
bool([])          # False
bool([1])         # True
```

## 这在干什么

Python 的变量本身带类型。刷题里最常见的错误不是算法错，而是把字符串当数字、把列表当整数、把浮点数拿去做精确判断。

## 整除与取整

```python
a // b                  # 向下整除
(a + b - 1) // b         # 正整数向上取整

import math
math.ceil(x)             # 向上取整
math.floor(x)            # 向下取整
round(x)                 # 四舍五入
```

## 我的提醒

`int(3.9)` 是截断成 `3`，不是四舍五入。

`/` 永远返回浮点数；整数除法用 `//`。

## 什么时候翻它

- 题目有“向上取整”“最少需要几组”。
- 你输出小数总是格式不对。
- 输入是字符串，但你准备拿它做数学运算。
