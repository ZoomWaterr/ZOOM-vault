# 递归入门

递归就是函数自己调用自己。树、DFS、回溯、很多 DP 都离不开它。

## 递归必须有两件事

1. 结束条件。
2. 把大问题拆成小问题。

## 阶乘例子

```python
def fact(n):
    if n == 1:
        return 1
    return n * fact(n - 1)
```

## 递归思维

不要在脑子里展开 100 层。你只要相信：

> 子问题能正确返回，我这一层就能根据子问题得到答案。

## 和循环的关系

循环更像“从前往后一步步做”。

递归更像“定义一个问题，然后调用更小的问题”。

## 易错点

- 忘记结束条件会无限递归。
- 每一层的局部变量是自己的。
- Python 默认递归深度有限，特别深的递归要小心。
