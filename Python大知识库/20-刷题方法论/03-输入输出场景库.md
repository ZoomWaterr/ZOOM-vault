# 输入输出场景库

## 一行一个数

```python
n = int(input())
```

## 一行多个数

```python
a, b = map(int, input().split())
```

## 一行列表

```python
nums = list(map(int, input().split()))
```

## n 行收集

```python
n = int(input())
data = []
for _ in range(n):
    data.append(int(input()))
```

## 矩阵

```python
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]
```

## 输出列表

```python
print(*nums)
print(" ".join(map(str, nums)))
```
