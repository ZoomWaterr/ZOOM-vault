# 小白到独当一面的路线图

这不是鸡血路线，而是能力路线。每一层都有明确的过关标准。

## L0：会写 Python 基础题

你能完成：

- 输入输出。
- 条件判断。
- 单层循环。
- 列表、字符串、字典基础操作。

过关标准：

> 简单模拟题不用看题解，能独立写完。

## L1：会解释循环和变量

你能解释：

- `i` 为什么从 0 开始。
- `range(i)` 为什么只看左边。
- `cnt` 是当前计数还是总计数。
- `ans` 什么时候更新。

过关标准：

> 写双层循环前能先说清楚两个变量分别代表谁。

## L2：会从暴力推优化

你能做到：

- 先写暴力。
- 估复杂度。
- 找重复计算。
- 改成哈希、前缀和、双指针、二分等模型。

过关标准：

> 遇到超时，不是只说“Python 慢”，而是能说出哪一层慢。

## L3：会识别基础模型

你能看到题目信号想到：

- 统计次数 -> [[哈希计数]]
- 区间和 -> [[前缀和]]
- 区间修改 -> [[差分]]
- 连续子串 -> [[滑动窗口]]
- 有序边界 -> [[二分查找]]

过关标准：

> 做题前能写出“这题我优先考虑什么模型”。

## L4：会搜索和状态

你能理解：

- DFS 是一条路走到底。
- BFS 是一层层扩散。
- 回溯是选择、递归、撤销。
- DP 是状态和状态转移。

过关标准：

> 能自己写出基础 DFS、BFS、线性 DP。

## L5：能独当一面

你能做到：

- 新题先分析，不急着看题解。
- 能写暴力、能优化、能解释复杂度。
- 看题解后能复盘模型，而不是只抄代码。
- 能把做过的题挂回知识库。

这时你就不是“会一些题”，而是有自己的算法知识网。
