# 字符串 str

> 字符串是一排字符，但不能原地修改。所有字符串方法都要接回来。

## 常用写法

| 我想…… | 代码 |
|:---|:---|
| 按空格拆 | `s.split()` |
| 去两端空白 | `s = s.strip()` |
| 替换 | `s = s.replace("-", "")` |
| 反转 | `s[::-1]` |
| 转大写 | `s.upper()` |
| 转小写 | `s.lower()` |
| 判断数字 | `s.isdigit()` |
| 判断字母 | `s.isalpha()` |

## join

```python
words = ["hello", "world"]
print(" ".join(words))

nums = [1, 2, 3]
print(" ".join(map(str, nums)))
```

## 易错点

```python
s.strip()
```

这一行不会改变 `s`。要写：

```python
s = s.strip()
```
