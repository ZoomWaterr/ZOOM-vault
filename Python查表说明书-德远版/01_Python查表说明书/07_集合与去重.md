# 集合与去重

tags: #python/集合 #算法/去重

## 应该怎么做

只关心“有没有出现过”，不关心顺序和次数，用集合。

```python
seen = set()
for x in nums:
    if x in seen:
        print("重复了")
    seen.add(x)
```

## 这在干什么

集合是一堆不重复的元素。它适合去重、查存在、做交并差。

## 常用写法

```python
s = set()
s.add(x)
s.discard(x)

unique = set(nums)
common = set(a) & set(b)
all_items = set(a) | set(b)
only_a = set(a) - set(b)
```

## 易错点

```python
empty_set = set()
empty_dict = {}
```

`{}` 是空字典，不是空集合。

## 什么时候翻它

- 去重。
- 判断是否出现过。
- 两组数据求共同元素、不同元素。
