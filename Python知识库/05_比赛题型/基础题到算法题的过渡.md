# 基础题到算法题的过渡

基础题和算法题最大的区别不是代码长度，而是“有没有模型”。

## 基础题

通常按题意直接模拟：

- 输入。
- 判断。
- 循环。
- 输出。

## 算法题

通常要先识别模型：

- 能不能暴力？
- 暴力复杂度多少？
- 慢在哪里？
- 能不能用哈希、前缀和、双指针、二分、DP？

## 过渡训练顺序

1. 基础模拟。
2. 数组枚举。
3. 哈希计数。
4. 前缀和 / 差分。
5. 双指针 / 滑动窗口。
6. 二分。
7. DFS / BFS。
8. DP。

## 判断自己有没有进步

不是看题做得多不多，而是看：

- 能不能说出题型。
- 能不能估复杂度。
- 能不能自己从暴力推到优化。
