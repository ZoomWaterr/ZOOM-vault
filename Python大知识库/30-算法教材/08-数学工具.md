# 数学工具

## 最大公约数

```python
import math
g = math.gcd(a, b)
l = a * b // g
```

Python 3.14 也可以用：

```python
math.lcm(a, b)
```

## 素数判断

```python
def is_prime(n):
    if n < 2:
return False
    i = 2
    while i * i <= n:
if n % i == 0:
    return False
i += 1
    return True
```

## 位数拆分

```python
ge = n % 10
shi = n // 10 % 10
bai = n // 100
```
