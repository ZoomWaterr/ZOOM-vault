# 递推与动态规划入门

> 递推就是当前状态由前面的状态算出来。DP 先别怕，入门时它就是“填表”。

## 斐波那契

```python
a = [0] * (n + 2)
a[1] = a[2] = 1

for i in range(3, n + 1):
    a[i] = a[i - 1] + a[i - 2]

print(a[n])
```

## 爬楼梯

每次走 1 级或 2 级：

```text
到第 n 级的方法数 = 到第 n-1 级 + 到第 n-2 级
```

## 给你的提醒

你在“研究斐波那契”里写下“我该怎么弄一个无限的这样的斐波那契数列”，这其实是很好的问题。它不是失败，是你正好撞到了“递推数组 / 生成到够用为止”的入口。
