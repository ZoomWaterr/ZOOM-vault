# 数据结构基础

## 先看这个

数据结构是为了让某些操作更快、更清楚。

## 栈

```python
stack = []
stack.append(x)
top = stack.pop()
```

## 队列

```python
from collections import deque
q = deque()
q.append(x)
x = q.popleft()
```

## 堆

```python
import heapq
heap = []
heapq.heappush(heap, x)
smallest = heapq.heappop(heap)
```

## 并查集

```python
parent = list(range(n))

def find(x):
    if parent[x] != x:
        parent[x] = find(parent[x])
    return parent[x]

def union(a, b):
    parent[find(a)] = find(b)
```
