# 排序与二分

## Python自带排序 ⭐⭐

```python
a.sort()                      # 从小到大，改原列表
a.sort(reverse=True)          # 从大到小
b = sorted(a)                 # 返回新列表，不改原列表
a.sort(key=lambda x: x[1])    # 按第二个元素排
```
⏱️ O(n log n)

## 冒泡排序 ⭐

> 每轮把最大数冒到最后。

```python
def bubble(nums):
    n = len(nums)
    for 轮数 in range(n):
        swapped = False
        for 当前位置 in range(n - 1 - 轮数):
            if nums[当前位置] > nums[当前位置 + 1]:
                nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
                swapped = True
        if not swapped: break
```

> 🔥 **最经典bug**：`if nums[j] > nums[i]`——i是外层轮数不是比较对象。用中文变量名根治。

## 二分查找 ⭐⭐

```python
import bisect
arr = [2,3,5,7,11]
bisect.bisect_left(arr, 5)   # → 2  第一个>=5的位置
bisect.bisect_right(arr, 5)  # → 3  第一个>5的位置
```

### 二分答案（"最大的最小"/"最小的最大"）

```python
lo, hi = 1, max_val
while lo < hi:
    mid = (lo + hi) // 2
    if check(mid):       # 这个答案可行？
        hi = mid         # 可行→试试更小
    else:
        lo = mid + 1     # 不可行→放宽
ans = lo
```
