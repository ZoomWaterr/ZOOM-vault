# 推导式

## 先看这个

        推导式是一行生成新容器。适合简单转换和筛选，不适合塞复杂逻辑。

        ## 什么时候用

        生成列表、筛选元素、批量转换类型时。

        ## 怎么写

        ```python
        squares = [i * i for i in range(1, n + 1)]
evens = [x for x in nums if x % 2 == 0]
labels = ["偶数" if x % 2 == 0 else "奇数" for x in nums]
        ```

        ## 我容易摔在哪里

        一行太长、条件太绕时就别写推导式，普通 for 循环更清楚。

## 三种形态

```python
# 列表
arr = [f(x) for x in data]

# 集合
s = {x % 10 for x in nums}

# 字典
d = {x: x * x for x in nums}
```

## 别想复杂

推导式是让简单逻辑更短，不是让复杂逻辑更炫。
