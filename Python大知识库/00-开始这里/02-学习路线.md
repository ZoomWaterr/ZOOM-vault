# 🛤️ 学习路线

> 现在的主线：基础语法稳定 → 基础题单刷稳 → 遇到算法再补算法。

## 阶段 1：把 Python 写稳

目标不是“背完语法”，而是看到题能立刻写出基本代码。

必练：

- 输入输出：`input()`、`split()`、`map()`、`print(*a)`。
- 容器：列表、字符串、字典。
- 流程：`if`、`for`、`while`、嵌套循环。
- 数学工具：`//`、`%`、`math.gcd()`、`math.ceil()`。

## 阶段 2：基础题单刷稳

你现在正在这里。重点不是疯狂扩题，而是把“想到 → 写出”之间的断层补上。

每题只要求多做一件事：AC 后问一句“这题沉淀了什么语法/套路？”

## 阶段 3：刷到算法再学算法

不提前焦虑算法。遇到前缀和，就学前缀和；遇到差分，就学差分；遇到搜索，再学 DFS/BFS。

优先级：

| 优先级 | 内容 |
|:---:|:---|
| 🔴 | 枚举与模拟、排序、字典计数、字符串 |
| 🟡 | 前缀和、差分、递推、二分 |
| 🟢 | DFS/BFS、贪心、动态规划入门 |

## 每周复盘

每周只复盘三件事：

- 我这周最常忘的语法是什么？
- 哪道题让我“想到了但写不出来”？
- 哪个错误以后可以用清单避免？
