# 常用标准库速查

tags: #python/标准库 #刷题/工具

## 应该怎么做

先用 Python 自带工具，别手搓所有轮子。

## `math`

```python
import math

math.ceil(a / b)
(a + b - 1) // b       # 正整数向上取整，不用浮点数
math.gcd(a, b)
math.lcm(a, b)
math.sqrt(x)
```

## `collections`

```python
from collections import Counter, defaultdict, deque

Counter(nums).most_common(1)

d = defaultdict(int)
d[x] += 1

q = deque([start])
q.append(x)
q.popleft()
```

## `bisect`

```python
import bisect

pos = bisect.bisect_left(a, x)   # 第一个 >= x 的位置
pos = bisect.bisect_right(a, x)  # 第一个 > x 的位置
```

## `heapq`

```python
import heapq

h = []
heapq.heappush(h, x)
smallest = heapq.heappop(h)
```

大顶堆：存负数。

```python
heapq.heappush(h, -x)
largest = -heapq.heappop(h)
```

## `itertools`

```python
from itertools import permutations, combinations

list(permutations([1, 2, 3]))
list(combinations([1, 2, 3], 2))
```

## 我的提醒

标准库不是“偷懒”，是 Python 刷题优势。你已经自然会用 `max(key=...)`、`dict.get()`、切片，下一步就是把 `Counter`、`deque`、`bisect`、`heapq` 放进常用工具箱。
