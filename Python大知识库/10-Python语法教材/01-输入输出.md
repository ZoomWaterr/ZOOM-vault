# 输入输出 · IO 模板

> 刷题里最先要稳定的是输入输出。题会不会做另说，输入先别读歪。

## 先看这个

```python
n = int(input())
a, b = map(int, input().split())
nums = list(map(int, input().split()))

matrix = [list(map(int, input().split())) for _ in range(n)]

print(*nums)
print(" ".join(map(str, nums)))
print(f"{x:.2f}")
```

## 完整解释

`input()` 每次读一行，而且读到的是字符串。`split()` 把一行按空格拆成字符串列表，`map(int, ...)` 把每个字符串转成整数。

## 常见输入形状

| 形状 | 代码 |
|:---|:---|
| 一行一个整数 | `n = int(input())` |
| 一行多个整数 | `a, b, c = map(int, input().split())` |
| 一行整数列表 | `nums = list(map(int, input().split()))` |
| n 行，每行一个整数 | `data = [int(input()) for _ in range(n)]` |
| n 行，每行多个整数 | `rows = [list(map(int, input().split())) for _ in range(n)]` |

## 常见输出形状

```python
print(ans)
print(a, b, c)
print(*nums)
print("\n".join(map(str, nums)))
print(f"{x:.3f}")
```

## 我容易摔在哪里

- 忘记 `int()`：字符串和整数混用会报错。
- 把多行读成一行：题目说 n 行，就要循环读。
- `print(nums)` 会输出 `[1, 2, 3]`，OJ 通常要 `1 2 3`。
- 小数位按题目要求来，别随手 `round()`。
