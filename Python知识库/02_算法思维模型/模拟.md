# 模拟

模拟题就是按题意一步步做。它看起来简单，但最考验变量和边界。

## 什么时候是模拟

- 题目给了清楚的过程。
- 没有明显高级算法。
- 重点是把规则翻译成代码。

## 写模拟题的步骤

1. 列出状态变量。
2. 列出每一步怎么改变状态。
3. 写结束条件。
4. 检查边界。

## 例子结构

```python
state = 初始状态
while not 结束:
    根据规则修改 state
print(答案)
```

## 易错点

- 规则顺序写错。
- 忘记最后一步也要统计。
- 边界刚好等于时判断错。
