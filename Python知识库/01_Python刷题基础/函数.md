# 函数

函数是把一段逻辑包装起来，方便重复使用。

## 基本写法

```python
def is_prime(n: int) -> bool:
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

## 为什么刷题要写函数

- 判断素数、回文、合法状态时更清楚。
- DFS / BFS / DP 常常需要拆函数。
- 能让主流程更像题意。

## 参数和返回值

```python
def add(a, b):
    return a + b
```

`return` 是把结果交回调用处，不是打印。

## 易错点

- 函数必须先定义再调用。
- `print` 只是输出，不能代替 `return`。
- 默认参数要放在普通参数后面。
