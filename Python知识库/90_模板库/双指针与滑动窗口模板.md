# 双指针与滑动窗口模板

## 左右指针

```python
l, r = 0, n - 1
while l < r:
    if 条件:
        l += 1
    else:
        r -= 1
```

## 滑动窗口

```python
left = 0
ans = 0
window = {}

for right, x in enumerate(nums):
    window[x] = window.get(x, 0) + 1

    while 窗口不合法:
        y = nums[left]
        window[y] -= 1
        left += 1

    ans = max(ans, right - left + 1)
```
