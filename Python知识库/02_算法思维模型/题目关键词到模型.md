# 题目关键词到模型

这是一个更口语化的题型翻译表。

| 题目里看到 | 脑子里先想到 |
|---|---|
| 出现次数、频率、最多字符 | [[哈希计数]] |
| 是否出现过、去重 | [[set集合]] |
| 连续一段的和 | [[前缀和]] |
| 很多次给区间加 | [[差分]] |
| 最长连续、不重复子串 | [[滑动窗口]] |
| 有序数组两个数 | [[双指针]] |
| 找边界、第一个满足 | [[二分查找]] |
| 最小的最大、最大的最小 | [[二分答案]] |
| 所有方案、所有排列 | [[回溯]] |
| 迷宫最短路、最少步数 | [[BFS]] |
| 能不能到达、连通块 | [[DFS]] |
| 当前依赖前面 | [[动态规划]] |
| 每次选最划算 | [[贪心]] |
| 第 k 大、动态最小 | [[堆]] |

## 但不要机械套

关键词只是提示，不是绝对答案。

例如“最优”可能是贪心，也可能是 DP。真正要看：

- 有没有局部最优证明？
- 有没有重复子问题？
- 当前选择会不会影响后续？
