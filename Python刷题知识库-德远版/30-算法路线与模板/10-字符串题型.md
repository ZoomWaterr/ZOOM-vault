# 字符串题型

## 先看这个

字符串题先想四件事：拆、扫、数、拼。

## 常见题型

| 题型 | 工具 |
|:---|:---|
| 回文 | `s == s[::-1]` |
| 单词处理 | `split` / `join` |
| 字符统计 | 字典 / Counter |
| 子串 | 双指针 / 滑动窗口 |
| 格式清理 | `strip` / `replace` |

## 模板：字符统计

```python
count = {}
for ch in s:
    count[ch] = count.get(ch, 0) + 1
```

## 模板：最长单词

```python
words = input().rstrip(".").split()
print(max(words, key=len))
```

## 我容易摔在哪里

字符串不能原地修改；所有方法都要接返回值。
