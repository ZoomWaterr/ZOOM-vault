# 04 递推与动态规划入门

递推就是：后面的状态由前面的状态推出。

动态规划入门时，可以先把它理解成“填表”。

---

## 斐波那契

```python
dp = [0] * (n + 2)
dp[1] = dp[2] = 1

for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])
```

---

## 递推三件事

| 问题 | 含义 |
|:---|:---|
| 初始状态是什么 | 表从哪里开始 |
| 状态怎么转移 | 后一项怎么由前面推出 |
| 答案在哪里 | 最后输出哪个位置 |

---

## 你代码里的入口

“我该怎么弄一个无限的这样的斐波那契数列”是一个很好的问题。

实际刷题里通常不需要无限，只需要生成到题目要求的范围：

```python
while len(fib) <= n:
    fib.append(fib[-1] + fib[-2])
```

---

## 小结

递推不是背公式，而是找“当前”和“前面”的关系。
