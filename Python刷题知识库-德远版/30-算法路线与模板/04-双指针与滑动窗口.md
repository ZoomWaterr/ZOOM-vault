# 双指针与滑动窗口

## 先看这个

双指针就是两个位置一起动；滑动窗口就是维护一段连续区间。

## 什么时候用

- 有序数组中找两个数。
- 连续子数组/子串。
- 需要边走边维护一段范围。

## 对撞指针

```python
left, right = 0, len(nums) - 1
while left < right:
    s = nums[left] + nums[right]
    if s < target:
        left += 1
    elif s > target:
        right -= 1
    else:
        print(left, right)
        break
```

## 滑动窗口

```python
left = 0
seen = set()
ans = 0
for right, ch in enumerate(s):
    while ch in seen:
        seen.remove(s[left])
        left += 1
    seen.add(ch)
    ans = max(ans, right - left + 1)
```
