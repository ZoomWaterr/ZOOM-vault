# 前缀和

前缀和就是从头累加

~~~py
原数组：  1  2  3  4  5
前缀和：  1  3  6  10 15
# 每个位置 = 它自己 + 前面所有数之和
~~~

~~~py
#实现代码：
for i in range(1, len(a)):
    a[i] += a[i-1]
~~~

# 理解差分数组

本质：差分数组记录的就是在区间内的***变化量***，前缀和把变化量***积累***成真实值

~~~python
**差分数组是解决"区间加法"的神器**

核心口诀就两句话：
```
在L位置 +C   → 表示从L开始，每人多C个
在R+1位置 -C → 表示到R结束，R+1之后不多了
```

举个例子，5个小朋友，区间[2,4]每人+3：
```
差分数组d：  0   0   0   0   0   0
                 ↑               ↑
             d[2]+=3         d[5]-=3

变成：       0   0   3   0   0  -3
```

然后对差分数组求**前缀和**，就还原出真实结果：
```
d:           0   0   3   0   0  -3
前缀和后：   0   0   3   3   3   0  ← 这就是答案！
~~~

**"一个范围内，批量做同一件事"** 这就是——区间操作！

***区间操作就想差分***

**差分核心思想：**

+ 与其真的去填满L到R，不如只**打两个标记** "L这里开始多了C" 和 "R+1这里少了C" 最后统一结算（前缀和）

## 取整

### 向上取整

`(a + b - 1) // b`	也是向上取整，但a和b都得是整数

~~~python
import math

math.ceil()	#这就是最简便向上取整的东西
~~~

### 向下取整

| 方法            | 适用场景                     | 例子                   | 说明                           |
| --------------- | ---------------------------- | ---------------------- | ------------------------------ |
| `//`            | 整数除法、求平均（整数结果） | `sum(can) // len(can)` | 直接对结果向下取整，**最常用** |
| `math.floor(x)` | 任意数字（正 / 负 / 小数）   | `math.floor(5.9) → 5`  | 标准向下取整，需导入 `math`    |
| `int(x)`        | 仅正数                       | `int(5.9) → 5`         | 负数会出错，不推荐通用场景     |
