# 变量命名救命规则

tags: #刷题/习惯 #python/变量

## 应该怎么做

嵌套循环不要默认 `i/j`。变量名必须说出它在题里的身份。

## 替换表

| 不稳写法 | 更稳写法 |
|---|---|
| `i` | `轮数`、`行`、`天数`、`当前位置` |
| `j` | `列`、`左边位置`、`比较位置` |
| `n` | `人数`、`数字个数`、`天数` |
| `s` | `原字符串`、`单词`、`输入行` |
| `cnt` | `出现次数`、`满足数量` |

## 车厢重组提醒

你曾经把冒泡排序里的外层轮数和内层当前位置混在一起。正确思路：

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
        if nums[当前位置] > nums[当前位置 + 1]:
            nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
```
