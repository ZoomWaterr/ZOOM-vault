## 素数判断 ⭐

```python
def is_prime(n: int) -> bool:
    if n < 2:
        return False
    # 只需检查到 sqrt(n)：如果 n=a×b，a 和 b 必有一个 ≤ sqrt(n)
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

⏱️ O(√n)

## 埃氏筛 — 批量求素数 ⭐⭐

```python
def sieve(n: int) -> list[int]:
    """返回 2~n 之间的所有素数"""
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False

    for i in range(2, int(n ** 0.5) + 1):
        if is_prime[i]:
            # 从 i² 开始标记（更小的倍数已被之前的素数筛过）
            for j in range(i * i, n + 1, i):
                is_prime[j] = False

    return [i for i in range(2, n + 1) if is_prime[i]]

# sieve(20) → [2, 3, 5, 7, 11, 13, 17, 19]
```

⏱️ O(n log log n)

### 手写 GCD：辗转相除法

```python
def gcd(a, b):
    """gcd(a,b) = gcd(b, a%b)，辗转直到余数为 0"""
    while b:
        a, b = b, a % b
    return a

# 演示：gcd(48, 18)
# 48%18=12 → 18%12=6 → 12%6=0 → 答案 6
```

### 手写 LCM

```python
def lcm(a, b):
    return a // gcd(a, b) * b    # 先除后乘防溢出
```

## 快速幂 ⭐⭐

> 💡 求 xⁿ mod m，指数折半、底数平方。O(log n)。

```
核心思想：x¹³ = 把 13 拆成二进制
13 = 1101₂ = 8 + 4 + 1
x¹³ = x⁸ × x⁴ × x¹
     └ 二进制哪一位是 1，就乘上对应的幂
```

```python
def fast_pow(base, exp, mod):
    """计算 (base^exp) % mod，复杂度 O(log exp)"""
    result = 1
    while exp:
        if exp & 1:                          # 当前最低位是 1
            result = (result * base) % mod
        base = (base * base) % mod           # 底数平方
        exp >>= 1                            # 指数右移
    return result

# fast_pow(2, 10, 10**9+7) → 1024
```

> Python 内置 `pow(base, exp, mod)` 也是 O(log n)，可以直接用。

## 模运算常用公式 ⭐⭐

```python
MOD = 10 ** 9 + 7    # 竞赛最常用的取模数

# 加、减、乘 都可以先取模再算
(a + b) % MOD = ((a % MOD) + (b % MOD)) % MOD
(a - b) % MOD = ((a % MOD) - (b % MOD) + MOD) % MOD  # ★ 减法要 +MOD 防负数！
(a * b) % MOD = ((a % MOD) * (b % MOD)) % MOD

# 除法不能直接取模！需要用费马小定理求逆元（进阶）
# (a / b) % MOD = (a * pow(b, MOD-2, MOD)) % MOD
```

## 💬 德远的笔记

> 素数判断那行 `for i in range(2, int(n ** 0.5) + 1)` 一开始不理解为什么只到根号 n。后来想清楚了：如果 n 有一个大于 √n 的因子，那它对应的因子必然小于 √n——所以查到 √n 就够了。这行把 O(n) 变成了 O(√n)，差了好几个量级。
>
> 辗转相除法的 while 循环一开始也绕。其实本质就是：gcd(a,b) = gcd(b, a%b)，不断替换直到 b 变成 0。
>
> 模运算里减法要 +MOD 这个细节第一次做的时候 WA 了好几次，后来每次取模都下意识在减完后再加一次 MOD 取模。
