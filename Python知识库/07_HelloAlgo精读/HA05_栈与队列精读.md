# HA05_栈与队列精读

对应原书：第 5 章 栈与队列。

## 本章一句话

栈处理“最近的东西”，队列处理“最早的东西”。

## 必须掌握

- 栈：后进先出。
- 队列：先进先出。
- 双向队列：两端都能进出。
- Python 队列优先用 `deque`。

## Python 小练习

1. 用栈判断括号是否匹配。
2. 用队列模拟排队。
3. 用 `deque` 写一个 BFS 骨架。

## 做题信号

- 最近一个未匹配：栈。
- 层层扩散、最短步数：队列 + BFS。
- 两端都操作：双向队列。
