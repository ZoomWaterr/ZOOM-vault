# ⚠️ 高频易错点

> 这里放的不是泛泛而谈，是从你的代码和基础刷题阶段里真正容易反复出现的坑。

## 输入没转类型

```python
n = input()      # n 是字符串
n = int(input()) # n 才是整数
```

## 输出列表格式

```python
print(nums)   # [1, 2, 3]
print(*nums)  # 1 2 3
```

## `range` 右边取不到

```python
for i in range(1, n + 1):
    ...
```

## 字符串不会原地修改

```python
s.strip()      # 没接回来，白干
s = s.strip()  # 对
```

## 嵌套循环变量混

`i`、`j` 没错，但新手期容易打架。遇到二维、冒泡、矩阵题，可以写成 `行`、`列`、`轮数`、`当前位置`。

## 数学思路不清楚就退回暴力

这不是不能退，暴力是保底方案。但如果你已经看见规律，就先手算 5 个小例子，让公式站稳。
