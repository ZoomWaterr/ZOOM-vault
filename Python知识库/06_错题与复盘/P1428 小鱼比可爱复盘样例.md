# P1428 小鱼比可爱复盘样例

## 来源

洛谷：P1428 小鱼比可爱  
源码：`F:\Code\PythonCode\洛谷\【入门4】数组\P1428 小鱼比可爱.py`

## 题目在问什么

对每条鱼，统计它左边有多少条鱼比它小。

## 正确模型

[[枚举]] + [[for循环]]。

这是典型的“当前对象和它左边所有对象比较”。

## 关键变量

- `i`：当前鱼的位置。
- `j`：当前鱼左边某条鱼的位置。
- `cnt`：当前鱼左边比它小的数量。
- `ans`：每条鱼的答案。

## 为什么是双层循环

外层枚举每条鱼：

```python
for i in range(n):
```

内层只看它左边：

```python
for j in range(i):
```

`range(i)` 的含义是 `0, 1, ..., i-1`，刚好是当前鱼左边。

## 完整代码

```python
n = int(input())
nums = list(map(int, input().split()))

ans = []
for i in range(n):
    cnt = 0
    for j in range(i):
        if nums[j] < nums[i]:
            cnt += 1
    ans.append(cnt)

print(*ans)
```

## 复杂度

外层 `n` 次，内层平均也接近 `n` 次，所以是 `O(n^2)`。

这题数据范围允许，所以不用优化。

## 易错点

- 内层不能写 `range(n)`，否则会把右边也算进去。
- `cnt` 每处理一条新鱼都要重新变成 0。
- `ans.append(cnt)` 要放在内层循环结束后。

## 下次识别信号

看到“左边有多少个满足条件”，先想到：

```python
for i in range(n):
    for j in range(i):
```
