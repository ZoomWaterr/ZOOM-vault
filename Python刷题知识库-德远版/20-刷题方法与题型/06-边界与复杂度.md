# 边界与复杂度

## 先看这个

边界决定你会不会 WA，复杂度决定你会不会 TLE。

## 边界样例

- 最小：`n=0`、`n=1`、空字符串。
- 相等：所有数一样、刚好达到条件。
- 极端：最大值、最小值、全是负数。
- 格式：末尾有没有点、空格、换行。

## 复杂度速查

| 写法 | 复杂度 | 备注 |
|:---|:---|:---|
| 单层循环 | O(n) | 基础可接受 |
| 双层循环 | O(n²) | n 到 10^5 基本不行 |
| 排序 | O(n log n) | 很常用 |
| 字典/集合查找 | 平均 O(1) | 适合频繁判断存在 |
| 二分 | O(log n) | 需要有序或单调 |
