# 排序

tags: #算法/排序 #python/列表

## 应该怎么做

刷题里优先用 Python 自带排序。

```python
nums.sort()
nums.sort(reverse=True)
new_nums = sorted(nums)
```

## 这在干什么

排序把乱序数据变成有序数据。有序之后，相邻关系、最大最小、贪心选择、二分查找都会变简单。

## 自定义排序

```python
students = [("A", 90), ("B", 80)]
students.sort(key=lambda x: x[1])
```

## 冒泡排序理解版

```python
def 冒泡排序(nums):
    n = len(nums)
    for 轮数 in range(n):
        for 当前位置 in range(n - 1 - 轮数):
            if nums[当前位置] > nums[当前位置 + 1]:
                nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
```

## 我的提醒

你在“车厢重组”里出现过 `nums[j]` 和 `nums[i]` 混乱。嵌套循环不要用没意义的 `i/j` 硬撑：

- 外层叫 `轮数`
- 内层叫 `当前位置`

名字一清楚，逻辑就不容易写歪。
