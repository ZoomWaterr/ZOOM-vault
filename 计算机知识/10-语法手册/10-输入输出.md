# 输入输出 · IO 模板

> 刷题 80% 的代码都是输入处理 + 输出格式。把模板背熟，剩下的时间留给算法。

---

## 输入

### 一行一个数

```python
n = int(input())
x = float(input())
```

### 一行多个数

```python
a, b = map(int, input().split())                    # 两个整数
x, y, z = map(float, input().split())               # 三个浮点数
nums = list(map(int, input().split()))               # → 整数列表
```

### n 行，每行一个数

```python
n = int(input())
data = [int(input()) for _ in range(n)]
```

### n 行，每行多个数

```python
n = int(input())
data = [list(map(int, input().split())) for _ in range(n)]
```

### 二维矩阵

```python
n, m = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(n)]
```

### 读到 EOF（文件末尾）

```python
import sys
for line in sys.stdin:
    a, b = map(int, line.split())
    # 处理...
```

### 极速输入（数据量 > 10⁵）

```python
import sys
raw = sys.stdin.buffer.read().split()
n = int(raw[0])
nums = list(map(int, raw[1:]))
```

## 输出

### 基本输出

```python
print(答案)                       # 自动换行
print(答案, end=" ")              # 不换行，末尾空格
```

### 列表空格分隔 ← 最常用

```python
a = [1, 2, 3, 4, 5]

print(*a)                         # → 1 2 3 4 5   解包，最简洁
print(" ".join(map(str, a)))      # → 1 2 3 4 5   join，速度更快
```

### 每行输出一个

```python
for x in a:
    print(x)

# 或
print("\n".join(map(str, a)))
```

### 格式控制

```python
print(f"{pi:.3f}")                # → 3.142       保留 3 位小数
print(f"{h:02d}:{m:02d}")         # → 08:05       补零对齐
print(f"{x:.6g}")                 # 6 位有效数字，自动删末尾 0
```

## 拆分 → 处理 → 输出 模板

```python
lst = input().split()              # 拆成字符串列表
for item in lst:
    processed = 对item的处理()
    print(processed, end=" ")      # 同行空格输出
```

## 读入 → 收集 → 输出 模板

```python
n = int(input())
result = []
for _ in range(n):
    x = int(input())
    result.append(处理(x))
print(*result)
```

## 输出速度对比

| 数据量 | 推荐方式 | 速度 |
|:---|:---|:---:|
| < 10⁴ | `input()` / `print()` | 够用 |
| 10⁵ ~ 10⁶ | `sys.stdin.readline()` | 快 |
| > 10⁶ | `sys.stdin.buffer.read().split()` | 最快 |

## 💬 德远的笔记

> `print(*a)` 是我觉得 Python 在竞赛里最爽的东西之一——C++ 要写循环打印，Python 一个星号搞定。还有 f-string 的格式控制 `f"{x:.3f}"`，做浮点输出的时候省了很多事。
>
> 不过也犯过错：`print(*a)` 输出 `[1, 2, 3]` 直接写了 `print(a)`，输出是 `[1, 2, 3]` 带方括号，OJ 判错。**OJ 要的是空格分隔的数字，不是 Python 的列表格式！**
