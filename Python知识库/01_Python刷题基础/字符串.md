# 字符串

字符串可以看成不可修改的字符列表。

## 常用操作

```python
s.split()
s.strip()
s.rstrip(".")
s.replace("a", "b")
s.find("ab")
s.upper()
s.lower()
s[::-1]
```

## 回文判断

```python
def is_palindrome(s):
    return s == s[::-1]
```

## 拼接输出

```python
words = ["I", "love", "Python"]
print(" ".join(words))
```

数字列表要先转字符串：

```python
print(" ".join(map(str, nums)))
```

## 易错点

- 字符串不可修改，`replace`、`strip` 都返回新字符串。
- 下标从 0 开始。
- `find` 找不到返回 `-1`。
