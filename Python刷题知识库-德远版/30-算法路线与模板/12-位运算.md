# 位运算

## 先看这个

位运算不是当前阶段最急，但常见套路要认识。

## 速查

| 写法 | 意思 |
|:---|:---|
| `a & b` | 按位与 |
| `a | b` | 按位或 |
| `a ^ b` | 异或 |
| `a << k` | 左移，乘 2^k |
| `a >> k` | 右移，除 2^k 向下取整 |

## 常见套路

```python
# 判断奇偶
if x & 1:
    print("odd")

# 统计 1 的个数
cnt = x.bit_count()

# 其他数都出现两次，找唯一出现一次
ans = 0
for x in nums:
    ans ^= x
```
