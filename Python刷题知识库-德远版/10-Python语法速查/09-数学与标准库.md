# 数学与标准库

## 先看这个

        标准库是刷题工具箱。知道什么时候用，比全部背下来更重要。

        ## 什么时候用

        取整、最大公约数、队列、统计、二分、堆、排列组合时。

        ## 怎么写

        ```python
        import math
from collections import Counter, deque, defaultdict
import bisect
import heapq

g = math.gcd(a, b)
ceil_div = (a + b - 1) // b
cnt = Counter(nums)
q = deque([start])
        ```

        ## 我容易摔在哪里

        `math.ceil(a / b)` 能用，但整数向上取整更推荐 `(a + b - 1) // b`，避免浮点误差。

## 常用标准库

| 工具 | 什么时候用 |
|:---|:---|
| `math.gcd` | 最大公约数 |
| `Counter` | 快速统计次数 |
| `defaultdict` | 字典默认值 |
| `deque` | BFS 队列、频繁头部弹出 |
| `bisect` | 有序列表二分 |
| `heapq` | 优先队列、前 K 大 |
| `itertools` | 排列组合、累积 |

## GCD 手写理解

```python
while b != 0:
    a, b = b, a % b
print(a)
```
