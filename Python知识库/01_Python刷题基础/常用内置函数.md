# 常用内置函数

刷题常用函数要熟到不用查。

| 函数 | 作用 |
|---|---|
| `len(x)` | 长度 |
| `sum(x)` | 求和 |
| `min(x)` | 最小值 |
| `max(x)` | 最大值 |
| `sorted(x)` | 返回排序后的新列表 |
| `abs(x)` | 绝对值 |
| `round(x, n)` | 四舍五入 |
| `enumerate(x)` | 同时拿下标和值 |
| `zip(a, b)` | 并行遍历 |

## 按特殊标准找最大

```python
words = ["a", "hello", "hi"]
print(max(words, key=len))
```

## 字典中找出现次数最多

```python
best = max(count, key=count.get)
```
