# 列表 list

> 刷题**用得最多**的容器。有序、可重复、可修改。

---

## 创建

```python
a = []                          # 空
a = [1, 2, 3]                   # 直接写
a = [0] * 5                     # [0,0,0,0,0] — int安全
a = list(map(int, input().split()))  # 输入→列表
```

> ⚠️ `[[0]*3]*3` 是错的！三行同一个列表。✅ `[[0]*3 for _ in range(3)]`

---

## 推导式

```python
[i**2 for i in range(1, 6)]                     # → [1,4,9,16,25]
[i for i in range(20) if i % 2 == 0]            # 筛选偶数
["偶数" if i%2==0 else "奇数" for i in range(5)] # 带判断
[[0]*m for _ in range(n)]                       # 二维列表
```

---

## 切片

```
a[开始:结束:步长]    前闭后开
```

```python
a[2:5]    # 下标2,3,4
a[:5]     # 开头到4
a[5:]     # 下标5到末尾
a[::2]    # 跳着取
a[::-1]   # 反转！
a[-3:]    # 最后3个
```

> ⚠️ `a[:-2]` 是到倒数第二个结束，步长是第三个参数。

---

## 添加

```python
a.append(x)       # 末尾加 — O(1)
a.insert(i, x)    # 位置i插入 — O(n)，后面全部后移
a.extend(lst)     # 合并另一个列表
```

---

## 删除 — pop vs remove vs del

> 🔥 **最容易搞混的三个。pop 用下标，remove 用值。**

```python
# pop(i) — 删下标i，返回被删的值
a = [10, 20, 30, 40]
x = a.pop(1)       # 删下标1(20), x=20, a变[10,30,40]
x = a.pop()        # 不写→删最后一个
a.pop(0)           # 删第一个 ← O(n)！后面全部前移

# remove(x) — 删第一个值为x的元素，不返回
a = [10, 20, 30, 20]
a.remove(20)       # 删第一个20, a变[10,30,20]
a.remove(100)      # 报错！100不在列表里

# del — 按下标删，不返回
del a[0]
del a[1:3]         # 删一片
```

### 什么时候用哪个？

| 场景 | 用什么 |
|:---|:---|
| 知道要删的**值** | `remove(x)` |
| 知道要删的**位置** | `pop(i)` |
| 删最后一个 | `pop()` |
| 删一片 | `del a[1:3]` |

> 🔥 `a.pop(max(a))` 想把最大值删掉 → 错！`max(a)` 返回的是值(比如99)，pop当成下标99→ IndexError。
> ✅ `a.remove(max(a))` 才是按值删。
>
> 🔥 `num = num.pop(0)` → num 变成了被删的那个整数，不再是列表。后面 `sum(num)` 全炸。

## 查找 / 排序 / 遍历

```python
a.index(x)         # 找x第一次出现的下标，找不到报错
a.count(x)         # x出现了几次
len(a)             # 长度

a.sort()           # 从小到大，改原列表
a.sort(reverse=True)
b = sorted(a)      # 返回新列表，不改原列表

# 遍历
for i, x in enumerate(a):   # 要下标和值
for x in a:                 # 只要值
```

---

## 时间复杂度

| 操作 | 复杂度 |
|:---|:---:|
| `a[i]` `a.append(x)` `a.pop()` | **O(1)** |
| `a.pop(0)` `a.insert(0,x)` `a.remove(x)` `x in a` | **O(n)** ⚠️ |

> 🔥 循环里 `if x in list` 是 O(n²)！转 set。循环里 `list.pop(0)` 是 O(n²)！用 deque。

## 💬 德远的笔记

> pop 和 remove 这个坑在 P5725 求三角形那题实打实踩过。想删最高分写了 `num.pop(max(num))`，pop 把 max 返回值当成了下标——列表才5个元素但 max 可能是99，直接炸。正确写法是 `num.remove(max(num))`。
>
> 还有 `num = num.pop(0)` 之后 num 不再是列表而是被删的那个整数，后面 `sum(num)` 全报错。pop 会返回被删的值，不需要赋回去。
