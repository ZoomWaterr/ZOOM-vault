# 动态规划

DP 不要先背公式，先理解一句话：

> 当前答案可以由更小的已知答案推出来，并且这些小答案会被重复使用。

## 题目信号

- 求最优值。
- 求方案数。
- 当前选择影响后续。
- 有重复子问题。
- 可以从小规模推到大规模。

## DP 五步

1. 定义状态：`dp[i]` 表示什么？
2. 写转移：`dp[i]` 从哪里来？
3. 定初始值：最小情况是什么？
4. 确定遍历顺序：从小到大？从前到后？
5. 返回答案：答案在 `dp[n]` 还是 `max(dp)`？

## 斐波那契例子

```python
n = int(input())
dp = [0] * (n + 2)
dp[1] = dp[2] = 1

for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])
```

这里：

- `dp[i]` 表示第 `i` 项答案。
- `dp[i]` 来自前两项。
- 初始值是 `dp[1]` 和 `dp[2]`。

## 常见类型

- 线性 DP。
- 背包 DP。
- 区间 DP。
- 树形 DP。
- 状态压缩 DP。

你现在先打穿线性 DP 和基础背包，不急着碰很复杂的类型。
