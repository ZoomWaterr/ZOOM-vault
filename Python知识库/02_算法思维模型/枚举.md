# 枚举

枚举就是把可能情况一个个试出来。它是很多算法的起点。

## 一句话理解

不会优化时，先想：我能不能把所有可能都试一遍？

## 单变量枚举

```python
for i in range(n):
    ...
```

## 双变量枚举

```python
for i in range(n):
    for j in range(i + 1, n):
        ...
```

这枚举所有 `i < j` 的数对。

## 枚举的价值

- 帮你得到正确的暴力解。
- 帮你看出重复计算在哪里。
- 很多优化就是从枚举改出来的。

## 从枚举到优化

| 暴力枚举 | 常见优化 |
|---|---|
| 每次查前面有没有 | 用哈希 |
| 每次算区间和 | 用前缀和 |
| 两层循环找有序数组答案 | 用双指针 |
| 枚举答案逐个试 | 用二分答案 |

## 易错点

- 枚举范围要不重不漏。
- 明确每个变量代表什么。
- 写完先估复杂度。
