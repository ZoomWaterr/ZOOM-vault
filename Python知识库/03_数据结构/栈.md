# 栈

栈是后进先出：最后放进去的，最先拿出来。

## Python 写法

```python
stack = []
stack.append(x)
x = stack.pop()
```

## 题目信号

- 括号匹配。
- 最近的一个未匹配元素。
- 撤销操作。
- 单调栈题。

## 括号匹配模板

```python
stack = []
pairs = {")": "(", "]": "[", "}": "{"}

for ch in s:
    if ch in "([{":
        stack.append(ch)
    elif ch in pairs:
        if not stack or stack[-1] != pairs[ch]:
            print("NO")
            break
        stack.pop()
else:
    print("YES" if not stack else "NO")
```
