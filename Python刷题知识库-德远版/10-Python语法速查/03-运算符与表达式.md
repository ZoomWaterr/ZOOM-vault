# 运算符与表达式

## 先看这个

        表达式最后要么算出一个值，要么算出一个真假。复杂条件先加括号，别让脑子硬扛。

        ## 什么时候用

        做判断、取整、求余、组合多个条件时。

        ## 怎么写

        ```python
        if 1 <= x <= 100 and x % 2 == 0:
    print("ok")

q = a // b
r = a % b
ceil_div = (a + b - 1) // b
        ```

        ## 我容易摔在哪里

        `/` 永远返回浮点数；`//` 才是整数除法。`and` 是全真才真，`or` 是有真就真。

## 常用表

| 写法 | 意思 |
|:---|:---|
| `a + b` | 加 |
| `a - b` | 减 |
| `a * b` | 乘 |
| `a / b` | 真除法，结果 float |
| `a // b` | 整除 |
| `a % b` | 取余 |
| `a ** b` | 次方 |

## 逻辑表达

```python
if x > 0 and y > 0:
    ...
if x == 0 or y == 0:
    ...
if not found:
    ...
```

## 别想复杂

不管条件多复杂，最后都只是在问：这一坨是 `True` 还是 `False`。
