# ⚡ 一分钟速查总表

## Python 写法

| 我要做什么 | 直接写 |
|---|---|
| 读整数列表 | `nums = list(map(int, input().split()))` |
| 空格输出列表 | `print(*nums)` |
| 数字列表拼字符串 | `print(" ".join(map(str, nums)))` |
| 统计次数 | `count[x] = count.get(x, 0) + 1` |
| 找最长 | `max(words, key=len)` |
| 反转 | `s[::-1]` |
| 正整数向上取整 | `(a + b - 1) // b` |
| 二维列表 | `grid = [[0] * m for _ in range(n)]` |

## 算法触发器

| 题目出现 | 想 |
|---|---|
| 区间和、多次查询 | 前缀和 |
| 区间批量加 | 差分 |
| 出现次数、频率最高 | 字典 / Counter |
| 连续子数组/子串 | 滑动窗口 / 前缀和 |
| 有序数组找位置 | 二分 |
| 最大值最小/最小值最大 | 二分答案 |
| 最短步数 | BFS |
| 所有方案 | 回溯 |
| 选或不选、最优值 | DP |
