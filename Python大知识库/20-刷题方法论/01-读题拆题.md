# 读题拆题

> 读题不是把字看完，而是把题目变成变量和操作。

## 三句话拆题

1. 输入给了我什么？
2. 我要反复做什么？
3. 最后输出什么？

## 四类关键词

| 题目出现 | 你要警觉 |
|:---|:---|
| 出现次数、最多、超过一半 | 字典计数 |
| 连续、子段、一段 | 前缀和、双指针 |
| 每个区间都加 | 差分 |
| 第 n 项、走楼梯、前几项 | 递推 |

## 手算样例

不要只看样例输入输出，要手算一遍中间过程。手算能帮你发现变量应该叫什么。
