## 应该怎么做

数学题不要在脑子里空想。先写小例子，再推规律。

```text
1. 手算 n=1,2,3,4,5
2. 看差值、倍数、周期
3. 写公式
4. 用 n=6 验证
5. 再写代码
```

## GCD / LCM

```python
import math

g = math.gcd(a, b)
l = math.lcm(a, b)
```

手写辗转相除：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

## 素数判断

```python
def is_prime(n):
    if n < 2:
        return False
    i = 2
    while i * i <= n:
        if n % i == 0:
            return False
        i += 1
    return True
```

## 快速幂

```python
ans = pow(a, b, mod)
```

手写版：

```python
def fast_pow(a, b, mod):
    ans = 1
    a %= mod
    while b:
        if b & 1:
            ans = ans * a % mod
        a = a * a % mod
        b >>= 1
    return ans
```
