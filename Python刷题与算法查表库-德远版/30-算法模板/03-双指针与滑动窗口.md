# 🪟 双指针 / 滑动窗口

## 🚀 快速定位

右边扩张，条件坏了就左边收缩。

## 💡 一句话

两个边界移动，避免枚举所有区间。

## 🎯 适用场景

- 连续子数组
- 连续子串
- 有序两数之和
- 最长/最短连续段

## 🧠 原理理解

labuladong 的窗口框架很好：扩大窗口、更新数据、收缩窗口、更新答案。

## 📝 标准模板

```python
left = 0
window = 0
for right, x in enumerate(nums):
    window += x
    while window > target:
        window -= nums[left]
        left += 1
```

## ⚠️ 易错点

- 有负数时窗口和不单调
- left/right 含义要固定
- 更新答案的位置要想清楚

> [!tip] 💬 德远的笔记
> 滑窗不是死模板，关键是窗口内维护了什么。
