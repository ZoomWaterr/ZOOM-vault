# 图与并查集

**快速结论**
图处理点和边；并查集处理连通分组。

**什么时候用**
- 判断两个元素是否属于同一组。
- 合并集合。
- 朋友圈、连通块。
- Kruskal 最小生成树入门。

**图的邻接表**
```python
graph = [[] for _ in range(n)]
for _ in range(m):
    a, b = map(int, input().split())
    graph[a].append(b)
    graph[b].append(a)
```

**并查集模板**
```python
parent = list(range(n))
size = [1] * n


def find(x):
    if parent[x] != x:
        parent[x] = find(parent[x])
    return parent[x]


def union(a, b):
    ra, rb = find(a), find(b)
    if ra == rb:
        return False
    if size[ra] < size[rb]:
        ra, rb = rb, ra
    parent[rb] = ra
    size[ra] += size[rb]
    return True


def connected(a, b):
    return find(a) == find(b)
```

**连通块数量**
```python
cnt = n
for a, b in edges:
    if union(a, b):
        cnt -= 1
```

**复杂度**
路径压缩 + 按大小合并后，单次操作接近 `O(1)`。

**易错点**
- 下标是 0 基还是 1 基要统一。
- `find` 要路径压缩。
- 并查集只回答连通关系，不回答最短路径。
