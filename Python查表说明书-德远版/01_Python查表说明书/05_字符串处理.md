# 字符串处理

tags: #python/字符串 #刷题/高频

## 应该怎么做

先决定：你是在“拆字符串”“清洗字符串”“反转字符串”，还是“统计字符”。

## 这在干什么

字符串是字符的容器：有序、不可修改、可遍历。任何修改都会产生新字符串。

## 高频模板

```python
s = input()
words = s.split()
s = s.strip()
s = s.rstrip(".")
s = s.replace(".", " ")
rev = s[::-1]
```

## 拼接输出

```python
words = ["I", "love", "Python"]
print(" ".join(words))
```

数字列表要先转字符串：

```python
nums = [1, 2, 3]
print(" ".join(map(str, nums)))
```

## 找最长单词

```python
words = input().replace(".", " ").split()
long_word = max(words, key=len)
print(long_word)
```

## 我的提醒

你代码里写过：

```python
words = input().rstrip(".").split()
long_word = max(words, key=len)
```

这在样例末尾只有句号时没问题，但如果中间也有标点，最好先 `replace(".", " ")` 再 `split()`。

## 什么时候翻它

- 题目出现“单词”“字符”“回文”“翻转”。
- 要把列表输出成一行。
- 要按长度、出现次数等非数字标准找最大值。
