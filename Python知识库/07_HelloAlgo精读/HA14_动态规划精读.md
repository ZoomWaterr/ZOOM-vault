# HA14_动态规划精读

对应原书：第 14 章 动态规划。

## 本章一句话

DP 是把重复子问题的答案存起来，再按状态转移推出更大问题。

## 必须掌握

- 暴力搜索会重复计算。
- 记忆化搜索是自顶向下。
- 动态规划是自底向上。
- 状态定义是第一核心。
- 状态转移是第二核心。
- 背包是 DP 入门重点。

## Python 小练习

1. 斐波那契：递归、记忆化、DP 三种写法。
2. 爬楼梯。
3. 0-1 背包基础版。
4. 完全背包基础版。

## 做题信号

- 求最大/最小。
- 求方案数。
- 当前结果依赖前面。
- 有重复子问题。
- 选择会影响后续。

## DP 五问

```text
dp[i] 表示什么？
dp[i] 从哪里来？
初始值是什么？
按什么顺序遍历？
答案在哪里？
```
