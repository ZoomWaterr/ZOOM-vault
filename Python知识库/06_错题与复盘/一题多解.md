# 一题多解

有些题值得记录多种解法，因为它能帮你看见算法之间的关系。

## 模板

```md
# 题目名

## 解法一：暴力
思路：
复杂度：
为什么慢：

## 解法二：优化
思路：
复杂度：
优化了哪里：

## 解法三：更优或更简洁
思路：
复杂度：
代价：

## 我的选择
比赛时我会用哪种？为什么？
```

## 什么时候值得做一题多解

- 暴力和优化差别很明显。
- 一个题能用哈希，也能用排序。
- 一个题能用 DFS，也能用 BFS。
- 一个题能用 DP，也能用贪心，但需要证明。
