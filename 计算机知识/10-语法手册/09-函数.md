## 参数

```python
# 位置参数
reg_stu("德远", 18, "男", "北京")

# 关键字参数
reg_stu(name="德远", age=18, gender="男")

# 默认参数(必须在普通参数之后)
def reg_stu(name, age, gender, city="北京"):
    ...

# 不定长参数
def f(*args, **kwargs):   # args→元组, kwargs→字典
    ...
```

## 返回值与解包

```python
def min_max(arr):
    return min(arr), max(arr)       # 本质返回元组

最小值, 最大值 = min_max([3,1,4,1,5])  # 解包接收

# 拓展解包
a, *b, c = (9, 2, 34, 123)     # a=9, b=[2,34], c=123
```
