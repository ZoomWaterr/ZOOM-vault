# 哈希表与计数

## 快速结论

需要快速回答“出现过没有、出现几次、对应什么”，就用哈希表，也就是 Python 的 `dict` / `set`。

## 什么时候用

- 统计频率。
- 判断元素是否出现过。
- 两数之和。
- 前缀和计数。
- 建立映射关系。

## 基础计数模板

```python
cnt = {}
for x in nums:
    cnt[x] = cnt.get(x, 0) + 1
```

## 找最高频

```python
best = max(cnt, key=cnt.get)
print(best, cnt[best])
```

## 两数之和模板

```python
pos = {}
for i, x in enumerate(nums):
    need = target - x
    if need in pos:
        print(pos[need], i)
        break
    pos[x] = i
```

## Counter 模板

```python
from collections import Counter

cnt = Counter(nums)
print(cnt.most_common(1))
```

## 前缀和 + 哈希

```python
cnt = {0: 1}
pre = 0
ans = 0

for x in nums:
    pre += x
    ans += cnt.get(pre - k, 0)
    cnt[pre] = cnt.get(pre, 0) + 1
```

## dict 和 set 怎么选

| 需求 | 用 |
|---|---|
| 只判断有没有出现 | `set` |
| 记录次数 | `dict` / `Counter` |
| 记录位置 | `dict` |
| 记录映射关系 | `dict` |

## 复杂度

平均插入、查询、删除都是 `O(1)`。

## 易错点

- `d[key]` 在 key 不存在时报错；安全写法用 `d.get(key, default)`。
- `set` 不记录次数。
- 字典遍历键值对用 `for k, v in d.items()`。
