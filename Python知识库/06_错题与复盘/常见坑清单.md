# 常见坑清单

## 输入输出

- 忘记 `input()` 是字符串。
- `map` 没有转成 `list`。
- `join` 拼数字列表时忘记 `map(str, ...)`。

## 循环

- `range(n)` 不包含 `n`。
- 双层循环没分清 `i` 和 `j`。
- 内层循环边界多算或少算。

## 下标

- 题目 1 下标，代码 0 下标。
- `r` 是否包含。
- `r + 1` 越界。

## 字典

- 直接 `d[x]` 导致 key 不存在报错。
- 计数时忘记默认值。

## 算法

- 没看数据范围就写暴力。
- DFS 忘记 visited。
- BFS 出队才标记导致重复入队。
- DP 状态定义不清就写转移。
