# 哈希计数

## 一句话理解

看到“出现次数”，就用字典记录每个东西出现几次。

## 题目信号

- 某个数出现了几次。
- 找出现最多的字符。
- 判断有没有超过一半。
- 统计不同元素数量。

## 模板

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

## 找出现最多

```python
best = max(count, key=count.get)
print(best, count[best])
```

## 判断超过一半

```python
for x, c in count.items():
    if c > n // 2:
        print(x)
        break
else:
    print("no")
```

## 复杂度

- 时间：`O(n)`。
- 空间：`O(k)`，`k` 是不同元素个数。
