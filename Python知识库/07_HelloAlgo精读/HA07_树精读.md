# HA07_树精读

对应原书：第 7 章 树。

## 本章一句话

树是层级结构，树题的核心通常是递归遍历。

## 必须掌握

- 根、父节点、子节点、叶子节点。
- 二叉树最多两个孩子。
- 前序、中序、后序遍历。
- 层序遍历本质是 BFS。
- 二叉搜索树利用左小右大的性质。
