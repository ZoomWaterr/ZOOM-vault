# 前缀和与差分

---

## 前缀和 ⭐

> 从头累加。O(1) 查区间和。

```python
# 构造（原地）
nums = [1, 2, 3, 4, 5]
for i in range(1, len(nums)):
    nums[i] += nums[i - 1]         # → [1, 3, 6, 10, 15]

# 查询区间[L,R]和
def range_sum(pre, L, R):
    return pre[R] if L == 0 else pre[R] - pre[L - 1]
```

🎯 多次查询区间和 · 子数组和为K(前缀和+哈希表)

## 差分数组 ⭐⭐

> 记录变化量。区间加减只改两个位置。前缀和还原。

```
5个小朋友，第2~4个每人+3：
差分：[0, 0, 0, 0, 0, 0]
           ↑+3       ↑-3   (L=2加, R+1=5减)
变： [0, 0, 3, 0, 0,-3]
前缀和还原：[0, 0, 3, 3, 3, 0]  ← 第2~4位每人+3 ✓
```

```python
n, m = map(int, input().split())
d = [0] * (n + 2)                    # +2防R+1越界
for _ in range(m):
    L, R, C = map(int, input().split())
    d[L] += C
    d[R + 1] -= C                    # R+1，不是R！
for i in range(1, n + 1):
    d[i] += d[i - 1]                 # 前缀和还原
print(*d[1:n + 1])
```

⏱️ O(n+m) vs 暴力 O(n×m)

🎯 "区间批量加减"→立刻想差分

### ⚠️ 标记是L加、R+1减；数组多开2位；最后必须前缀和还原
