**快速结论**
排序让数据有序；二分利用有序性或答案单调性，每次砍掉一半。

**Python 排序**

```python
nums.sort()
nums.sort(reverse=True)
new_nums = sorted(nums)

```

自定义排序：

```python
items.sort(key=lambda x: x[1])

```

**bisect 查位置**

```python
import bisect

left_pos = bisect.bisect_left(arr, x)    # 第一个 >= x
right_pos = bisect.bisect_right(arr, x)  # 第一个 > x

```

**手写二分查找**

```python
left, right = 0, len(arr) - 1
while left <= right:
    mid = (left + right) // 2
    if arr[mid] == target:
        print(mid)
        break
    elif arr[mid] < target:
        left = mid + 1
    else:
        right = mid - 1

```

**二分答案模板**
找最小可行值：

```python
def check(x):
    # x 是否可行
    return True

left, right = low, high
while left < right:
    mid = (left + right) // 2
    if check(mid):
        right = mid
    else:
        left = mid + 1

print(left)

```

**什么时候二分答案**
- “最大值最小”。
- “最小值最大”。
- 答案越大越容易满足，或越小越容易满足。
- 可以写出 `check(x)` 判断某个答案是否可行。

**复杂度**
排序 `O(n log n)`；二分查找 `O(log n)`；二分答案 `O(logV * check复杂度)`。

**易错点**
- 二分答案必须有单调性。
- `bisect_left` 和 `bisect_right` 含义不同。
- 手写二分要保证区间每轮都缩小。
- 嵌套循环排序题里，外层轮数和内层当前位置不要混。
