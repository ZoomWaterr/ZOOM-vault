## collections — 数据结构全家桶

### deque — 双端队列（BFS 标配）

```python
from collections import deque

q = deque()
q.append("新任务")        # 从右边进
q.appendleft("紧急任务")  # 从左边进（插队）
q.pop()                   # 从右边出
q.popleft()               # 从左边出 ← BFS 用这个

# ⚠️ list.pop(0) 是 O(n)，deque.popleft() 是 O(1)！
# 需要频繁从头部删除时，必须用 deque 而不是 list

# BFS 经典用法：
q = deque([起点])
while q:
    cur = q.popleft()     # 取出队首
    for nxt in 相邻节点:
        q.append(nxt)     # 加入队尾
```

### Counter — 自动统计频率

```python
from collections import Counter

c = Counter(["张", "李", "张", "张", "李"])
c["张"]                   # → 3
c.most_common(2)          # → [("张", 3), ("李", 2)]  前两名
```

### defaultdict — 不用判断 key 存不存在

```python
from collections import defaultdict

# 普通写法：
count = {}
for name in names:
    if name in count:
        count[name] += 1
    else:
        count[name] = 1

# defaultdict 写法：
count = defaultdict(int)      # 默认值是 0
for name in names:
    count[name] += 1          # 直接 ++，不用判断！

# 其他常用默认值：
groups = defaultdict(list)   # 默认空列表，用于分组
uniq = defaultdict(set)      # 默认空集合
```

## math — 数学工具箱

```python
import math

math.gcd(12, 8)          # → 4         最大公约数
math.lcm(12, 8)          # → 24        最小公倍数 (Python 3.9+)
math.isqrt(10)           # → 3         整数平方根（向下取整）
math.comb(5, 2)          # → 10        C(5,2) 组合数
math.ceil(3.1)           # → 4         向上取整
math.floor(3.9)          # → 3         向下取整
```

## bisect — 有序列表的二分查找

```python
import bisect

arr = [2, 3, 5, 7, 11]

# bisect_left：找「第一个 >= 目标值」的位置
bisect.bisect_left(arr, 5)    # → 2（arr[2]=5）
bisect.bisect_left(arr, 6)    # → 3（6 应插在 5 和 7 之间）

# bisect_right：找「第一个 > 目标值」的位置
bisect.bisect_right(arr, 5)   # → 3
```

## heapq — 堆 / 优先队列

```python
import heapq

# 小顶堆
heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappop(heap)           # → 1  弹出最小的

# 大顶堆：存负数！
heapq.heappush(heap, -x)
max_val = -heapq.heappop(heap)

# 持续维护前 K 大
topk = []
for x in data:
    heapq.heappush(topk, x)
    if len(topk) > K:
        heapq.heappop(topk)   # 踢出最小的
```

## itertools — 排列组合

```python
from itertools import combinations, permutations, accumulate

# 组合 C(n,k)：选 k 个，不关心顺序
list(combinations([1,2,3], 2))    # → [(1,2), (1,3), (2,3)]

# 排列 P(n,k)：选 k 个，顺序重要
list(permutations([1,2,3], 2))    # → [(1,2),(1,3),(2,1),(2,3),(3,1),(3,2)]

# accumulate — 一行前缀和
list(accumulate([1,2,3,4,5]))     # → [1, 3, 6, 10, 15]
```

## 💬 德远的笔记

> `collections.Counter` 太好用了，统计频率一行搞定，再也不用自己写 `count.get(x, 0) + 1`。但也有代价——引入外部库会让代码变长一点点。简单统计我一般直接 dict.get 解决了。
>
> `deque` 一定要记住：**BFS 要么用它，要么自己用 list 然后 `pop(0)` 然后超时**。这是 TLE 的经典原因。
