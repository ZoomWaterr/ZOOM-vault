# 数学工具

## 先看这个

数学工具题不一定难，很多是固定写法。关键是别看到数学就先退缩。

## 最大公约数

```python
import math
g = math.gcd(a, b)
l = a * b // g
```

## 素数判断

```python
def is_prime(x):
    if x < 2:
        return False
    i = 2
    while i * i <= x:
        if x % i == 0:
            return False
        i += 1
    return True
```
