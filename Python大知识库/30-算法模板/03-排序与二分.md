## Python 自带排序 ⭐⭐

> 刷题直接用，不需要手写排序。

```python
a.sort()                      # 从小到大，直接修改原列表
a.sort(reverse=True)          # 从大到小
b = sorted(a)                 # 返回新列表，原列表不变

# 自定义排序规则
students = [("张三", 85), ("李四", 92), ("王五", 78)]
students.sort(key=lambda s: s[1])    # 按成绩排序
```

⏱️ O(n log n)

## 冒泡排序 ⭐

### 💡 一句话理解

每轮把当前最大的数**冒**到最后面，就像气泡浮出水面。

```
演示：把 [4, 3, 2, 1] 从小到大排

第 1 轮（把 4 冒到最后）：
  [4,3,2,1] → 4>3 交换 → [3,4,2,1] → 4>2 交换 → [3,2,4,1] → 4>1 交换 → [3,2,1,4] ✓

第 2 轮（忽略最后的 4）：
  [3,2,1,4] → 3>2 交换 → [2,3,1,4] → 3>1 交换 → [2,1,3,4] ✓

第 3 轮（忽略最后两个）：
  [2,1,3,4] → 2>1 交换 → [1,2,3,4] ✓  完成！
```

### 📝 代码模板

```python
def bubble_sort(nums: list[int]) -> None:
    n = len(nums)
    for 轮数 in range(n):
        swapped = False                   # 优化：一轮都没交换=已排好
        for 当前位置 in range(n - 1 - 轮数):
            #         -1：比较 j 和 j+1 防越界
            #         -轮数：最后「轮数」个已经排好了
            if nums[当前位置] > nums[当前位置 + 1]:
                nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
                swapped = True
        if not swapped:
            break
```

### ⚠️ 最常见的 bug（我自己踩过）

**内层循环比较的是 `j` 和 `j+1`（相邻两个），不是 `j` 和 `i`！**
写成 `if nums[j] > nums[i]` 就全乱了——`i` 是外层轮数，跟内层比较毫无关系。

✅ 解法：给外层叫「轮数」，内层叫「当前位置」，名字一清二楚就不会写错。

## 二分查找 ⭐⭐

### 💡 一句话理解

每次**砍掉一半范围**，在有序数据中快速定位。

### 标准库版（日常首选）

```python
import bisect

arr = [2, 3, 5, 7, 11]
bisect.bisect_left(arr, 5)    # → 2  第一个 >= 5 的位置
bisect.bisect_right(arr, 5)   # → 3  第一个 > 5 的位置
```

### 手写版（理解原理）

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2   # 防溢出（Python 不怕但习惯好）
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

### 二分答案（"最大的最小" / "最小的最大"）

> 如果题问"最大值最小是多少"，且答案越大越容易满足条件（单调性）→ **二分答案值**。

```python
lo, hi = 1, max_val           # 答案在 [lo, hi] 之间
while lo < hi:
    mid = (lo + hi) // 2
    if check(mid):            # 这个答案可行吗？
        hi = mid              # 可行 → 试试能不能更小
    else:
        lo = mid + 1          # 不可行 → 放宽条件
ans = lo
```

### 🎯 触发信号

- 数组有序 → 二分查找
- "最大值最小 / 最小值最大" → 二分答案
- 答案范围已知且有单调性 → 二分答案
