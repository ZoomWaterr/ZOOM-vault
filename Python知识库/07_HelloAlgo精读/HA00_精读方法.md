# HA00_精读方法

`hello-algo` 的优势是图解清楚、结构完整、代码能跑。你的目标不是记住书里的每句话，而是把每章转化成三种能力。

## 三种能力

### 1. 说人话

看完一节后，能用一句话解释：

```text
这个东西解决什么问题？
```

例如：

> 哈希表就是用额外空间换快速查找。

### 2. 写代码

每章至少手写一个 Python 小实现。

不追求工业级，只追求你知道它怎么动。

### 3. 认题型

读完后必须能回答：

```text
题目出现什么信号，我要想到它？
```

## 每章笔记模板

```md
## 本章一句话
...

## 我必须掌握的概念
...

## Python 要会写
...

## 做题信号
...

## 和我的旧题连接
...

## 易错点
...
```

## 不建议的读法

- 只看图，不写代码。
- 一口气看很多章，不做题。
- 看到复杂章节就跳过基础章节。
- 把书中代码复制过来就算会了。

## 最推荐的读法

读一节，关掉书，自己写：

```text
这个概念是：
它解决：
Python 里我会这样写：
题目里看到：
```
