## 常见错误速查

| 报错 | 原因 | 修 |
|:---|:---|:---|
| IndexError | 下标越界 | 数组多开几位 |
| KeyError | dict直接取 | 换 `d.get(key,默认)` |
| RecursionError | DFS太深 | `sys.setrecursionlimit(1000000)` |
| ZeroDivisionError | 除0 | 先 `if b!=0` |
| ValueError | 输入格式不对 | 加 `.strip()` |

| WA现象 | 原因 | 修 |
|:---|:---|:---|
| 样例对提交WA | 没测边界 | 跑上面清单 |
| 输出像但错 | /和//搞混 | 整数全用// |
| 数组莫名变了 | sort()改原列表 | 用sorted() |

| TLE原因 | 修 |
|:---|:---|
| `list.pop(0)` O(n) | `deque.popleft()` |
| `x in list` 循环里O(n²) | 转set |
| n=10⁵写O(n²) | 看n范围改算法 |

## 调试三法

**print大法**：输入→print确认、中间结果→print确认、输出

**二分定位**：注释一半代码→还错？bug在前半。对了？bug在刚注释那段。

**缩到最小**：大样例报错→缩成n=2~3手算对比。

**一次只改一处**，改完就交。
