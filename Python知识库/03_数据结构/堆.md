# 堆

堆适合动态维护最大值或最小值。

Python 的 `heapq` 是小根堆。

## 基本写法

```python
import heapq

heap = []
heapq.heappush(heap, x)
smallest = heapq.heappop(heap)
```

## 大根堆写法

```python
heapq.heappush(heap, -x)
largest = -heapq.heappop(heap)
```

## 题目信号

- 第 k 大 / 第 k 小。
- 动态取最小值。
- 合并代价最小。
- 优先队列。

## 复杂度

- 插入：`O(log n)`。
- 弹出最小：`O(log n)`。
- 看堆顶：`O(1)`。
