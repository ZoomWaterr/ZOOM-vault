# 双指针与滑动窗口

---

## 对撞指针 ⭐⭐

> 两个指针从**两端向中间**移动。把 O(n²) 降到 O(n)。

```python
# 有序数组中找两数之和 = target
def two_sum(arr, target):
    left, right = 0, len(arr) - 1
    while left < right:
        cur = arr[left] + arr[right]
        if cur == target:
            return (left, right)
        elif cur < target:
            left += 1              # 和太小 → 左指针右移
        else:
            right -= 1             # 和太大 → 右指针左移
    return (-1, -1)

# [2, 7, 11, 15], target=9 → (0, 1)  2+7=9
```

## 快慢指针 ⭐⭐

> 一快一慢同向走。经典应用：去重、链表环检测。

```python
# 原地删除有序数组中的重复元素，返回不重复元素个数
def remove_duplicates(nums):
    if not nums:
        return 0
    slow = 0                         # 已整理区域的最后位置
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow]: # 发现新元素
            slow += 1
            nums[slow] = nums[fast]  # 搬过去
    return slow + 1

# [1,1,2] → 前 2 个元素变成 [1,2]，返回 2
```

## 滑动窗口 ⭐⭐

> 💡 维护一个**满足条件的窗口**，右扩张、左收缩，像毛毛虫一样移动。

**为什么能用滑动窗口**：找"最长的满足某条件的连续子数组"——当你右边界往右扩时，左边界只会往右（不会向左回退）。这就是能从 O(n²) 降到 O(n) 的根本原因。

```
演示：找"无重复字符的最长子串"

  "a b c a b c b b"
   L R             窗口 = {a}          len=1
   L   R           窗口 = {a,b}        len=2
   L     R         窗口 = {a,b,c}      len=3
   L       R       a 重复！左缩
       L   R       窗口 = {b,c,a}      len=3
       L     R      b 重复！左缩
           L R     窗口 = {c,a,b}      len=3
最长 = 3（"abc" 或 "bca" 或 "cab"）
```

### 📝 代码模板

```python
def longest_no_repeat(s: str) -> int:
    window = set()                     # 当前窗口里都有谁
    left = 0
    best = 0

    for right, ch in enumerate(s):
        # ★ 关键：字符重复了就不停收缩左边界
        while ch in window:
            window.remove(s[left])
            left += 1
        # 到这里窗口里保证没有重复
        window.add(ch)
        best = max(best, right - left + 1)

    return best

# "abcabcbb" → 3
```

### ⚠️ 易错点

- 收缩用 `while` 不是 `if`——可能连续弹出多个元素
- 窗口更新和答案更新的先后顺序要搞对

### 🎯 触发信号

"子数组"、"子串"、"连续"、"最长/最短" → 几乎必然是滑动窗口
