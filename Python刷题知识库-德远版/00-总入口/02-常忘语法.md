# 常忘语法

## 先看这个

这里不是按教材顺序排，而是按你刷题现场最容易忘的动作排。

## 我容易摔在哪里

- `input()` 读进来永远是字符串，不转类型就会错。
- `/` 得到浮点数，OJ 要整数时通常用 `//`。
- 字符串方法不会改原字符串，`s.strip()` 后要接住返回值。
- `d[key]` 第一次访问不存在会炸，计数优先用 `get`。
