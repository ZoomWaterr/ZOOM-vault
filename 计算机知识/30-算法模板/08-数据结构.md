## 栈 ⭐⭐

> 💡 后进先出 (LIFO)。括号匹配、表达式求值。

```python
# ── 括号匹配 ──
def is_valid_brackets(s: str) -> bool:
    stack = []
    pair = {')': '(', ']': '[', '}': '{'}

    for ch in s:
        if ch in "([{":
            stack.append(ch)
        else:                              # 右括号
            if not stack or stack.pop() != pair[ch]:
                return False

    return not stack                       # 栈空了才算完全匹配

# "([])" → True   "([)]" → False   "(" → False
```

## 单调栈 ⭐⭐

> 💡 维护一个**始终保持单调**的栈。找"下一个更大/更小元素"专用。

```python
def next_greater(nums: list[int]) -> list[int]:
    """每个元素右边第一个比它大的值，没有就是 -1"""
    n = len(nums)
    ans = [-1] * n
    stack = []                             # 存下标！栈内值保持递减

    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            idx = stack.pop()
            ans[idx] = nums[i]             # i 是栈顶的「下一个更大」
        stack.append(i)

    return ans

# [2, 1, 2, 4, 3] → [4, 2, 4, -1, -1]
```

⏱️ O(n)（每个元素最多进栈出栈一次）

## 并查集 ⭐⭐

> 💡 管理**分组**和**连通关系**。"A 和 B 同组吗？"、"把 A 和 B 合并"。

```
每个集合是一棵树，节点指向父节点。
  find(x)：顺藤摸瓜找到根节点
  union(x,y)：把两棵树合并
  路径压缩：查找时把路径上所有节点直接指向根，加速
```

```python
class DSU:
    def __init__(self, n: int):
        self.parent = list(range(n))       # 一开始每人自己是老大

    def find(self, x: int) -> int:
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # 路径压缩
        return self.parent[x]

    def union(self, a: int, b: int) -> None:
        root_a = self.find(a)
        root_b = self.find(b)
        if root_a != root_b:
            self.parent[root_a] = root_b   # 一个老大认另一个当上级

    def same(self, a: int, b: int) -> bool:
        return self.find(a) == self.find(b)
```

⏱️ 均摊 O(α(n)) ≈ O(1)

## 堆（优先队列）⭐⭐

> 💡 总是取出最小（或最大）的那个。Top-K 问题必用。

```python
import heapq

# 小顶堆
heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappop(heap)                      # → 1  总是弹出最小的

# 大顶堆：存负数
heapq.heappush(heap, -x)
max_val = -heapq.heappop(heap)

# 持续维护前 K 大
def top_k(stream, k):
    heap = []
    for x in stream:
        heapq.heappush(heap, x)
        if len(heap) > k:
            heapq.heappop(heap)           # 踢出最小的
    return heap
```
