# 如何复盘一道题

复盘不是把题解抄一遍。复盘的目标是让你下次更快看出模型。

## 五问复盘法

1. 这题最朴素的暴力做法是什么？
2. 暴力慢在哪里？
3. 这题真正的模型是什么？
4. 哪个变量最关键？它代表什么？
5. 下次看到什么信号能想到它？

## 复盘模板

```md
# 题名

## 题目来源
平台 / 题号 / 链接

## 我第一次怎么想
...

## 卡住的位置
...

## 正确模型
...

## 关键变量
- `i`：
- `j`：
- `cnt`：

## 暴力复杂度
...

## 优化思路
...

## 下次识别信号
...
```

## 复盘要写人话

好复盘：

> `j in range(i)` 表示只看当前元素左边，不能写成 `range(n)`，否则会把右边也算进去。

差复盘：

> 注意循环。

第二种太空了，下次还是不会。
