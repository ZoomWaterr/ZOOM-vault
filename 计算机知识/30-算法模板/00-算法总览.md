## 学习路线建议

```
第一阶段（当前）
├─ 枚举、模拟 → 这些不用专门学，刷题自然就会
├─ 排序（Python 自带 sort/sorted）
├─ 前缀和 → 差分数组
└─ 字典计数（统计+映射）

第二阶段
├─ 双指针 → 滑动窗口
├─ 贪心
├─ 二分查找 → 二分答案
└─ 素数筛 → 快速幂

第三阶段
├─ DFS → BFS → 回溯
├─ 栈 → 并查集 → 堆
└─ DP 入门（爬楼梯 → 01背包）

第四阶段
├─ 单调栈
├─ 位运算
└─ 复杂 DP
```
