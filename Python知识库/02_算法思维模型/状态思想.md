# 状态思想

状态是算法里非常重要的词。DP、BFS、DFS、模拟题，本质都在处理状态。

## 什么是状态

状态就是“当前局面需要哪些信息才能描述清楚”。

例子：

- 走楼梯：当前在第几级。
- 迷宫：当前坐标 `(x, y)`。
- 背包：当前考虑到第几个物品，容量还剩多少。
- 滑动窗口：当前窗口左右边界和窗口内计数。

## 状态和变量的区别

变量是代码里的名字。

状态是问题里的局面。

好变量应该准确表达状态。

## BFS 里的状态

```python
q.append((x, y, step))
```

这里 `(x, y)` 是位置状态，`step` 是走了几步。

## DP 里的状态

```python
dp[i]
```

必须先说清楚：

> `dp[i]` 表示什么？

如果这句话说不清，后面的转移一定乱。

## 模拟题里的状态

例如攻击次数题：

```python
blood = 2025
turn = 1
```

当前状态至少包括：

- 还剩多少血。
- 当前第几回合。

每一轮根据规则修改状态。

## 练习方法

每道题写代码前，先写：

```text
这题的状态是：
状态会怎么变化：
什么时候结束：
答案从哪个状态得到：
```
