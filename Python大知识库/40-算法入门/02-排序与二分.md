# 02 排序与二分

排序让数据变得有秩序。很多题一旦排序，就能从混乱变清楚。

---

## Python 排序

```python
nums.sort()
nums.sort(reverse=True)
new_nums = sorted(nums)
```

---

## 按规则排序

```python
words.sort(key=len)
pairs.sort(key=lambda x: x[1])
```

---

## 冒泡排序的教学价值

冒泡排序不一定实战常用，但它能训练嵌套循环边界。

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
        if nums[当前位置] > nums[当前位置 + 1]:
            nums[当前位置], nums[当前位置 + 1] = nums[当前位置 + 1], nums[当前位置]
```

---

## 二分的入口

二分先记一句：如果答案有单调性，或者数据已经有序，就可能用二分。

---

## 小结

排序是把问题规整化。二分是利用规整后的顺序快速排除一半答案。
