# 模型误判

模型误判比做错更值得记录。因为算法能力的核心就是识别模型。

## 常见误判

- 看到“最优”就以为贪心，其实是 DP。
- 看到“连续”没想到滑动窗口/前缀和。
- 看到“多次区间”没想到差分。
- 看到“最短步数”用 DFS，其实应该 BFS。
- 有序数组还写双层循环，没想到二分/双指针。
