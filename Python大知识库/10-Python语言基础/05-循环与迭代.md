# 05 循环与迭代

循环解决的是重复。

只要题目里有“一批数”“每个字符”“重复直到”，就会出现循环。

---

## for：遍历已知对象

```python
for x in nums:
    print(x)
```

如果需要下标：

```python
for i in range(n):
    print(i, nums[i])
```

---

## range 的边界

```python
range(1, n + 1)
```

右端取不到。所以想遍历 `1` 到 `n`，要写 `n + 1`。

---

## while：不知道循环几次，但知道什么时候停

```python
while 条件:
    ...
```

例如小玉游泳这类“累加到超过目标”的题，就很适合 `while`。

---

## 嵌套循环

嵌套循环最容易错在变量名混乱。必要时写得直白一点：

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
        ...
```

变量名会说人话，调试成本会低很多。

---

## 小结

循环不是“把代码重复很多遍”，而是维护一个不断变化的状态。写循环时最重要的问题是：每一轮以后，哪个变量变了？
