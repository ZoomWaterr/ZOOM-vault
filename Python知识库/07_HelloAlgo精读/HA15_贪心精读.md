# HA15_贪心精读

对应原书：第 15 章 贪心。

## 本章一句话

贪心是每一步做局部最优选择，但必须能证明这样不会破坏全局最优。

## 必须掌握

- 贪心很快，但不是所有最优问题都能贪心。
- 常常需要排序。
- 正确性需要证明或交换论证。
- 找反例是判断贪心是否靠谱的重要方法。

## Python 小练习

1. 分数背包。
2. 最大容量双指针版。
3. 整数切分最大乘积的贪心思路。
4. 找一个“看起来能贪心但其实不行”的反例。

## 做题信号

- 每次选当前最早、最小、最大、最划算。
- 排序后从一端处理。
- 局部选择似乎不会后悔。
