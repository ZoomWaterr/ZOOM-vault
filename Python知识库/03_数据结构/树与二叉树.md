# 树与二叉树

树是一种层级结构。二叉树是每个节点最多有两个孩子的树。

## 树题核心

很多树题其实就是递归：

```python
def dfs(node):
    if not node:
        return
    dfs(node.left)
    dfs(node.right)
```

## 三种遍历

| 遍历 | 顺序 |
|---|---|
| 前序 | 根 左 右 |
| 中序 | 左 根 右 |
| 后序 | 左 右 根 |

## 题目信号

- 父子关系。
- 层级。
- 子树。
