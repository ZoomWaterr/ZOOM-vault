# 循环枚举模板

## 枚举每个元素

```python
for i in range(n):
    # i 是下标
    x = nums[i]
```

## 枚举下标和值

```python
for i, x in enumerate(nums):
    ...
```

## 枚举所有数对，不重复

```python
for i in range(n):
    for j in range(i + 1, n):
        # i < j
        ...
```

## 枚举当前元素左边

```python
for i in range(n):
    for j in range(i):
        # j < i
        ...
```

## 遍历矩阵

```python
for i in range(n):
    for j in range(m):
        x = grid[i][j]
```
