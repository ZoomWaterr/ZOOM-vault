# 2973 出现次数超过一半复盘样例

## 来源

C语言网：2973 出现次数超过一半的数  
源码：`F:\Code\PythonCode\C语言网\算法练习题杂烩合集\2973 出现次数超过一半的数.py`

## 题目在问什么

给一组数，找有没有某个数出现次数超过一半。

## 正确模型

[[哈希计数]]。

因为题目核心就是“每个数字出现几次”。

## 关键变量

- `nums`：所有输入数字。
- `count`：数字到出现次数的映射。
- `num`：当前遍历到的数字。
- `cnt`：这个数字出现了几次。

## 核心代码

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1

for num, cnt in count.items():
    if cnt > n / 2:
        print(num)
        break
else:
    print("no")
```

## 为什么用字典

因为你需要建立：

```text
数字 -> 出现次数
```

这就是字典最典型的场景。

## 复杂度

- 统计一遍：`O(n)`。
- 遍历字典：`O(k)`，`k` 是不同数字个数。
- 总体：`O(n)`。

## 易错点

- `count.get(x, 0) + 1` 是计数模板。
- `for ... else` 里的 `else` 是循环没被 `break` 时执行。
- `cnt > n / 2` 可以写成 `cnt * 2 > n`，避免小数。

## 进阶

这题还有 Boyer-Moore 投票算法，可以做到 `O(1)` 额外空间。但当前阶段先把哈希计数吃透。
