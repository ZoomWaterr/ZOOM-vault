# 比赛前速查

比赛前不要看大段理论，看这些最容易忘的点。

## 输入输出

```python
a, b = map(int, input().split())
nums = list(map(int, input().split()))
print(*ans)
```

## 计数

```python
count = {}
for x in nums:
    count[x] = count.get(x, 0) + 1
```

## 前缀和

```python
prefix = [0] * (n + 1)
for i in range(n):
    prefix[i + 1] = prefix[i] + nums[i]
# [l, r]
s = prefix[r + 1] - prefix[l]
```
