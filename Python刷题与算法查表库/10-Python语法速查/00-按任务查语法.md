**输入输出**

| 任务 | 写法 |
|---|---|
| 读一个整数 | `n = int(input())` |
| 读两个整数 | `a, b = map(int, input().split())` |
| 读整数列表 | `nums = list(map(int, input().split()))` |
| 输出列表 | `print(*nums)` |
| 保留小数 | `print(f"{x:.3f}")` |

**处理列表**

| 任务 | 写法 |
|---|---|
| 创建 n 个 0 | `a = [0] * n` |
| 二维列表 | `grid = [ [0] * m for _ in range(n) ]` |
| 追加 | `a.append(x)` |
| 排序 | `a.sort()` |
| 不改原列表排序 | `sorted(a)` |
| 反转 | `a[::-1]` |

**处理字符串**

| 任务 | 写法 |
|---|---|
| 拆单词 | `s.split()` |
| 去空白 | `s.strip()` |
| 替换 | `s.replace(old, new)` |
| 拼接 | `" ".join(words)` |
| 找最长 | `max(words, key=len)` |

**统计**

| 任务 | 写法 |
|---|---|
| 字典计数 | `count[x] = count.get(x, 0) + 1` |
| Counter | `Counter(nums)` |
| 找最高频 | `max(count, key=count.get)` |
