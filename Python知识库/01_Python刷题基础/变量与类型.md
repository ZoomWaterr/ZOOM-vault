# 变量与类型

变量不是随便起名字。刷题时变量名要能提醒自己“它代表什么”。

## 常见类型

| 类型 | 例子 | 刷题用途 |
|---|---|---|
| `int` | `3` | 计数、下标、答案 |
| `float` | `3.14` | 小数题 |
| `str` | `"abc"` | 字符串处理 |
| `bool` | `True` | 条件判断 |
| `list` | `[1, 2, 3]` | 存一排数据 |
| `dict` | `{1: 2}` | 建立映射、计数 |
| `set` | `{1, 2}` | 去重、快速判断存在 |

## 变量名建议

| 变量 | 常见含义 |
|---|---|
| `n` | 数量 |
| `m` | 另一组数量、边数、操作数 |
| `i` | 当前下标 |
| `j` | 配合 `i` 的另一个下标 |
| `cnt` | 当前计数 |
| `ans` | 最终答案 |
| `left`, `right` | 左右边界 |
| `prefix` | 前缀和数组 |
| `visited` | 是否访问过 |

## 核心习惯

写循环前先写一句注释：

```python
# i 表示当前处理到第几个数
# cnt 表示当前数左边比它小的数量
```

变量含义清楚，循环就不会那么乱。
