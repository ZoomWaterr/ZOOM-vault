# 维护说明

## 这个库怎么维护

- 新体会先写到 [[../00-开始这里/04-以后写给AI看的笔记区]]。
- 每次整理时，先读总导航和 AI 整理规则。
- 不要把题目批量搬进来。
- 不要把外部参考链接堆到文末。
- Markdown 必须能正常渲染：frontmatter 顶格、表格顶格、代码围栏顶格。

## 内容风格

- 像教材：结构清楚，有解释。
- 像你自己：保留口语、emoji、真实提醒。
- 不装：不会的地方就标“待补”，不要硬写。
