# AI 整理规则

## 必须遵守

- 先读 [[../00-开始这里/00-总导航]]。
- 再读 [[../00-开始这里/04-以后写给AI看的笔记区]]。
- 原始注释和真实想法只提炼，不抹掉。
- 不要按平台切笔记。
- 不要批量塞题。
- 不要添加参考文档链接堆。
- 优先把图片和零散内容转成文字总结。
- 不做过度复杂的系统。

## 写作格式

每篇尽量遵循：

1. 一句话理解。
2. 什么时候用。
3. 怎么写。
4. 完整解释。
5. 我容易摔在哪里。
