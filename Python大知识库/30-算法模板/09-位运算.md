# 位运算 ⭐⭐

> 直接在二进制位上操作，快而且省空间。

---

## 速查表

| 要做什么 | 怎么写 | 原理 |
|:---|:---|:---|
| 判断奇偶 | `x & 1` | 奇数末位是 1 |
| 拿到第 k 位 | `(x >> k) & 1` | 右移到末位再取 |
| 第 k 位置为 1 | `x \| (1 << k)` | 生成只有第 k 位是 1 的数，或上去 |
| 第 k 位置为 0 | `x & ~(1 << k)` | 取反变成「只有第 k 位是 0」，与上去 |
| 翻转第 k 位 | `x ^ (1 << k)` | 异或 1 就是翻转 |
| 去掉最低位的 1 | `x & (x - 1)` | **统计二进制里有多少个 1 的核心** |
| 保留最低位的 1 | `x & -x` | 树状数组的核心操作 |
| 判断是不是 2 的幂 | `x>0 and x&(x-1)==0` | 2 的幂二进制只有一个 1 |
| 乘 2 / 除 2 | `x << 1` / `x >> 1` | 左移=乘2，右移=除2（向下取整） |

## 统计二进制中 1 的个数

```python
def count_bits(x: int) -> int:
    cnt = 0
    while x:
        x &= x - 1           # 每次消掉最低位的 1
        cnt += 1
    return cnt

# 13 = 1101₂ → 3
```

## 异或的三个妙用

```python
# ① 找出唯一出现一次的数（其他都恰好出现两次）
def single_number(nums):
    result = 0
    for x in nums:
        result ^= x          # 相同数异或=0，最后剩下单身狗
    return result

# single_number([4,1,2,1,2]) → 4

# ② 判断两数符号是否相同
def same_sign(a, b):
    return (a ^ b) >= 0      # 同号 → 最高位相同 → 异或后最高位是 0

# ③ 交换两数（装逼用，Python 直接 a,b = b,a 就行）
a ^= b; b ^= a; a ^= b
```
