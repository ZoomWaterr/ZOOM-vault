# 双指针与滑动窗口

---

## 对撞指针 ⭐⭐

> 左右同时向中间走。有序数组两数之和 O(n)。

```python
def two_sum(arr, target):
    l, r = 0, len(arr) - 1
    while l < r:
        cur = arr[l] + arr[r]
        if cur == target: return (l, r)
        elif cur < target: l += 1
        else: r -= 1
    return (-1, -1)
```

---

## 快慢指针 ⭐⭐

> 一快一慢同向走。原地去重。

```python
def remove_dup(nums):
    slow = 0
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]
    return slow + 1
```

## 滑动窗口 ⭐⭐

> 窗口像毛毛虫移动。右扩左缩。子串/子数组问题 O(n)。

```python
def longest_no_repeat(s):
    window = set()
    left = best = 0
    for right, ch in enumerate(s):
        while ch in window:           # ★ while不是if！可能弹多个
            window.remove(s[left])
            left += 1
        window.add(ch)
        best = max(best, right - left + 1)
    return best
```

🎯 "子数组""子串""连续""最长/最短" → 滑动窗口

### ⚠️ 收缩用while不是if；更新顺序要搞对
