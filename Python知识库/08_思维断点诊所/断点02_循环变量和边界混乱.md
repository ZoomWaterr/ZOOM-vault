# 断点02_循环变量和边界混乱

## 症状

写 `for i in range(...)` 时不知道范围怎么定；双层循环里 `i`、`j` 越写越乱。

## 根因

你在写循环前没有定义变量角色。

## 先写角色表

```text
i 表示：
j 表示：
i 的范围：
j 的范围：
每轮更新：
```

## 常见循环意义

```python
for i in range(n):
```

`i` 表示每个位置。

```python
for j in range(i):
```

`j` 表示 `i` 左边的位置。

```python
for j in range(i + 1, n):
```

`j` 表示 `i` 右边的位置。

```python
for i in range(n):
    for j in range(m):
```

`i` 是行，`j` 是列。

## 最小练习

解释下面代码，不运行：

```python
for i in range(5):
    for j in range(i):
        print(i, j)
```

你应该能说出：

> 它枚举所有 `j < i` 的数对。
