# 流程控制

## 先看这个

        `if` 负责分情况，`for` 负责遍历已知范围，`while` 负责循环次数不确定的过程。

        ## 什么时候用

        题目有分类判断、重复处理、寻找满足条件的元素时。

        ## 怎么写

        ```python
        for i in range(n):
    ...

for idx, value in enumerate(nums):
    ...

while x > 0:
    x //= 10
        ```

        ## 我容易摔在哪里

        嵌套循环别死用 `i/j`。如果含义复杂，用 `轮数`、`当前位置`、`行`、`列` 这种变量名，能少摔很多。

## if / elif / else

```python
if score >= 90:
    print("A")
elif score >= 60:
    print("B")
else:
    print("C")
```

## for 与 range

```python
for i in range(5):      # 0,1,2,3,4
    print(i)

for i in range(1, 6):   # 1,2,3,4,5
    print(i)
```

## 嵌套循环写清楚

```python
for 行 in range(n):
    for 列 in range(m):
        print(grid[行][列])
```

## 我容易摔在哪里

你代码里循环很多，但函数很少。现阶段没问题，基础题平铺也能做；但一旦模拟题变长，就要开始主动分块：读入、处理、输出。
