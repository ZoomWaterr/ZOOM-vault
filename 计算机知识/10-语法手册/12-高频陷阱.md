## 列表相关

### 1. 二维列表创建 — 最经典的大坑

```python
# ❌ 错误：三行指向同一个列表！改一行全变
bad = [[0] * 5] * 3

# ✅ 正确：每行独立
good = [[0] * 5 for _ in range(3)]
```

### 2. list.pop(0) 是 O(n)

```python
# ❌ 频繁从头部删除 → TLE
while lst:
    x = lst.pop(0)       # 每次 O(n)

# ✅ 用 deque
from collections import deque
q = deque(lst)
x = q.popleft()           # O(1)
```

### 3. 循环里 if x in list

```python
# ❌ O(n²) → n > 10⁴ 就超时
for x in data:
    if x in lst:          # 每次 O(n)
        ...

# ✅ 把 list 转成 set
s = set(lst)
for x in data:
    if x in s:            # O(1)
        ...
```

### 4. 遍历时删除元素

```python
# ❌ 遍历中删除 → 漏掉元素
for x in lst:
    if condition(x):
        lst.remove(x)

# ✅ 推导式新建
lst = [x for x in lst if not condition(x)]
```

## 字符串相关

### 5. 忘了接收返回值

```python
# ❌ 字符串方法不修改原字符串！
s.strip()          # 这样写，s 没变！

# ✅ 必须赋值
s = s.strip()
```

### 6. split 和 rstrip 的边界

```python
# rstrip(".") 只去末尾一个字符，不是全部
"hello...".rstrip(".")   # → "hello"  ✓ 实际上去掉所有连续的点
# 但如果 "." 是不连续的，要小心

# rstrip 不是「去掉最后一个」，而是「去掉末尾所有匹配字符」
"test..".rstrip(".")     # → "test"  去掉末尾所有句号
"test.a.".rstrip(".")    # → "test.a"  只去末尾，不去中间
```

## 字典相关

### 7. 直接用 d[key] 而不是 get

```python
# ❌ key 不存在 → KeyError
count[x] += 1              # 第一次碰到 x 直接崩溃

# ✅ 用 get
count[x] = count.get(x, 0) + 1
```

### 8. 空字典 vs 空集合

```python
d = {}       # 这是空字典！
s = set()    # 这才是空集合！
```

## 循环相关

### 9. 嵌套循环用 i、j 导致混乱

这是我本人踩得最多的坑。在「车厢重组 (2117)」那道冒泡排序里：

```python
for i in range(n):
    for j in range(n-1-i):
        if nums[j] > nums[i]:      # ← 应该是 j+1！i 是外层轮数
```

✅ 解法：**嵌套循环用中文变量名**

```python
for 轮数 in range(n):
    for 当前位置 in range(n - 1 - 轮数):
        if nums[当前位置] > nums[当前位置 + 1]:
```

## 算术相关

### 10. / 和 // 搞混

```python
# / 永远返回 float
5 / 2    # → 2.5
4 / 2    # → 2.0  不是 2！

# // 才是整数除法
5 // 2   # → 2
4 // 2   # → 2   整数！
```

> OJ 判输出格式时经常因为这个判 WA。

### 11. int() 截断 vs round() 四舍五入

```python
int(3.9)    # → 3    截断
round(3.9)  # → 4    四舍五入
```

### 12. 浮点数比较

```python
0.1 + 0.2 == 0.3      # → False！浮点精度问题
abs(0.1+0.2 - 0.3) < 1e-9   # → True   正确做法
```

## 函数相关

### 13. 默认参数只创建一次

```python
# ❌ 所有调用共享同一个列表
def add(x, lst=[]):
    lst.append(x)
    return lst

# ✅
def add(x, lst=None):
    if lst is None:
        lst = []
    lst.append(x)
    return lst
```
