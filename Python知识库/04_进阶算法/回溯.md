# 回溯

回溯就是“试一个选择，继续走；走不通或走完后撤销选择”。

## 题目信号

- 所有排列。
- 所有组合。
- 所有子集。
- 棋盘放置。
- 需要尝试所有选择。

## 模板

```python
ans = []
path = []

def backtrack(选择列表):
    if 到达结束条件:
        ans.append(path[:])
        return

    for choice in 选择列表:
        if choice 不合法:
            continue
        path.append(choice)
        backtrack(新的选择列表)
        path.pop()
```

## 核心变量

- `path`：当前已经做出的选择。
- `ans`：所有答案。
- `used`：某些题里表示元素是否用过。

## 易错点

- 加入答案时要 `path[:]`，不能直接放 `path`。
- 做选择后要撤销。
- 回溯本质通常复杂度很高，要注意剪枝。
