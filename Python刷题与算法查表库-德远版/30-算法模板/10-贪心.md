# 🎲 贪心

## 🚀 快速定位

先排序，再按规则选。

## 💡 一句话

每一步选当前最优，并且这个选择不会害后面。

## 🎯 适用场景

- 排序后选择
- 区间选点
- 排队等待
- 局部最优明显

## 🧠 原理理解

贪心难点不是写，是为什么局部最优能推出全局最优。

## 📝 标准模板

```python
items.sort(key=lambda x: x[0])
ans = 0
for item in items:
    if can_choose(item):
        choose(item)
        ans += 1
```

## ⚠️ 易错点

- 看起来对不一定真对
- 先找反例
- 排序规则决定成败
