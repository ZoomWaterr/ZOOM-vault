**快速结论**
回溯用于枚举所有方案：做选择、递归、撤销选择。

**什么时候用**
- 全排列。
- 组合。
- 子集。
- 棋盘放置。
- 题目要求输出所有方案。

**核心框架**

```text
路径：已经做过的选择
选择列表：当前还能做的选择
结束条件：什么时候记录答案

```

**全排列模板**

```python
ans = []
path = []
used = [False] * len(nums)

def backtrack():
    if len(path) == len(nums):
        ans.append(path[:])
        return

    for i, x in enumerate(nums):
        if used[i]:
            continue
        used[i] = True
        path.append(x)
        backtrack()
        path.pop()
        used[i] = False

backtrack()

```

**组合模板**

```python
ans = []
path = []

def backtrack(start):
    if len(path) == k:
        ans.append(path[:])
        return

    for i in range(start, n + 1):
        path.append(i)
        backtrack(i + 1)
        path.pop()

```

**常见变体**
- 排列：需要 used 数组。
- 组合：用 start 防止重复。
- 子集：每个位置选或不选。
- N 皇后：行、列、对角线约束。

**复杂度**
排列通常 `O(n!)`；子集通常 `O(2^n)`。

**易错点**
- 选了什么，递归后必须撤销什么。
- 保存答案要用 `path[:]` 拷贝。
- 组合题不要从 0 重新循环，否则会重复。
