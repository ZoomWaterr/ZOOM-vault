# 贪心

## 先看这个

贪心是每一步都做当前看起来最好的选择，并且这个选择能导向整体最优。

## 什么时候用

- 排序后按某个规则选。
- 每次选择不会影响未来可行性，或影响可以证明是好的。
- 题目问最多、最少、最小代价。

## 怎么写

```python
times.sort()
total = 0
wait = 0
for t in times:
    total += wait
    wait += t
print(total)
```

## 我容易摔在哪里

贪心最难不是代码，而是“为什么这样选一定对”。新手阶段可以先积累典型题，不急着硬证明所有贪心。
